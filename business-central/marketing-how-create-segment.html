    Segmente erstellen

    Sie können Segmente zur Auswahl einer Gruppe von Kontakten erstellen, z. B. wenn Sie eine Aktivität mit mehreren Kontakten erstellen möchten (z. B. Direktmail).

    So erstellen Sie ein Segment

    1. Wählen Sie die Symbol. Geben Sie Segmente ein, und wählen Sie dann den zugehörigen Link.
    2. Wählen Sie die Aktion Neu.

    3. Geben Sie im Abschnitt Allgemein im Feld Nr. eine Nummer für das Segment ein.

      Wenn Sie auf der Seite Marketing Vertrieb Einr. Nummernserien für Segmente eingerichtet haben, können Sie die Eingabetaste auswählen, um die nächste verfügbare Segmentnummer einzufügen.

    4. Füllen Sie die anderen Felder im Kopf aus.

    Jetzt können Sie Kontakte zum Segment hinzufügen. Weitere Informationen finden Sie unter Kontakte zu Segmenten hinzufügen.
