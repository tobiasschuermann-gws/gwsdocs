    Debitorenposten holen

    Um Debitorenposten in das Factoring einzubeziehen, werden diese in ein Factoring Buch.-Blatt hinzugefügt. Sie können Posten entweder manuell hinzufügen, um zum Beispiel die Forderung eines bestimmten offenen Postens zu verkaufen oder Sie können, sobald sich eine gewisse Regel definieren lässt, die Debitorenposten automatisiert zusammenstellen lassen. Das wäre zum Beispiel der Fall, wenn Sie alle Forderungen gegenüber eines bestimmten Debitoren verkaufen möchten. Wenn Sie alle gewünschten Debitorenposten in das Buch.-Blatt aufgenommen haben, müssen Sie es buchen, um fortzufahren.

    Das Factoring Buch.-Blatt erreichen Sie wie folgt:

    1. Klicken Sie in der rechten oberen Ecke auf die Lupe und geben Sie Factoring Buch.-Blatt ein. Wählen Sie dann den zugehörigen Link aus.
    2. Falls Sie mehrere Factoring Buch.-Blattvorlagen angelegt haben, müssen Sie jetzt die gewünschte Vorlage auswählen.

    Manuelles Zusammenstellen

    Auf der Factoring Buch.-Blatt-Seite können Sie mithilfe des Feldes Debitorenposten Lfd. Nr. Debitorenposten hinzufügen.

    Tipp

    Falls Sie mit mehreren Factoring-Partnern zusammenarbeiten sollten, bedenken Sie, dass ein Factoring Buch.-Blatt einem Factoring-Partner zugeordnet ist und Sie das richtige verwenden.

    Einmaliges automatisiertes Zusammenstellen

    Auf der Factoring Buch.-Blatt-Seite gibt es die Aktion Buch.-Blatt füllen..., womit Sie die Debitorenposten einmalig automatisiert zusammenstellen lassen können. Sofern ein Factoring-Partner am Factoring Buch.-Blatt bzw. an der Factoring Buch.-Blattvorlage hinterlegt ist, werden die dort hinterlegten Optionen und Filter vorgeschlagen.
    Diese Funktion können Sie nutzen, wenn Sie z. B. einmalig alle Forderungen gegenüber eines Debitors verkaufen möchten. Falls Sie regelmäßig alle Forderungen verkaufen möchten, nutzen Sie bitte das wiederkehrende automatisierte Zusammenstellen.

    Wiederkehrendes automatisiertes Zusammenstellen

    Damit die oben genannte Funktion regelmäßig, also täglich oder wöchentlich ausgeführt wird, ist Sie in die Aufgabenwarteschlange einzuhängen. Unter Automatischer Export über die Aufgabenwarteschlange ist das Vorgehen hierzu geschildert.
