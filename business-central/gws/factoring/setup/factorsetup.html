    Factoring-Partner einrichten

    Die Einstellungen, die für die Zusammenstellung der Debitorenposten und deren Export an Ihren Factoring-Partner benötigt werden, sind am Factoring-Partner hinterlegt. Der Factoring-Partner wird wiederum einem Factoring Buch.-Blatt zugeordnet.

    1. Klicken Sie in der rechten oberen Ecke auf die Lupe und geben Sie Factoring-Partner ein. Wählen Sie dann den zugehörigen Link aus.
    2. Wählen Sie die Aktion Neu aus.

    3. Füllen Sie die Felder auf der Factoring-Partner-Seite, wie in der folgenden Tabelle beschrieben.

      Feld Beschreibung
      Code Geben Sie einen eindeutigen Code für den Factoring-Partner ein.
      Beschreibung Geben Sie eine Beschreibung für den Factoring-Partner ein.
      Exportiere nur neue Posten Aktivieren Sie diese Funktion, wenn Ihr Factoring-Partner nur neue Debitorenposten erwartet. Sobald ein Debitorenposten bereits über das Factoring Buch.-Blatt gebucht wurde, wird er später nicht noch einmal gebucht.
      Kunden-/Vertragsnummer Geben Sie Ihre Kunden- oder Vertragsnummer an, die Sie von Ihrem Factoring-Partner erhalten.
      Debitorenexport Datenaustauschdefinition Geben Sie eine Export Datenaustauschdefinition an. Sie wird während des Exports der Debitorenstammdaten benutzt, um die Business Central-Felder auf die Felder zu mappen, die Ihr Factoring-Partner benötigt.
      Debitorenlöschexport Datenaustauschdefinition Geben Sie eine Export Datenaustauschdefinition an. Sie wird während des Exports verwendet, wenn Sie Ihrem Factoring-Partner Debitorenstammdaten mit einem Löschkennzeichen zukommen lassen möchten.
      Postenexport Datenaustauschdefinition Geben Sie eine Export Datenaustauschdefinition an. Sie wird während des Exports der Factoringposten benutzt, um die Business Central-Felder auf die Felder zu mappen, die Ihr Factoring-Partner benötigt.
      MwSt.-Schlüssel Mapping Code Geben Sie einen Code für das MwSt.-Schlüssel Mapping ein. Hiermit können Sie die Business Central MwSt.-Schlüssel auf die MwSt.-Schlüssel Ihres Factoring-Partners mappen.
      Export Datentransfer-Endpunkt Geben Sie einen Code für den Datentransfer-Endpunkt an, der für den Export verwendet werden soll. Weitere Einstellungen bzgl. des Export können über das Datentransfer Framework vorgenommen werden.
      Debitorenposten Export Dateinamen Gibt den Dateinamen der Debitorenposten-Datei an. Es können Platzhalter verwendet werden um bestimmte dienstleisterspezifische Informationen im Dateinamen unterzubringen.
      Debitoren Export Dateinamen Gibt den Dateinamen der Debitoren-Datei an. Es können Platzhalter verwendet werden um bestimmte dienstleisterspezifische Informationen im Dateinamen unterzubringen.
      Import Datenaustauschdefinition Geben Sie die Import Datenaustauschdefinition an. Sie wird benutzt, um die Felder, die Sie von Ihrem Factoring-Partner erhalten, auf die Business Central-Felder zu mappen.

    4. Falls gewünscht, nutzen Sie die Aktion Debitorenpostenfilter setzen, um eine Filterung auf die Debitorenposten für das Zusammenstellen des Factoring Buch.-Blatts festzulegen.

    5. Falls gewünscht, nutzen Sie die Aktion Debitorenfilter setzen, um eine Filterung auf die Debitoren für das Zusammenstellen des Factoring Buch.-Blatts festzulegen.
    6. Bevor Sie Ihre offenen Debitorenposten an einen Factoring-Partner verkaufen können, benötigt dieser in der Regel vorab die Debitorenstammdaten.
      Diese Informationen können über Debitoren exportieren... als CSV-Datei heruntergeladen und manuell überreicht werden oder direkt an die angegebene E-Mail-Adresse Ihres Factoring-Partners gesendet werden.
