    Kreditorenzahlungen einrichten

    Sie können in gevis ERP | VEO die Zahlung von Kreditorenrechnungen veranlassen. Hinterlegen Sie in der Kreditorenkarte den zutreffenden Zahlungsformcode und das Bankkonto des Kreditors.
    Das Modul Continia OPplus365 erstellt Zahldateien im neuen SEPA-Format, das von allen Banken gelesen werden kann. Für Zahlvorgänge mit diesem Modul sind zusätzlich Empfängerinformationen in der OPplus Kreditorenkarte zu hinterlegen und eine Zahlungsbuchungsmatrix für SEPA einzurichten.

    So richten Sie die Kreditorenzahlungen in der Kreditorenkarte ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kreditoren ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Kreditorenkarte.
    3. Klicken Sie im Inforegister Zahlungen in das Feld Zlg.-Bedingungscode und wählen Sie einen Code aus oder klicken Sie auf Neu, um einen neuen Datensatz anzulegen.
    4. Klicken in das Feld Zahlungsformcode und wählen Sie den Datensatz Bank aus.
    5. Klicken Sie in das Feld Bevorzugter Bankkontocode und wählen Sie einen Code aus.

    So legen Sie ein neues Bankkonto für den Kreditor an

    1. Klicken Sie optional im Menüband auf die Schaltfläche Zugehörig > Kreditor > Sonstiges > Bankkonten.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie in das Feld Code und geben Sie einen Code für das Bankkonto des Kreditors ein.
    4. Klicken Sie in die restlichen Felder und geben Sie entsprechende Werte ein, um ein Bankkonto des Kreditors zu hinterlegen.

    So richten Sie in Ihrer Bankkontokarte eine Bankkontobuchungsgruppe für Kreditorenzahlungen ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Bankkonten ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffenen Sie eine Bankkontokarte.
    3. Klicken Sie im Inforegister Buchung in das Feld Bankkontobuchungsgruppe und wählen Sie den Datensatz BANK aus.

    So richten Sie die OPplus Kreditor Bankkontokarte ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld OPplus Kreditor Bankkontokarte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Geben Sie im Inforegister Transfer die Daten für das Bankkonto des Kreditors ein.

    So richten Sie den automatisieren Mailversand für die Kreditorenzahlungen ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Zgl. Export Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Geben Sie im Inforegister Avise, Gruppe Mailversand im Feld E-Mail Absender eine E-Mail Adresse ein.
    3. Klicken Sie in das Feld Mahnmethodencode für Avis-E-Mail-Text und wählen Sie die Option Avis aus.
    4. Klicken Sie in das Feld Avisversand Automatik und wählen Sie die Option Nur E-Mail aus.

    So hinterlegen Sie Empfängerinformationen in der OPplus Kreditorenkarte

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld OPplus Kreditorenkarte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Zlg. Export in das Feld Avisversandart und wählen Sie die Option E-Mail aus.
    3. Klicken Sie in das Feld Avisversandadresse und wählen Sie einen Datensatz aus.

    So richten Sie eine Zahlungsbuchungsmatrix für SEPA ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Zahlungsbuchungsmatrix ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie in das Feld Zahlungsartencode und wählen Sie den Datensatz SEPA aus.
    4. Klicken Sie in das Feld Bankkonto und wählen Sie ein Bankkonto aus.
    5. Klicken Sie in das Feld Buchungskontoart und wählen Sie die Option Sachkonto aus.
    6. Klicken Sie in das Feld Buchnungkontonr. und wählen Sie einen Datensatz aus.
    7. Geben Sie im Feld Kontoname eine Kurzbeschreibung ein (zum Beispiel Geldtransit).
    Hinweis

    Sie können für den Zahlungsartencode SEPA einen Avis-Text einrichten auf der Seite Zahlungsarten. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Zahlungsarten ein und klicken Sie auf das übereinstimmende Suchergebnis. Im Feld Text Verwendungszweck Kreditor können Sie einen entsprechenden Text eingeben.
