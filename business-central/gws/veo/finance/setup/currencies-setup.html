    Währungscodes einrichten

    Für den internationalen Warenverkehr können Sie Finanzdaten in mehreren Währungen erfassen. Die Mandatenwährung (MW) richten Sie auf der Seite Finanzbuchhaltung Einrichtungen ein. Legen Sie Währungscodes für weitere Währungen auf der Seite Währungen an.

    So richten Sie die Mandantenwährung (MW) ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Finanzbuchhaltung Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister Allgemein den Schalter EWU-Währung.
    3. Geben Sie im Feld Mandantenwährungscode den Wert EUR ein.
    4. Geben Sie im Feld Symbol der Landeswährung den Wert € ein.
    5. Geben Sie im Feld Währungsbeschreibung den Wert Euro ein.
    Hinweis

    Sobald die Mandantenwährung (MW) angelegt ist, wird sie als leere Währung dargestellt. Ist das Feld Währung in den Belegen leer, handelt es sich um die Mandantenwährung (MW).

    So richten Sie Währungscodes für weitere Währungen ein

    Hier am Beispiel der Währung für das Land Kanada:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Währungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Code eine Bezeichnung (zum Beispiel CAD) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Kanadische Dollar) ein.
    4. Geben Sie im Feld ISO-Code den in ISO 4217 definierten Währungscode mit drei Buchstaben ein. In unserem Beispiel ist das der Wert CAD.
    5. Geben Sie im Feld Nummerischer ISO-Code den in ISO 4217 definierten Währungscode mit drei Ziffern ein. In unserem Beispiel ist das der Wert 124.
