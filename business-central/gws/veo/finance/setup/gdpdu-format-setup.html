    Export von GDPdU-Daten einrichten

    Um Daten im GDPdU-Format exportieren zu können, benötigen Sie Codes für den Export und eine Datensatzdefinition. In der Datensatzdefinition muss zudem die DTD-Datei ergänzt werden.

    So richten Sie einen Code für den Export ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Datenexporte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Code eine Bezeichnung (zum Beispiel GDPDU) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Export der GDPDU Daten) ein.

    So richten Sie einen Code für die Datensatzdefinition ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Datenexporte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie eine Zeile (in unserem Beispiel mit dem Code GDPDU).
    3. Klicken Sie im Menüband auf die Schaltfläche Exportieren > Datensatzdefinitionen.
    4. Geben Sie im Feld Datenexport - Datensatz-Typcode eine Bezeichnung (zum Beispiel GDPDU) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Export der GDPDU Daten) ein.
    5. Um die DTD-Datei hochzuladen, klicken Sie im Menüband auf die Schaltfläche DTD-Datei > Importieren.

    6. Laden Sie die DTD-Datei hoch, indem Sie sie per Drag-and-Drop in den markierten Bereich ablegen.

      Hinweis

      Wenn Sie eine neue DTD-Datei importieren müssen, um eine vorhandene Datei zu ersetzen, müssen Sie die vorhandene DTD-Datei zuerst exportieren. Klicken Sie im Menüband auf die Schaltfläche DTD-Datei > Exportieren.

    So richten Sie die Datensatzdefinition ein

    Für einen Standard-Export können Tabellen- und Felddefinitionen per Konfigurationspaket importiert werden. Bei Bedarf können zusätzliche Tabellen oder Felder hinzugefügt werden.

    So übernehmen Sie ein Konfigurationspaket für den Export von Daten

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Konfigurationspakete ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Inforegister Allgemein im Feld Code eine Bezeichnung (zum Beispiel GDPDU) ein.
    4. Wählen Sie im Feld Paketname den Namen des Konfigurationspakets aus.
    5. Klicken Sie im Menüband auf die Schaltfläche Paket übernehmen.

    So fügen Sie zusätzliche Tabellen oder Felder hinzu

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Datenexporte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie eine Zeile (in unserem Beispiel mit dem Code GDPDU)
    3. Klicken Sie im Menüband auf die Schaltfläche Exportieren > Datensatzdefinitionen.
    4. Klicken Sie im Menüband auf die Schaltfläche Datensatzdefinition > Datensatzherkunft.
    5. Klicken Sie in das Feld Tabellennr. und wählen Sie einen Datensatz aus.
    6. Klicken Sie alternativ im Menüband des Inforegisters Felder auf die Schaltfläche Hinzufügen, wählen Sie einen Datensatz aus und bestätigen Sie Ihre Auswahl mit OK.
