    Debitorenzahlungen einrichten

    Sie können in gevis ERP | VEO Lastschriften einziehen, wenn Sie mit Ihren Debitoren die Zahlungsbedingung per Lastschrift vereinbart haben. Hinterlegen Sie für das Lastschriftverfahren in der Debitorenkarte den zutreffenden Zahlungsformcode und das Bankkonto des Debitors.
    Das Modul Continia OPplus365 erstellt Zahldateien im neuen SEPA Format, das von allen Banken gelesen werden kann. Für Zahlvorgänge mit diesem Modul sind zusätzlich Empfängerinformationen in der OPplus Debitorenkarte zu hinterlegen und eine Zahlungsbuchungsmatrix für SEPA einzurichten.

    So richten Sie Debitorenzahlungen in der Debitorenkarte ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Debitoren ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Debitorenkarte.
    3. Klicken Sie im Inforegister Zahlungen in das Feld Partnerart und wählen Sie die Option Unternehmen aus. Für die CORE-Lastschrift wählen Sie die Option Person aus.
    4. Klicken Sie in das Feld Zahlungsformcode und wählen Sie den Datensatz LS aus.

    5. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Debitor > Sonstiges > Lastschrift-Mandate.

      Hinweis

      Um ein Lastschrift-Mandat anlegen zu können, muss auf der Seite Einrichtung Debitoren und Verkauf im Inforegister Nummernserie im Feld Lastschrift-Mandat-Nr. eine Nummernserie ausgewählt sein.

    6. Klicken Sie in das Feld Debitor Bankkontocode und wählen Sie das Bankkonto aus, für das das Lastschriftmandat aktiviert ist.
    7. Geben Sie in den Feldern Gültigkeit ab und Gültig bis jeweils das richtige Datum ein.

    So legen Sie ein neues Bankkonto für den Debitor an

    1. Klicken Sie im Menüband der Debitorenkarte auf die Schaltfläche Debitor > Bankkonten.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie ins das Feld Code und geben Sie einen Code für das Bankkonto des Debitors ein.
    4. Klicken Sie in die restlichen Felder und geben Sie entsprechende Werte ein, um ein Bankkonto des Debitors für das Lastschriftverfahren zu hinterlegen.

    So richten Sie in Ihrer Bankkontokarte eine Bankkontobuchungsgruppe für Debitorenzahlungen ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Bankkonten ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Bankkontokarte.
    3. Klicken Sie im Inforegister Buchung in das Feld Bankkontobuchungsgruppe und wählen Sie den Datensatz BANK aus.

    So richten Sie die OPplus Debitor Bankkontokarte ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld OPplus Debitor Bankkontokarte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Geben Sie im Inforegister Transfer die Daten für das Bankkonto des Debitors ein.
    3. Klicken Sie im Inforegister OPplus und geben Sie im Feld Mandatsdatum das richtige Datum ein.
    4. Klicken Sie in das Feld SEPA-Verfahren und wählen Sie die Option CORE aus.
    5. Klicken Sie in die Felder Mandatsart sowie Mandatsstatus und wählen Sie jeweils die Option Wiederkehrend aus.

    So richten Sie den automatisieren Mailversand für Debitorenzahlungen ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Zgl. Export Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Geben Sie im Inforegister Avise, Gruppe Mailversand im Feld E-Mail Absender eine E-Mail Adresse ein.
    3. Klicken Sie in das Feld Mahnmethodencode für Avis-E-Mail-Text und wählen Sie die Option Avis aus.
    4. Klicken Sie in das Feld Avisversand Automatik und wählen Sie die Option Nur E-Mail aus.

    So hinterlegen Sie Empfängerinformationen in der OPplus Debitorenkarte

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld OPplus Debitorenkarte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Zlg. Export in das Feld Avisversandart und wählen Sie die Option E-Mail aus.
    3. Klicken Sie in das Feld Avisversandadresse und wählen Sie einen Datensatz aus.

    So richten Sie eine Zahlungsbuchungsmatrix für SEPA ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Zahlungsbuchungsmatrix ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie in das Feld Zahlungsartencode und wählen Sie den Datensatz SEPA aus.
    4. Klicken Sie in das Feld Bankkonto und wählen Sie ein Bankkonto aus.
    5. Klicken Sie in das Feld Buchungskontoart und wählen Sie die Option Sachkonto aus.
    6. Klicken Sie in das Feld Buchnungkontonr. und wählen Sie einen Datensatz aus.
    7. Geben Sie im Feld Kontoname eine Kurzbeschreibung ein (zum Beispiel Geldtransit).
    Hinweis

    Sie können für den Zahlungsartencode SEPA einen Avis-Text einrichten. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Zahlungsarten ein und klicken Sie auf das übereinstimmende Suchergebnis. Im Feld Text Verwendungszweck Debitor können Sie einen entsprechenden Text eingeben.
