    Mapping von Einheiten einrichten

    Allen Einheiten aus ZUGFeRD-Rechnungen werden den Einheiten in gevis ERP | VEO zugeordnet. Für abweichende Einheiten sind zusätzliche Transformationsregeln einzurichten.

    So richten Sie das Einheiten Mapping in der ZUGFeRD Einrichtung ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld ZUGFeRD Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Import Zuordnungen.
    3. Klicken Sie im Menüband auf die Schaltfläche Neu.
    4. Klicken Sie in das Feld Tabellen-Nr. und wählen Sie die Objekt-ID 204 für die Tabelle Einheit aus.
    5. Klicken Sie in das Feld Feld-Nr. und wählen Sie über das Dreipunkt-Menü die Nr. 1 für den Feldnamen Code aus. Bestätigen Sie Ihre Auswahl mit OK.
    6. Klicken Sie in das Feld Interner Wert und wählen Sie über das Dreipunkt-Menü eine Einheit in gevis ERP | VEO aus. Bestätigen Sie die Auswahl mit OK.
    7. Geben Sie im Feld Externer Wert den Wert der Einheit aus ZUGFeRD-Belegen ein, den Sie dem internen Wert zuordnen wollen. Bestätigen Sie Ihre Eingabe mit Enter.
    8. Bestätigen Sie die Änderung im Dialogfeld mit Ja.

    So richten Sie eine Transformationsregelkarte für ZUGFeRD ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Transformationsregeln ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Inforegister Allgemein im Feld Code eine Bezeichnung (zum Beispiel ZUGFERDUNIT) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel ZUGFeRD Einheit) ein.
    4. Klicken Sie in das Feld Transformationstyp und wählen Sie die Option Benutzerdefiniert aus.

    5. Klicken Sie in das Feld Erw. Benutzerdefinierter Typ und wählen Sie die Option Lookup aus.

      Hinweis

      Sobald Sie die Option Lookup ausgewählt haben, wird das Inforegister Lookup eingeblendet.

    6. Geben Sie im Inforegister Lookup im Feld Tabellenname den Wert 5561706 ein für die Tabelle IMA Import Zuordnung.
    7. Wählen Sie im Feld Feldname über das das Dreipunkt-Menü den Datensatz Externer Wert aus und bestätigen Sie Ihre Auswahl mit OK.
    8. Wählen Sie im Feld Rückgabefeldname über das Dreipunkt-Menü den Datensatz Interner Wert aus und bestätigen Sie Ihre Auswahl mit OK.
    9. Klicken Sie neben dem Feld Anzahl Filter auf das Dreipunkt-Menü, um die Seite Filter zu öffnen.
    10. Klicken Sie im Menüband auf + Filter, wählen Sie die Nummer 2 aus und bestätigen Sie Ihre Auswahl mit OK.

    11. Wiederholen Sie den Schritt wie in Punkt 10 beschrieben für die Nummern 4 und 6, da bei einer Mehrfachauswahl nur der letzte Schritt übernommen wird.

      Hinweis

      Auf der Seite Filter sind jetzt im Feld Feldname die Werte Tabellen-Nr., Zuordnungs-Typ und Feld-Nr. eingeblendet.

    12. Markieren Sie die Zeile mit dem Feldnamen Zuordnungs-Typ.
    13. Klicken Sie in das Feld Filter und wählen Sie über das Dreipunkt-Menü den Wert ZUGFeRD aus.

    14. Schließen Sie die Seite Filter über die Schaltfläche Schließen.

      Hinweis

      Die eingerichtete Transformationsregel ist in der aktuellen Datenaustauschdefinition hinterlegt und wird beim Import einer ZUGFeRD-Rechnung ausgeführt.
