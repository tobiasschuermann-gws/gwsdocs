    Unterstützte Einrichtung für ZUGFeRD

    Für die Einrichtung von ZUGFeRD steht Ihnen ein Assistent auf der Seite Unterstütztes Setup zur Verfügung.

    So richten Sie ZUGFeRD mithilfe der unterstützten Einrichtung ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Unterstütztes Setup ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie in der Ebene ZUGFeRD im Feld Titel auf den Wert Assistent zur Einrichtung von ZUGFeRD.
    3. Klicken Sie auf die Schaltfläche Nächste, um den Import einzurichten.
    4. Klicken Sie in das Feld Kreditorenidentifizierung und wählen Sie den Datensatz aus oder klicken Sie auf Neu, um einen neuen Datensatz anzulegen.
    5. Klicken Sie in das Feld Artikelidentifizierung und wählen Sie den Datensatz aus oder klicken Sie auf Neu, um einen neuen Datensatz zu anzulegen.
    6. Klicken Sie in das Feld Frachtkosten Art und wählen Sie eine Option aus: Artikel (ohne Bestand), Ressource oder Artikel Zu-/Abschlag.
    7. Klicken Sie in das Feld Frachtkosten Nr. und wählen Sie einen Datensatz aus oder klicken Sie auf Neu, um einen neuen Datensatz anzulegen. Weitere Informationen finden Sie unter Import einrichten.
    8. Aktivieren Sie optional den Schalter Rechnung mit geb. Lieferung verknüpfen, wenn eine automatische Zuordnung der erstellten Einkaufsrechnungszeilen zu Einkaufslieferungszeilen erfolgen soll.
    9. Geben Sie optional in den Feldern Max. Toleranz für Gesamtbetrag inkl. MwSt. und Max. Toleranz für Gesamt MwSt. einen Wert ein. Die importierte Einkaufsrechnung oder Einkaufsgutschrift wird dann nur freigegeben, wenn der Toleranzbereich für die ausgewählten Gesamtbeträge nicht überschritten ist.
    10. Klicken Sie auf die Schaltfläche Nächste, um den Export einzurichten.
    11. Geben Sie im Feld Token URL die URL des Autorisierungsservers ein, der das Token ausstellt.
    12. Geben Sie im Feld Client ID die ID ein, die zur Autorisierung verwendet wird.
    13. Geben Sie im Feld Client Secret das Geheimnis für die Autorisierung ein.
    14. Geben Sie im Feld API Scope den Verweis auf den API-Bereich ein, der die Berechtigungen definiert, die für den Zugriff auf die API erforderlich sind. Der Wert im Feld API Scope sollte im Format einer URL oder eines spezifischen Identifikators vorliegen, der vom API-Anbieter bereitgestellt wird (zum Beispiel api://12345678-1234-1234-1234-1234567890ab/.default).
    15. Geben Sie im Feld API Endpunkt Export die URL des Export-Webservices ein.
    16. Klicken Sie auf die Schaltfläche Nächste.
    17. Aktivieren Sie optional den Schalter Aufgabenwarteschlange aktivieren, wenn fehlgeschlagene oder noch nicht übermittelte Einträge (nochmals) verarbeitet werden sollen.
    18. Aktivieren Sie optional den Schalter Status auf 'Bereit' festlegen.
    19. Klicken Sie auf die Schaltfläche Nächste.
    20. Klicken Sie auf die Schaltfläche Ende, um die Einrichtung zu speichern.
