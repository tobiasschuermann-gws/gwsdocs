    Generische Stapel

    Mithilfe von generischen Stapeln können Sie Stapel passend zu individuellen Aufgaben erstellen. Generische Stapel sind eine Ergänzung zu den allgemeinen Stapeln und können für jede Tabelle eingerichtet werden. Für jeden Benutzer können bis zu zehn generische Stapel erstellt werden.

    So richten Sie generische Stapel als Administrator ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Generische Stapeleinrichtung - Administrator ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie optional in das Feld Profil und wählen Sie einen Datensatz aus (zum Beispiel PROJECT MANAGER), um einen generischen Stapel für ein bestimmtes Profil einzurichten.

    4. Klicken Sie optional in das Feld Benutzername und wählen Sie einen Datensatz aus, um einen generischen Stapel für einen bestimmten Benutzer einzurichten.

      Hinweis

      Generische Stapel können variabel eingerichtet werden:

      • Wenn Sie die Felder Profil und Benutzername leer lassen, können Sie generische Stapel für alle Benutzer einrichten.
      • Wenn Sie im Feld Profil zum Beispiel die Option PROJECT MANAGER auswählen und das Feld Benutzername leer lassen, werden generische Stapel für alle Projektleiter erstellt.
      • Sie können innerhalb einer Profilgruppe für bestimmte Benutzer gesonderte generische Stapel einrichten. Wählen Sie in diesem Fall im Feld Benutzername die passenden Datensätze aus.
      • Wenn einem Anwender mehrere Profile zugeordnet sind (zum Beispiel PROJECT MANAGER und SECURITY ADMINISTRATOR), können Sie pro Profil unterschiedliche generische Stapel einrichten. Wählen Sie in diesem Fall sowohl im Feld Profil als auch im Feld Benutzername einen Datensatz aus.

    5. Richten Sie generische Stapel über diese Felder ein:

      Feld Beschreibung
      Stapel ID Klicken Sie in das Feld und wählen Sie eine ID aus. Mit der Auswahl der Stapel ID legen Sie fest, in welcher Reihenfolge die generischen Stapel angezeigt werden.
      Tabellen ID Klicken Sie in das Feld und wählen Sie eine Tabelle aus, für die im generischen Stapel der Wert berechnet werden soll.
      Tabellenname Gibt den Namen der Tabelle an, die im Feld Tabellen ID ausgewählt wurde.
      Kalkulationsmethode Klicken Sie in das Feld und wählen Sie eine Option aus: Anzahl oder Summe.
      Feldnummer Klicken Sie in das Feld und wählen Sie eine Feldnummer aus, die dem Stapel zugeordnet ist.
      Feldname Gibt den Namen des Feldes an, das im Feld Feldnummer ausgewählt wurde.
      Stapel Anzeigename Geben Sie in diesem Feld den Namen des Stapels ein, der angezeigt werden soll.
      Anzahl Stapelfilter Gibt die Anzahl der Filter für die Berechnung des Stapels an. Klicken Sie in das Feld oder alternativ im Menüband auf die Schaltfläche Stapelfilter, wenn Sie einen Filter ergänzen wollen. Klicken Sie auf der Seite Filter auf die Schaltfläche Filter, wählen Sie einen Datensatz aus und bestätigen Sie Ihre Auswahl mit OK.
      Darstellung unterer Bereich Klicken Sie in das Feld und wählen Sie eine Option für die Farbe des Indikators aus. Weitere Informationen finden Sie unter Farbindikatoren für Stapel einrichten. Die Farbe des Indikators wird angezeigt, wenn der Wert der Daten im Stapel kleiner ist als der Wert, der im Feld Schwellenwert 1 angegeben ist.
      Schwellenwert 1 Geben Sie einen Wert ein, ab dem sich bei niedrigeren Werten als dem hier eingegebenen die Farbe des Indikators in die im Feld Darstellung unterer Bereich angegebene Farbe ändert.
      Darstellung mittlerer Bereich Klicken Sie in das Feld und wählen Sie eine Option für die Farbe des Indikators aus. Weitere Informationen finden Sie unter Farbindikatoren für Stapel einrichten. Die Farbe des Indikators wird angezeigt, wenn der Wert der Daten im Stapel zwei Bedingungen erfüllt: Er entspricht dem Wert im Feld Schwellenwert 1 oder er ist größer. Zudem ist er nicht größer als der Wert im Feld Schwellenwert 2.
      Schwellenwert 2 Geben Sie einen Wert ein, ab dem sich bei größeren Werten als dem hier eingegebenen die Farbe des Indikators in die im Feld Darstellung hoher Bereich angegebene Farbe ändert.
      Darstellung hoher Bereich Klicken Sie in das Feld und wählen Sie eine Option für die Farbe des Indikators aus. Weitere Informationen finden Sie unter Farbindikatoren für Stapel einrichten. Die Farbe des Indikators wird angezeigt, wenn der Wert im Stapel größer als der Wert im Feld Schwellenwert 2 ist.

    6. Setzen Sie optional im Feld Seite modal öffnen einen Haken, wenn Seiten aus einem generischen Stapel heraus im Bearbeitungsmodus geöffnet werden sollen. Sie können in diesem Fall direkt auf der geöffneten Seite Änderungen vornehmen. Modal geöffnete Seiten schließen Sie wieder über die Schaltfläche Schließen.

      Hinweis

      Das Feld Seite modal öffnen ist sehr wichtig für die Rechnungsfreigabeprüfung. Nur wenn der Haken gesetzt ist, können auch aus einem generischen Stapel heraus aufgerufene Rechnungsfreigabezeilen direkt bearbeitet werden. Weitere Informationen finden Sie unter Rechnungsfreigabeprüfung.

    So richten Sie generische Stapel als Benutzer ein

    Als Benutzer können Sie die vom Administrator eingerichteten generischen Stapel individuell anpassen und zum Beispiel einen Schwellenwert oder die Auswahl der Tabelle ändern. Einen weiteren Stapel kann nur der Administrator für Sie einrichten.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Generische Stapeleinrichtung - Benutzer ein und klicken Sie auf das übereinstimmende Suchergebnis.

    2. Markieren Sie eine Zeile und passen Sie den generischen Stapel bei Bedarf über diese Felder an:
