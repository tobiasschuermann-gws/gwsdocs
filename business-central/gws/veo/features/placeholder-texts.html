    Platzhaltertexte

    Mit dieser Funktion können Sie Platzhaltertexte in Ihren Verkaufs- und Servicebelegen ausgeben.

    So aktivieren Sie Platzhaltertexte

    1. Geben Sie im Suchfeld (Symbol Lupe im Menüband) Einrichtung Ein-/Ausschalter ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister ERP, Gruppe Verkauf & Marketing den Schalter Platzhaltertext für Verkaufs- und Servicebelege verwenden, um eine automatisch individualisierte Benennung (zum Beispiel für Lieferungen einer Sammelrechnung) hinterlegen zu können.

    So richten Sie Platzhaltertexte ein

    So richten Sie Platzhaltertexte für den Verkauf ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Debitoren und Verkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Verkaufsrechnung in das Feld Textcode für Verkaufsrechnung.
    3. Klicken Sie in der Drop-Down-Liste auf Neu, um einen neuen Datensatz anzulegen.
    4. Geben Sie im Feld Code eine Bezeichnung und im Feld Beschreibung eine Kurzbeschreibung ein.
    5. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Platzhalter Einrichtung.
    6. Klicken Sie im Feld Feldnr. auf das Dreipunkt-Menü und wählen Sie eine Zeile.
    7. Bestätigen Sie Ihre Auswahl mit OK.

    So richten Sie Platzhaltertexte für den Service ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Serviceeinrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Servicerechnung in das Feld Textcode für Servicerechnung auf Neu, um einen neuen Datensatz anzulegen.
    3. Geben Sie im Feld Code eine Bezeichnung und im Feld Beschreibung eine Kurzbeschreibung ein.
    4. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Platzhalter Einrichtung.
    5. Klicken Sie im Feld Feldnr. auf das Dreipunkt-Menü und wählen Sie eine Zeile.
    6. Bestätigen Sie Ihre Auswahl mit OK.

    So hinterlegen Sie Platzhaltertexte

    So hinterlegen Sie Platzhaltertexte für den Verkauf

    Sie haben die Möglichkeit, Platzhaltertexte in den Belegen der Verkaufsrechnung, Verkaufsreklamation und Verkaufsgutschrift auszugeben.
    Hier am Beispiel für Verkaufsrechnungen:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Debitoren und Verkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Verkaufsrechnung in das Feld Textcode für Verkaufsrechnung wählen Sie einen Platzhaltertext aus.
    Hinweis
    • Wenn eine Verkaufsrechnung aus einer Verkaufslieferung erstellt wird, füllt das System automatisch das Feld Textcode für Verkaufsrechnung im Inforegister Allgemein der Debitorenkarte. Wenn das Feld Textcode für Verkaufsrechnung in der Debitorenkarte leer ist, wird stattdessen der Textcode von der Seite Einrichtung Debitoren und Verkauf verwendet.
    • Weitere Platzhaltertexte können Sie in den Inforegistern Verkaufsreklamation und Verkaufsgutschrift hinterlegen.

    So hinterlegen Sie Platzhaltertexte für den Service

    Sie haben die Möglichkeit, Platzhaltertexte in den Belegen der Servicerechnung und des Servicevertrags auszugeben.
    Hier am Beispiel für Servicerechnungen:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Serviceeinrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Servicerechnung in das Feld Textcode für Servicerechnung wählen Sie einen Platzhaltertext aus.
    Hinweis

    Weitere Platzhaltertexte können Sie im Inforegister Verträge hinterlegen.
