    Text Editor

    Mit dem Text Editor können Sie bequem Texte über mehrere Zeilen erfassen und auch längere Texte in den Text Editor kopieren. Die Texte werden nach der Bestätigung der Eingabe automatisch auf die passende Zeichenlänge in den Textbausteinen und Bemerkungen umgebrochen.

    Hinweis

    Den Text Editor finden Sie in gevis ERP | VEO überall dort, wo Sie einen längeren Text eingeben können.

    Beispiele

    So öffnen Sie den Text Editor, wenn Sie in den Belegen einen Textbaustein anlegen

    1. Öffnen Sie einen Beleg.
    2. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Dokument Vor-/Nachtexte > Dokument > Vortexte hinzufügen oder Nachtexte hinzufügen.
    3. Klicken Sie im Menüband auf die Schaltfläche Neu > Text Editor.
    4. Geben Sie einen Text ein und bestätigen Sie Ihre Eingaben mit OK.

    So öffnen Sie den Text Editor, wenn Sie in den Belegzeilen eine Bemerkung anlegen

    1. Öffnen Sie einen Beleg.
    2. Markieren Sie eine Belegzeile.
    3. Klicken Sie im Menüband des Inforegisters Zeilen auf die Schaltfläche Zeile > (Funktionen >) Text Editor.
    4. Geben Sie einen Text ein und bestätigen Sie Ihre Eingaben mit OK.

    So öffnen Sie den Text Editor, wenn Sie in der Artikelkarte einen Textbaustein anlegen

    1. Öffnen Sie eine Artikelkarte.
    2. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Artikel > Textbausteine.
    3. Klicken Sie auf die Schaltfläche + Neu, um einen neuen Datensatz anzulegen.
    4. Klicken Sie im Menüband des Inforegisters Zeilen auf die Schaltfläche Zeile > Text Editor.
    5. Geben Sie einen Text ein und bestätigen Sie Ihre Eingaben mit OK.

    So öffnen Sie den Text Editor, wenn Sie in den Stammdaten (zum Beispiel Debitorenkarte) eine Bemerkung anlegen

    1. Öffnen Sie eine Debitorenkarte.
    2. Klicken Sie im Menüband auf die Schaltfläche Debitor > Bemerkungen.
    3. Klicken Sie im Menüband auf die Schaltfläche Aktionen > Text > Text Editor.
    4. Geben Sie einen Text ein und bestätigen Sie Ihre Eingaben mit OK.
