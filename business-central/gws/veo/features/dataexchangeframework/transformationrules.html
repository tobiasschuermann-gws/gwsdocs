    Transformationsregeln einrichten

    Um den jeweils korrekten Wert aus Ihren IMA Zuordnungen in Ihre EDI-Belege übertragen zu können, benötigen Sie benutzerdefinierte Transformationsregeln.

    So richten Sie Transformationsregeln ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Transformationsregeln ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie eine Transformationsregel, die sie ändern möchten.
    3. Klicken Sie im Menüband auf die Schaltfläche Neu, um eine neue Transformationsregel zu erstellen. Alternativ klicken Sie auf die Schaltfläche Verwalten > Bearbeiten, um eine vorhandene Transformationsregel zu bearbeiten.
    4. Geben Sie im Inforegister Allgemein im Feld Code eine Bezeichnung und im Feld Beschreibung eine Kurzbeschreibung ein.
    5. Klicken Sie in das Feld Transformationstyp und wählen Sie die Option Benutzerdefiniert aus.
    6. Klicken Sie optional in das Feld Nächste Transformationsregel und wählen Sie eine Transformationsregel aus, die im Anschluss ausgeführt werden soll.
    7. Klicken Sie in das Feld Erw. Benutzerdefinierter Typ und wählen Sie die Option Lookup aus.
    8. Geben Sie im Inforegister Lookup im Feld Tabellenname eine Tabellennummer ein, in der Sie einen Wert nachschlagen wollen (zum Beispiel den Wert 5561706 für die Tabelle IMA Import Zuordnung oder den Wert 5561705 für die Tabelle IMA Export Zuordnung).
    9. Geben Sie im Feld Feldname eine Feldnummer ein, welche mit dem Wert der jeweils gewählten Spalte gefiltert werden soll, um einen Datensatz zu identifizieren (zum Beispiel den Wert 9 für das Feld Externer Wert).
    10. Geben Sie im Feld Rückgabefeldname eine Feldnummer ein (zum Beispiel den Wert 8 für das Feld Interner Wert).
    11. Klicken Sie neben dem Feld Anzahl Filter auf das Dreipunkt-Menü, um Filter für die angegebene Tabelle zu definieren. Um einen neuen Filter hinzuzufügen, klicken Sie im Menüband auf die Schaltfläche Filter, markieren Sie eine Zeile und bestätigen Sie ihre Auswahl mit OK. Geben Sie im Feld Filter Ihren Filtertext ein und beenden Sie Ihre Einrichtung mit Schließen.
    12. Aktivieren Sie optional den Schalter Leerer Text bei fehlerhaften Lookup wenn keine Fehlermeldung, sondern ein leerer Text zurückgegeben werden soll, wenn kein Datensatz in der Zieltabelle gefunden wird.
    13. Klicken Sie in das Feld Lookup Operation und wählen Sie eine Option aus, was passieren soll, wenn mehrere Datensätze in der Zieltabelle gefunden werden.

    So prüfen Sie Ihre Transformationsregeln auf Funktionalität

    1. Geben Sie im Inforegister Prüfen in das Feld Testtext einen Wert ein, für den Sie den Rückgabewert überprüfen möchten.
    2. Klicken Sie auf die Schaltfläche Aktualisieren, um die Überprüfung zu starten. Das Ergebnis des Tests wird in das Feld Ergebnis eingetragen. Wird kein Datensatz in der Zieltabelle gefunden, wird entweder eine Fehlermeldung ausgegeben oder es wird ein leerer Text in das Feld eingetragen, wenn der Schalter Leerer Text bei fehlerhaften Lookup aktiviert wurde.
