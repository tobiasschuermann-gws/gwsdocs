    Kalkulationsformeln einrichten

    Während des Imports von Daten haben Sie die Möglichkeit, Werte aus anderen Feldern in ein anderes Feld mit Hilfe einer Kalkulationsformel zu übertragen. Sie definieren somit, wie das jeweilige Feld während des Imports berechnet wird.

    So richten Sie die Kalkulationsformel ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Datenaustauschdefinitionen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie eine Datenaustauschdefinition und klicken Sie in das Feld Code, um diese zu öffnen.
    3. Markieren Sie im Inforegister Spaltendefinition eine Zeile und geben Sie im Feld Kalkulationsformel eine Formel ein.
    4. Klicken Sie im Menüband des Inforegisters Spaltendefinition auf die Schaltfläche Verwalten > Prüfe Kalkulationsformel, um Ihre eingegebene Kalkulationsformel auf Funktionalität zu überprüfen.
    Hinweis

    Für die Eingabe von Kalkulationsformeln können Sie die Syntax von Excel-Formeln verwenden. Die folgenden Operatoren werden unterstützt:

    • = (gleich)
    • <> (ungleich)
    • IF (wenn)

    Beispiel
    Wenn Sie im Inforegister Spaltendefinition drei Zeilen mit den Spaltennummern 10000, 20000 und 30000 eingerichtet haben, könnte die Kalkulationsformel folgendermaßen aussehen: =IF([10000]<>'';[10000];[20000] + " " + [30000])
    Übersetzt bedeutet diese Formel: Wenn der Wert in Spaltennummer 10000 ungleich leer ist, dann schreibe den Wert aus 10000 in die Spalte, sonst schreibe den Wert aus 20000 in die Spalte, hänge ein Leerzeichen an und hänge den Wert aus 30000 an.
