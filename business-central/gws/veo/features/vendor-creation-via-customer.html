    Kreditorenerstellung über die Debitorenkarte

    Über die Debitorenkarte kann ein in OPplus verknüpfter Kreditor erstellt werden. Diese Funktion erleichtert die Anlage eines zugehörigen Kreditors, da die Stammdaten direkt aus der Debitorenkarte übernommen werden können.

    So richten Sie einen Kreditor ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kreditoren Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.

    2. Richten Sie im Inforegister Kreditorenerstellung aus Debitor den Kreditor über diese Felder und Schalter ein:

      Feld / Schalter Beschreibung
      Adressdaten vererben Aktivieren Sie diesen Schalter, wenn die Adressdaten bei der Kreditorenerstellung von der Debitorenkarte übernommen werden sollen.
      Kontaktdaten vererben Aktivieren Sie diesen Schalter, wenn die Kontaktdaten bei der Kreditorenerstellung von der Debitorenkarte übernommen werden sollen.
      Primäre Kontaktnr. vererben Aktivieren Sie diesen Schalter, wenn die primäre Kontaktnummer bei der Kreditorenerstellung von der Debitorenkarte übernommen werden soll.
      Verkäufercode vererben Aktivieren Sie diesen Schalter, wenn der Verkäufercode bei der Kreditorenerstellung in den Einkäufercode übernommen werden soll.
      Bankkonto vererben Aktivieren Sie diesen Schalter, wenn das Bankkonto bei der Kreditorenerstellung von der Debitorenkarte übernommen werden soll.
      Geschäftsbuchungsgruppe Klicken Sie in das Feld und wählen Sie eine Option aus: EU, EXPORT oder INLAND. Die Auswahl gibt an, welche Geschäftsbuchungsgruppe bei der Kreditorenerstellung von der Debitorenkarte verwendet werden soll.
      Kreditorenbuchungsgruppe Klicken Sie in das Feld und wählen Sie eine Option aus: EU, EXPORT oder INLAND. Die Auswahl gibt an, welche Kreditorenbuchungsgruppe bei der Kreditorenerstellung von der Debitorenkarte verwendet werden soll.
      Kred.Nr. basierend auf Deb.Nr. Aktivieren Sie diesen Schalter, wenn die Nummer des neuen Kreditors auf der Debitorennummer basieren soll.
      Debitoren Muster (wird ersetzt) Klicken Sie in das Feld und geben Sie einen Teil der Debitorennummer ein, wenn dieser in der neuen Kreditorennummer ersetzt werden soll.
      Kreditor Muster (ersetzt Deb. Muster) Klicken Sie in das Feld und geben Sie einen Ersatztext ein, wenn der zuvor definierte Teil der Debitorennummer ersetzt werden soll.
      EK-Preis inkl. MwSt. Klicken Sie in das Feld und wählen Sie eine Option aus: Nicht setzen, Setzen oder Aus Debitor übernehmen. Die Auswahl gibt an, wie die Mehrwertsteuer in Einkaufspreise bei der Kreditorenerstellung von der Debitorenkarte berücksichtigt wird.
      Zahlungsbedingungscode vererben Aktivieren Sie diesen Schalter, wenn der Zahlungsbedingungscode bei der Kreditorenerstellung von der Debitorenkarte übernommen werden soll.
      Zahlungsformcode vererben Aktivieren Sie diesen Schalter, wenn der Zahlungsformcode bei der Kreditorenerstellung von der Debitorenkarte übernommen werden soll.
      Kreditor Zahlungsformcode Klicken Sie in das Feld und wählen Sie einen Datensatz aus. Die Auswahl gibt an, wie die Mehrwertsteuer in Einkaufspreise bei der Kreditorenerstellung von der Debitorenkarte berücksichtigt wird.

    3. Aktivieren Sie optional im Inforegister Kreditorenbemerkungen einen oder mehrere Schalter, um Bemerkungen auf einem Einkaufsbeleg auszugeben. Weitere Informationen finden Sie unter So richten Sie ein, in welchen Einkaufsbelegen die Bemerkungen zu sehen sind.

    So erstellen Sie einen verknüpften Kreditor

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Debitoren ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Debitorenkarte.
    3. Klicken Sie im Menüband auf die Schaltfläche Aktionen > Verknüpften Kreditor erstellen.
    4. Bestätigen Sie das Dialogfenster mit OK.
    Hinweis

    Der verknüpfte Kreditor enthält alle Informationen, die Sie in So richten Sie einen Kreditor ein eingerichtet haben, wobei der Name des Debitors übernommen wird.
