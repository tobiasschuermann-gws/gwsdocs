    Erweiterte Druckerauswahl

    Mit der erweiterten Druckerauswahl haben Sie die Möglichkeit, dem Benutzer einen Drucker an einem Lagerort zuzuordnen. Zusätzlich können Sie die Auswahl des Druckers so einrichten, dass das Profil berücksichtigt wird.

    Hinweis

    Für die erweiterte Druckerauswahl auf der Seite Druckerauswahlen muss ein Datensatz (Benutzer-ID oder Berichts-ID) ausgewählt sein. Klicken Sie alternativ in das Feld Druckername, um die Schaltfläche Erweiterte Druckerauswahl nutzen zu können.

    So ordnen Sie einem Benutzer einen Drucker an einem Lagerort zu

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Druckerauswahlen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie in das Feld Benutzer-ID und wählen Sie einen Datensatz aus. Lassen Sie alternativ das Feld leer, wenn ein Drucker an einem bestimmten Lagerort allen Benutzern zugeordnet werden soll.
    4. Klicken Sie optional in das Feld Berichts-ID und wählen Sie einen Datensatz aus, wenn Sie nur für eine bestimmte Berichts-ID die erweiterte Druckerauswahl einrichten wollen.
    5. Klicken Sie optional in das Feld Druckername, wenn Sie weder eine Benutzer-ID noch eine Berichts-ID ausgewählt haben.
    6. Klicken Sie im Menüband auf die Schaltfläche Erweiterte Druckerauswahl.
    7. Klicken Sie in das Feld Lagerortcode und wählen Sie einen Datensatz aus.
    8. Klicken Sie in das Feld Druckername und wählen Sie einen Datensatz aus.

    So ordnen Sie einem Profil einen Drucker an einem Lagerort zu

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Druckerauswahlen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie in das Feld Benutzer-ID und wählen Sie einen Datensatz aus, wenn Sie für einen Benutzer mit unterschiedlichen Profilen die erweiterte Druckerauswahl einrichten wollen. Lassen Sie alternativ das Feld leer, wenn ein Drucker an einem Lagerort allen Benutzern mit einem bestimmten Profil zugeordnet werden soll.
    4. Klicken Sie optional in das Feld Berichts-ID und wählen Sie einen Datensatz aus, wenn Sie nur für eine bestimmte Berichts-ID die erweiterte Druckerauswahl einrichten wollen.
    5. Klicken Sie optional in das Feld Druckername, wenn Sie weder eine Benutzer-ID noch eine Berichts-ID ausgewählt haben.
    6. Klicken Sie im Menüband auf die Schaltfläche Erweiterte Druckerauswahl.
    7. Klicken Sie in das Feld Lagerortcode und wählen Sie einen Datensatz aus.
    8. Klicken Sie in das Feld Profil Anzeigenname.
    9. Markieren Sie eine verfügbare Rolle und bestätigen Sie Ihre Auswahl mit OK.
    10. Klicken Sie in das Feld Druckername und wählen Sie einen Datensatz aus.
    Hinweis

    Für die Druckerauswahl hat die Zuordnung eines Profils die höchste Priorität. Das heißt, in der erweiterten Druckerauswahl wird zunächst der Drucker ausgewählt, dem ein Profil (und ein Lagerortcode) zugeteilt ist. Gibt es keinen Eintrag im Feld Profil Anzeigenname wird der Drucker ausgewählt, der einem Lagerort zugeordnet ist.
