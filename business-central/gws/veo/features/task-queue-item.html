    Aufgabenwarteschlangenposten

    Mithilfe von Aufgabenwarteschlangenposten können Sie Prozesse planen und anstoßen. Die Prozesse können einmalig oder automatisch wiederholt ausgeführt werden (zum Beispiel gebuchte Verkaufsaufträge senden oder Direktlieferungen abwickeln).

    So erstellen Sie einen Aufgabenwarteschlangenposten

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Aufgabenwarteschlangenposten ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.

    3. Legen Sie im Inforegister Allgemein die grundlegenden Eigenschaften fest:

      Feld Hinweis Beschreibung Beispiel
      Art des auszuführenden Objekts Pflichtfeld Art des Objekts, das durch den Aufgabenwarteschlangenposten ausgeführt werden soll. Bericht, CodeUnit
      ID des auszuführenden Objekts Pflichtfeld ID des konkreten Objekts, das durch den Aufgabenwarteschlangenposten ausgeführt werden soll. 5374391
      Beschreibung Wird automatisch ausgefüllt, kann bei Bedarf angepasst werden. Beschreibung des auszuführenden Objekts. FAVGetSalesOrderviaJobQueue
      Früheste(s) Startdatum/-uhrzeit Nur für wiederkehrende Ausführung Zeitpunkt, zu dem der Aufgabenwarteschlangenposten frühestens ausgeführt werden soll.
      Eingabeformat: TT/MM/JJJJ hh:mm      		 03/05/2023 10:45
      Zeitüberschreitung bei Projekt Maximale Stundenanzahl, die zur Ausführung des Aufgabenwarteschlangenpostens vergehen darf. 12
      Parameterzeichenfolge Weitere Informationen erhalten Sie von Ihrem Berater. Textzeichenfolge, die bei der Ausführung des Aufgabenwarteschlangenpostens als Parameter verwendet wird. Dieser ist im gewählten Objekt hinterlegt.
      Aufgabenwarteschlange - Kategoriencode Code der Aufgabenwarteschlangenkategorie, vorhandenen Datensatz auswählen oder neuen Datensatz anlegen. SACHBUCH - Sachbuchung
      Maximale Anzahl von Ausführungsversuchen Anzahl erneuter Versuche, die nach einer fehlgeschlagenen Ausführung gestartet werden sollen. 5
      Verzögerung der erneuten Ausführung (Sek.) Zeit in Sekunden, bevor ein erneuter Ausführungsversuch gestartet wird. 10
      Ablaufdatum/-uhrzeit Nur für wiederkehrende Ausführung Zeitpunkt, nach dem der Aufgabenwarteschlangenposten nicht mehr ausgeführt werden soll.
      Eingabeformat: TT/MM/JJJJ hh:mm      		 21/05/2023 13:25

    4. Falls Sie im Feld Art des auszuführenden Objekts die Option

      • Bericht gewählt haben, legen Sie im Inforegister Berichtsparameter die Einstellungen für den Bericht fest:

        Feld Beschreibung Beispiel
        Optionen Berichtsanforderungsseite Gibt an, ob Optionen auf der Berichtanforderungsseite für den Berichtsauftrag festgelegt wurden.
        Art der Berichtsausgabe Gibt das Ausgabeformat des Berichts an. PDF, Word, Excel, Drucken, Keine (nur Verarbeitung)
        Druckername Drucker, auf dem der Bericht gedruckt werden soll. Drucker Flur 3. OG

      • CodeUnit gewählt haben, ist das Inforegister Berichtsparameter ausgeblendet.

    So führen Sie einen Aufgabenwarteschlangenposten aus

    Aufgabenwarteschlangenposten können einmalig oder automatisch wiederkehrend ausgeführt werden.

    So starten Sie eine einmalige Ausführung

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Aufgabenwarteschlangenposten ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Karte für Aufgabenwarteschlangenposten.
    3. Klicken Sie auf die Schaltfläche Start > Einmal ausführen (Vordergrund).

    So richten Sie eine automatisch wiederkehrende Ausführung ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Aufgabenwarteschlangenposten ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Karte für Aufgabenwarteschlangenposten.
    3. Legen Sie die wiederkehrende Ausführung entweder über eine Datumsformel oder über definierte Zeitpunkte fest:
      • Datumsformel
        1. Geben Sie im Inforegister Wiederholungen im Feld Datumsformel für nächste Ausführung einen Wiederholrhythmus ein: T=Tag, W=Woche, M=Monat, Q=Quartal, J=Jahr (zum Beispiel 2T, der Aufgabenwarteschlangenposten wird alle zwei Tage ausgeführt).
      • Definierte Zeitpunkte
        1. Aktivieren Sie im Inforegister Wiederholungen die Schalter der Wochentage, an denen der Aufgabenwarteschlangenposten wiederkehrend ausgeführt werden soll (zum Beispiel Montags ausführen, der Aufgabenwarteschlangenposten wird jeden Montag ausgeführt).
        2. Geben Sie im Feld Startzeit eine Uhrzeit (zum Beispiel 08:00:00) ein, zu der die Ausführung des Aufgabenwarteschlangenpostens starten soll.
        3. Geben Sie im Feld Endzeit eine Uhrzeit (zum Beispiel 17:30:00) ein, zu der die Ausführung des Aufgabenwarteschlangenpostens enden soll.
        4. Geben Sie im Feld Anzahl der Minuten zwischen Ausführungen einen Wert ein, wie viele Minuten zwischen den Wiederholungen liegen sollen oder übernehmen Sie den voreingestellten Wert von 1440. Es müssen mindestens fünf Minuten zwischen den Wiederholungen liegen.
    4. Klicken Sie im Menüband auf die Schaltfläche Start > Status auf 'Bereit' festlegen.
