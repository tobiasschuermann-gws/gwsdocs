    EDI Qualifikatoren einrichten

    EDI-Rechnungen (INVOIC) im Ekser-Format können Kopfsummen enthalten. Jede Kopfsumme hat einen Qualifikator (Qualifier) der für eine bestimmte Art steht (zum Beispiel 64 für Frachtkosten). Durch ein Zuordnung von Qualifiern auf die Felder Art und Nummer, können diese in Rechnungszeilen übernommen werden.

    So richten Sie einen EDI Qualifier ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld EDI Qualifier Zuordnungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu, um eine neue Zeile anzulegen.
    3. Geben Sie im Inforegister General im Feld Code eine Bezeichnung und im Feld Beschreibung eine Kurzbeschreibung ein.
    4. Geben Sie im Inforegister EDI Qualifier Zuordnung Zeilen im Feld Qualifier einen Wert (zum Beispiel 64) ein.
    5. Geben Sie im Feld Beschreibung eine Bezeichnung für Ihren Qualifier (zum Beispiel Frachtkosten) ein.
    6. Klicken Sie in das Feld Art und wählen Sie eine Option (zum Beispiel Artikel) aus.
    7. Klicken Sie in das Feld Nr. und wählen Sie einen Datensatz aus.
    Hinweis

    Sie können von der GWS implementierte Qualifier über die Unterstütze Einrichtung für Esker-EDI-Belege anlegen.

    So weisen Sie die erstellten Qualifier einer Importquelle zu

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld EDI Importquellen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie eine Importquelle, klicken Sie in das Feld INVOIC Qualifier Zuordnung Code und wählen Sie einen EDI-Qualifier aus.
