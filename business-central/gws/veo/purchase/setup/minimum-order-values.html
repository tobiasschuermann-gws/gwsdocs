    Mindestbestellwerte einrichten

    Sofern Mindestbestellwerte von Ihren Kreditoren vorgegeben sind, können Sie diese auf der Kreditorenkarte hinterlegen. Falls eine Bestellung den Mindestbestellwert nicht erreicht, kann eine Hinweis- oder Fehlermeldung eingerichtet werden.
    In der Statistik Ihrer Einkaufsbestellung haben Sie die Möglichkeit, die Differenz zum Mindestbestellwert einzusehen, um im Anschluss Ihre Einkaufsbestellung anzupassen.

    So richten Sie einen Mindestbestellwert ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kreditoren ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Kreditorenkarte.
    3. Geben Sie im Inforegister Allgemein im Feld Mindestbestellwert einen Wert ein.
    Hinweis

    Der Mindestbestellwert bezieht sich auf den Gesamtbetrag Ihrer Einkaufsbestellung und wird in der Währung des Kreditoren erfasst und berechnet.

    So richten Sie einen Mindestbestellwert / eine Mindestbestellmenge für einzelne Artikel eines Kreditoren ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kreditoren ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Kreditorenkarte.
    3. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Einkauf > Mindestbestellwerte.
    4. Klicken Sie in das Feld Artikelnr. und wählen Sie einen Artikel aus.
    5. Geben Sie im Feld Mindestbestellwert (MW) und/oder Mindestbestellmenge (Einkaufseinheit) einen Wert ein.

    So richten Sie eine Meldung bei Nichterreichen der Mindestanforderungen ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kreditoren & Einkauf Einr. ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Einkaufsbestellung in das Feld Aktion bei unterschrittenem MBW und wählen Sie eine Art aus:
      • Standard: Es erscheint keine Meldung beim Freigeben oder Buchen der Bestellung.
      • Hinweis: Ist die Mindestanforderung nicht erreicht, erscheint beim Freigeben oder Buchen der Bestellung ein Hinweis. Die Bestellung wird freigegeben / gebucht.
      • Fehler: Ist die Mindestanforderung nicht erreicht, erscheint beim Freigeben oder Buchen der Bestellung eine Fehlermeldung. Die Bestellung wird nicht freigegeben / gebucht.

    So sehen Sie in einer Einkaufsbestellung die Differenz zum Mindestbestellwert ein

    1. Öffnen Sie eine Einkaufsbestellung.
    2. Klicken Sie im Menüband auf die Schaltfläche Bestellung > Statistik.
    3. Im Inforegister Allgemein im Feld Diff. zum Min.-Bestellwert finden Sie die Differenz zum Mindestbestellwert.
