    Unbekannte/Bekannte Direktlieferung buchen

    Grundlegende Informationen finden Sie unter Direktlieferungen machen.

    So richten Sie die Buchung einer Direktlieferung aus einer Einkaufsrechnung ein

    Mit dieser Einrichtung können Sie Einkaufsrechnungen bereits buchen, bevor die zugehörigen Verkaufsrechnungen gebucht wurden.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kreditoren & Einkauf Einr. ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister Auftragsbezogene Bestellungen den Schalter Buchung der EK-Rechnung vor VK-Rechnung zulassen.

    So buchen Sie eine bekannte Direktlieferung

    Bei einer bekannten Direktlieferung bestellt Ihr Debitor direkt bei Ihnen. Die Lieferung erfolgt direkt vom Kreditoren an Ihren Debitoren und berührt Ihr Lager nicht. Sie erhalten vom Kreditoren eine Einkaufsrechnung und können aus dieser Ihre Lieferscheine buchen. Anschließend können Sie die Positionen in die Einkaufsrechnung übertragen und ebenfalls zu buchen.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einkaufsrechnungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Erstellen Sie eine Einkaufsrechnung. Weitere Informationen finden Sie unter Einkaufsrechnung erstellen und buchen.
    3. Klicken Sie im Inforegister Zeilen auf die Schaltfläche Zeile > Funktionen > Wareneingangszeilen abrufen....
    4. Klicken Sie im Menüband auf Schaltfläche Bestellzeilen....

    5. Geben Sie in den Feldern Kred.-Lieferungsnr. und Lieferungsdatum einen Wert ein.

      Hinweis

      Es kann jeweils nur ein Lieferdatum pro zugeordneter Kreditoren-Lieferungsnummer eingegeben werden. Bei der Eingabe mehrerer Lieferdaten wird automatisch das zuletzt eingegebene Lieferdatum verwendet.

    6. Geben Sie optional im Feld Menge akt. Lieferung eine Menge ein.
    7. Klicken Sie im Menüband auf die Schaltfläche Lieferung Buchen und bestätigen Sie das Dialogfenster mit Ja.
    8. Schließen Sie die Seite Einkaufszeilen aus Direktlieferung über die Schaltfläche Schließen.
    9. Markieren Sie die Zeilen und bestätigen Sie Ihre Auswahl mit OK.
    10. Klicken Sie im Menüband auf die Schaltfläche Start > Buchen.

    So buchen Sie eine unbekannte Direktlieferung

    Bei einer unbekannten Direktlieferung bestellt Ihr Debitor direkt bei Ihrem Kreditoren. Die Lieferung erfolgt direkt vom Kreditoren an Ihren Debitoren und berührt Ihr Lager nicht. Damit die Einkaufsrechnung gebucht werden kann, können Sie die benötigten Belege (Einkaufsbestellung und Verkaufsauftrag) aus der Einkaufsrechnung heraus erstellen.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einkaufsrechnungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Erstellen Sie eine Einkaufsrechnung. Weitere Informationen finden Sie unter Einkaufsrechnung erstellen und buchen.
    3. Geben Sie im Inforegister Zeilen in den Feldern Kred.-Lieferungsnr., Debitorennr., Lief. an Code und Lieferungsdatum die Werte der erhaltenen Einkaufsrechnung ein.
    4. Klicken Sie im Menüband auf die Schaltfläche Start > Freigeben.
    5. Klicken Sie im Menüband auf die Schaltfläche Aktionen > Funktion > Direktlieferung > Belege erstellen und buchen.
