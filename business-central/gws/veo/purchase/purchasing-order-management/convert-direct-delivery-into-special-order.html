    Direktlieferung in Spezialauftrag umwandeln

    Wenn bei einer Direktlieferung die Ware versehentlich ins Lager statt an den Kunden geliefert wurde, müssen Sie die bestehende Einkaufsbestellung nicht löschen und neu erstellen. Stattdessen können Sie in der Einkaufsbestellung aus den Direktlieferungszeilen neue Spezialauftragszeilen erstellen. Voraussetzung ist, dass ein passender Einkaufscode vorhanden ist.

    Hinweis

    Sollten Felder oder Schalter in Ihrer Ansicht fehlen, können Sie diese über Personalisieren einblenden. Weitere Informationen finden Sie unter Arbeitsbereich personalisieren.

    So erstellen Sie einen Einkaufscode für die Umwandlung

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einkaufscodes ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Geben Sie im Feld Code eine Bezeichnung (zum Beispiel Umwandlung) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Direktlieferung zu Spezialauftrag) ein.
    3. Setzen Sie im Feld Spezialauftrag einen Haken.
    4. Setzen Sie im Feld Auftragsbezug einen Haken.

    So wandeln Sie eine Direktlieferung in einen Spezialauftrag um

    Hier am Beispiel des Einkaufscodes Umwandlung:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einkaufsbestellungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Einkaufsbestellung für eine Direktlieferung.
    3. Klicken Sie im Menüband auf die Schaltfläche Start > Freigeben > Status zurücksetzen. Der Wert im Inforegister Allgemein im Feld Status wird auf Offen gesetzt.
    4. Klicken Sie im Menüband auf die Schaltfläche Bestellung > Neu erstellen als Spezialauftrag... .

    5. Bestätigen Sie im Dialogfeld die Erstellung von Spezialauftragszeilen mit Ja.

      Hinweis
      • In der Einkaufsbestellung und im Verkaufsauftrag wird im Feld Einkaufscode der Wert Umwandlung gesetzt.
      • In der Einkaufsbestellung und im Verkaufsauftrag wird im Inforegister Zeilen im Feld Direktlieferung der Haken entfernt.
      • Alle Einkaufspreise (auch manuelle Änderungen), Einkaufskonditionen und Rabatte aus den Bestellzeilen werden übernommen.
      • Die Zeilenverknüpfungen und Zeilenzuweisungen werden gelöscht.
      • Der Wert im Feld Lief. an wird nicht geändert.
