    AB-Anmahnung (Auftragsbestätigung bei Anmahnung)

    Wenn Auftragsbestätigungen nicht innerhalb einer vorgegebenen Karenzzeit bei Ihnen ankommen, haben Sie die Möglichkeit, eine AB-Anmahnung (Auftragsbestätigung bei Anmahnung) an Kreditoren zu versenden.

    So richten Sie den Bericht "Einkauf - Bestellung / AB-Anmahnung" in der Berichtsauswahl - Einkauf ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Berichtsauswahl - Einkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie in das Feld Verwendung und wählen Sie die Option AB-Anmahnung aus.
    3. Geben Sie im Feld Berichts-ID den Wert 5374115 ein für den Bericht Einkauf – Bestellung / AB-Anmahnung.
    4. Setzen Sie, falls Sie eine AB-Anmahnung per E-Mail versenden wollen, einen Haken im Feld Für E-Mail-Text verwenden und/oder im Feld Für E-Mail-Anhang verwenden. Wählen Sie zusätzlich im Feld Layout-Beschreibung E-Mail-Text das passende Layout aus, wenn Sie einen Haken im Feld Für E-Mail-Text verwenden gesetzt haben. Weitere Informationen finden Sie unter Erstellen und Ändern benutzerdefinierter Berichtslayouts.

    So richten Sie die AB-Anmahnung ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Ein-/Ausschalter ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister ERP, Gruppe Einkauf den Schalter AB-Anmahnungen aktivieren.
    3. Geben Sie dann im Suchfeld (Symbol Lupe im Menüband) Kreditoren & Einkauf Einr. ein und klicken Sie auf das übereinstimmende Suchergebnis.
    4. Geben Sie im Inforegister Bestellung im Feld Karenzzeit AB-Anmahnung (Tage) die Anzahl der Tage ein, nach denen eine Anmahnung nach dem Versenden einer Bestellung beziehungsweise nach der letzten AB-Anmahnung erfolgen soll.

    Zusätzliche Einrichtungen für das Senden der AB-Anmahnung per E-Mail

    1. Geben Sie in der Kreditoren & Einkauf Einr. im Inforegister Einkaufsbestellung im Feld E-Mail Text für AB-Anmahnung einen Text für die E-Mail (maximal 250 Zeichen) ein.
    2. Geben Sie dann im Suchfeld (Symbol Lupe im Menüband) Mailvorlage Liste ein und klicken Sie auf das übereinstimmende Suchergebnis.
    3. Klicken Sie in das Feld Verwendung und wählen Sie die Option AB-Anmahnung aus.
    4. Geben Sie im Feld Betreffvorlage einen Text als Betreff ein (zum Beispiel AB-Anmahnung).
    5. Klicken Sie optional im Menüband auf die Schaltfläche Mailvorlage hochladen und wählen Sie eine Mailvorlage aus.

    So stellen Sie sicher, dass ein bestimmter Kreditor keine AB-Anmahnung erhält

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kreditoren ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Kreditorenkarte.
    3. Aktivieren Sie im Inforegister Allgemein den Schalter Keine AB-Anmahnung.

    So richten Sie einen Aufgabenwarteschlangenposten für die Verarbeitung von AB-Anmahnungen ein

    Um einen Aufgabenwarteschlangeposten für die Verarbeitung von AB-Anmahnungen zu erstellen, gehen Sie vor wie in Aufgabenwarteschlangenposten beschrieben.

    Hinweis

    Verwenden Sie diese Werte für die Einrichtung:

    • Art des auszuführenden Objekts: Bericht
    • ID des auszuführenden Objekts: 5374115 (Einkauf - Bestellung / AB-Anmahnung)

    So drucken oder senden Sie eine AB-Anmahnung aus der Einkaufsbestellung

    1. Geben Sie im Suchfeld (Symbol Lupe im Menüband) Einkaufsbestellungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Einkaufsbestellung.
    3. Klicken Sie im Menüband auf die Schaltfläche Aktionen > AB-Anmahnung > [Ihre Schaltfläche] (zum Beispiel AB-Anmahnung drucken oder AB-Anmahnung per E-Mail versenden).

    So drucken oder senden Sie eine AB-Anmahnung aus der Übersicht der nicht bestätigten Bestellungen

    1. Geben Sie im Suchfeld (Symbol Lupe im Menüband) Nicht bestätigte Bestellungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie eine nicht bestätigte Bestellung.
    3. Klicken Sie im Menüband auf die Schaltfläche Vorgang > [Ihre Schaltfläche] (zum Beispiel Drucken... oder An E-Mail anhängen).
