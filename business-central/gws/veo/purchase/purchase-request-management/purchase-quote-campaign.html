    Anfragenkampagne

    Sollen Artikel aus einer Kundenanfrage bei mehreren Lieferanten angefragt werden, kann der Vorgang schnell unübersichtlich werden. Für einen besseren Überblick gibt es die Anfragenkampagne, mit der Sie alle zugehörigen Vorgänge verwalten können. Es können beliebig viele Anfragenkampagnen erstellt werden.

    Bevor Sie Anfragenkampagnen nutzen können, müssen Sie diese zunächst aktivieren und eine Nummernserie hinterlegen. Dadurch können die Anfragenkampagnen eindeutig zugeordnet und kategorisiert werden. Sie haben zusätzlich die Möglichkeit, mit der Kachel Anfragenkampagnen einen Schnellzugriff zu den offenen Anfragenkampagnen einzurichten. Bei der Aktivierung der Nummernserie und der Einrichtung des Schnellzugriffs hilft Ihnen bei Bedarf Ihr Berater. Weitere Informationen finden Sie unter Nummerserien erstellen.

    So hinterlegen Sie eine Anfragenkampagnennummer

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kreditoren & Einkauf Einr. ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Nummernserie in das Feld Anfragenkampagnennummern und wählen Sie den Datensatz aus, der für Anfragenkampagnen angelegt wurde.

    So erstellen Sie eine Anfragenkampagne

    Hinweis

    Um eine Anfragenkampagne erstellen zu können, aktivieren Sie zunächst auf der Seite Einrichtung Ein-/Ausschalter im Inforegister ERP, Gruppe Einkauf den Schalter Anfragenkampagnen aktivieren.

    Die Erstellung einer Anfragenkampagne erfolgt in mehreren Schritten. Legen Sie zunächst die Anfragenkampagne an. Ergänzen Sie im nächsten Schritt die Artikel, deren Verfügbarkeit Sie bei einem Lieferanten anfragen wollen. Diese können Sie aus der Artikelliste oder aus bestehenden Angebotszeilen eines Verkaufsangebots auswählen. Wählen Sie anschließend die Kreditoren oder Kontakte aus, an welche die Anfrage versendet werden soll.

    So legen Sie eine Anfragenkampagne an

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Anfragenkampagnen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie im Inforegister Allgemein auf das Dreipunkt-Menü neben dem Feld Nr. und markieren Sie die Zeile mit dem Code, der für Anfragenkampagnen angelegt wurde. Bestätigen Sie Ihre Auswahl mit OK.
    4. Füllen Sie zusätzlich diese Felder aus:
    Feld Beschreibung
    Beschreibung Geben Sie einen Text zur Beschreibung der Anfragenkampagne ein.
    Belegdatum Geben Sie ein Datum ein, an dem die Kampagne gestartet und die Anfragen an die Lieferanten versendet werden sollen.
    Suchbegriff Geben Sie einen Suchbegriff ein. Das Feld wird mit dem Inhalt der Beschreibung vorbelegt, kann aber überschrieben werden.
    Reaktionsdatum Geben Sie ein Datum ein, bis zu welchem eine Antwort der angefragten Lieferanten erwartet wird.
    Status Dieses Feld beschreibt den aktuellen Status der Anfragenkampagne:
    • Erstellt
      Trifft zu, wenn die Kampagne erstellt, die Anfragen jedoch noch nicht an die Lieferanten versendet wurden.
    • Laufend
      Trifft zu, wenn die Anfragen an die Lieferanten versendet wurden.
    • Bestätigt
      Trifft zu, wenn Sie sich für einen Lieferanten entschieden haben, die Bestellung jedoch noch nicht erstellt wurde.
    • Geschlossen
      Trifft zu, wenn die Bestellung erstellt wurde.

    So wählen Sie Artikel aus der Artikelliste für die Anfragenkampagne aus

    1. Klicken Sie im Menüband der Seite Anfragenkampagnen auf die Schaltfläche Artikel auswählen.
    2. Markieren Sie einen oder mehrere Artikel und bestätigen Sie Ihre Auswahl mit OK.
      Die ausgewählten Artikel werden in das Inforegister Zeilen der Anfragenkampagne übertragen.
    3. Geben Sie für alle ausgewählten Artikel im Feld Menge (angefragt) die Menge ein, für die eine Anfrage gestartet werden soll.

    So wählen Sie Artikel aus bestehenden Angebotszeilen für die Anfragenkampagne aus

    1. Klicken Sie im Menüband der Seite Anfragenkampagnen auf die Schaltfläche Angebotszeilen auswählen.
    2. Markieren Sie eine oder mehrere Angebotszeilen und bestätigen Sie Ihre Auswahl mit OK.
      Neben der Artikelnummer, der Beschreibung und der Menge werden aus den Angebotszeilen auch die Werte der Felder Angebotsnummer, Angebotszeilennr. und Angebotspositionsnr. sowie Artikelvarianten übernommen.

    So wählen Sie Kreditoren oder Kontakte für die Anfragenkampagne aus

    1. Klicken Sie im Menüband der Seite Anfragenkampagnen auf die Schaltfläche Kreditor auswählen oder auf die Schaltfläche Kontakt auswählen (es stehen nur Kontakte zu Auswahl, die einen Kreditorenvorlagencode besitzen oder mit einem Kreditor verknüpft sind).
    2. Markieren Sie einen oder mehrere Kreditoren beziehungsweise Kontakte und bestätigen Sie Ihre Auswahl mit OK.
      Die ausgewählten Kreditoren und Kontakte werden in das Inforegister Zeilen übernommen.
    Hinweis

    Für jeden oben ausgewählten Artikel wird im Inforegister Zeilen eine weitere Zeile generiert. Diesmal mit der Lieferantennummer des Kreditors oder Konktakts. Im Feld Menge (angefragt) können Sie den Wert für die Menge, die angefragt werden soll, ändern. Sie können auch eine ganze Zeile löschen, wenn Sie einen bestimmten Artikel bei dem ausgewählten Kreditor nicht anfragen wollen.

    So erstellen und versenden Sie eine Anfrage

    Sie haben die Möglichkeit einzelne Artikel bei bestimmten Kreditoren oder Kontakten anzufragen sowie eine gesammelte Anfrage aller Artikel an alle ausgewählten Kreditoren oder Kontakte zu versenden.

    So erstellen und versenden Sie einzelne Anfragen

    1. Öffnen oder erstellen Sie eine Anfragenkampagne.
    2. Markieren Sie im Inforegister Zeilen eine oder mehrere Artikelzeilen, die einem Kreditor oder einem Kontakt zugeordnet sind. In den Feldern Lieferantennr. und Lieferantenname müssen Werte eingetragen sein.
    3. Klicken Sie im Menüband des Inforegisters Zeilen auf die Schaltfläche Zeile > [Ihre Schaltfläche] (zum Beispiel Anfrage erstellen oder Anfrage erstellen und per E-Mail versenden).

    So erstellen und versenden Sie die Anfragen gesammelt

    1. Öffnen oder erstellen Sie eine Anfragenkampagne.
    2. Markieren Sie im Inforegister Zeilen eine oder mehrere Artikelzeilen, die mehreren Kreditoren oder Kontakten zugeordnet sind. In den Feldern Lieferantennr. und Lieferantenname müssen Werte eingetragen sein.
    3. Klicken Sie im Menüband auf die Schaltfläche Anfragen erstellen > Alle Anfragen erstellen und per E-Mail versenden.
    Hinweis

    Unabhängig davon, ob Sie einzelne Anfragen oder alle Anfragen gesammelt erstellen und versenden, werden im Hintergrund für jeden Kreditor und jeden Kontakt Einzelanfragen erstellt und in den Angebotsanfragen abgelegt. Die Angebotsnummer wird zudem automatisch in das Feld Einkaufsangebotsnr. im Inforegister Zeilen der Anfragenkampagne übertragen.
