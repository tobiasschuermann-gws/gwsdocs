    Debitorenartikel einrichten

    Weisen Sie Ihren Debitoren Artikel oder Artikelkategorien zu, die sie bestellen dürfen. Beim Hinzufügen eines nicht zugewiesenen Artikels zu einem Verkaufsbeleg kann eine Hinweis- oder Fehlermeldung eingerichtet werden. Legen Sie außerdem fest, ob nur zugeordnete Artikel in der Artikelauswahl angezeigt werden.

    So richten Sie Debitorenartikel ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Debitoren ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Debitorenkarte.
    3. Klicken Sie im Menüband auf die Schaltfläche Aktionen > Debitoren Artikelzuordnung.
    4. Klicken Sie im Menüband auf die Schaltfläche Neu.
    5. Wählen Sie im Feld Art die Option Artikel oder Artikelkategorie aus.
    6. Wählen Sie im Feld Code den Artikel oder die Artikelkategorie aus.
    Hinweis

    Die Zuordnung bezieht sich auch auf die Artikelunterkategorien, solange diese nicht explizit ausgeschlossen werden.

    Tipp

    Aktivieren Sie in der Debitorenkarte im Inforegister Allgemein den Schalter Debitorenauswahlfilter ignorieren, damit Debitoren trotz Artikelzuweisung aus dem gesamten Sortiment wählen können.

    So richten Sie eine Meldung für nicht zugeordnete Debitorenartikel ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Debitoren und Verkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Allgemein in das Feld Nur Debitorenartikel verwenden und wählen Sie eine Art aus:
      • Leer: Artikel können ohne Einschränkungen ausgewählt werden.
      • Hinweis: Ein Hinweis wird angezeigt, wenn der Kunde diesen Artikel nicht bestellen darf. Der Artikel wird dem Verkaufsbeleg hinzugefügt.
      • Fehler: Eine Fehlermeldung wird angezeigt, wenn der Kunde diesen Artikel nicht bestellen darf. Der Artikel wird dem Verkaufsbeleg nicht hinzugefügt.
    Hinweis

    Aktivieren Sie im Inforegister Allgemein den Schalter Gefilterte Debitorenartikel-Ansicht, um in der Artikelauswahl nur zugeordnete Artikel anzuzeigen.
