    Erweiterte Debitorensuche

    Mit der erweiterten Debitorensuche können Sie im Verkaufsauftrag und in der Verkaufsrechnung gezielt nach einem Debitor suchen, indem Sie mehrere Begriffe verwenden. Zudem können Sie individuell einrichten, mit welchen Feldern in der Debitorenkarte die Suche abgeglichen werden soll. Auch verknüpfte Informationen wie Lieferadressen und Kontakte können bei der erweiterten Debitorensuche berücksichtigt werden.

    So richten Sie die erweiterte Debitorensuche ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Volltextsuche Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister Erweiterte Debitorensuche den Schalter Aktiviere erweiterte Debitorensuche.
    3. Konfigurieren Sie bei Bedarf zusätzliche Suchfelder. Klicken Sie dazu in einer neuen Tabellenzeile im Feld Feldnr. auf das Dreipunkt-Menü.
    4. Wählen Sie ein Feld aus und bestätigen Sie Ihre Auswahl mit OK.

    5. Klicken Sie im Feld Suchtabellennr. auf das Dreipunkt-Menü, wenn eine mit dem Debitor verknüpfte Tabelle für die erweiterte Debitorensuche berücksichtigt werden soll. Wählen Sie eine Tabelle und bestätigen Sie Ihre Auswahl mit OK.

      Hinweis

      Sollte ein Anwender keine Leserechte für eine Tabelle haben, wird die erweiterte Debitorensuche dadurch nicht abgebrochen. Diese Tabelle wird lediglich nicht durchsucht. So können unterschiedliche Suchergebnisse angezeigt werden, je nach dem welcher Anwender die erweiterte Debitorensuche gestartet hat.

    Tipp

    Im Feld Priorität können Sie jedem Suchfeld eine Priorität zuteilen und so die Darstellung der Suchergebnisse beeinflussen. Je höher die Priorität für ein Suchfeld ist, desto weiter oben in der Liste der Suchergebnisse wird ein Treffer einsortiert.

    Bei der Aktivierung der erweiterten Debitorensuche werden standardmäßig die Suchfelder Nr., Name, Ort, Kontakt, Telefonnr. und PLZ-Code konfiguriert. Sie können diese Konfiguration abändern oder wie oben beschrieben mit zusätzlichen Suchfeldern ergänzen. Änderungen können Sie jederzeit wieder zurücksetzen: Klicken Sie im Menüband des Inforegisters Erweiterte Debitorensuche auf die Schaltfläche Suchfelder > Standardwerte wiederherstellen.

    Hinweis

    Bitte beachten Sie, dass jedes zusätzlich hinzugefügte Feld zu einem Verlust der Performance führen kann.

    In der Volltextsuche Einrichtung können Sie auch einrichten, wie viele Suchergebnisse Ihnen maximal angezeigt werden sollen. Geben Sie im Inforegister Erweiterte Debitorensuche im Feld Max. Anzahl an erw. Debitorensuchergebnissen einen Wert ein, wenn Sie die Anzahl der Suchergebnisse einschränken wollen.

    So finden Sie einen Debitor mit der erweiterten Debitorensuche

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Verkaufsaufträge oder Verkaufsrechnungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie auf die Schaltfläche Neu.

    3. Geben Sie im Feld Debitorenname mehrere Begriffe ein, die hilfreich sind, um einen bestimmten Debitor zu finden.

      Tipp

      Geben Sie Suchbegriffe getrennt durch Leerzeichen ein. Sie können im Feld Debitorenname auch nach Teilbegriffen (Wortbeginn oder Wortende) sowie mit dem Platzhalter * suchen.

    4. Wählen Sie in der Ergebnisliste auf der Seite Debitorensuche Ergebnisse den gesuchten Debitor aus.

    5. Bestätigen Sie Ihre Auswahl mit OK.

      Hinweis
      • Die Seite Debitorensuche Ergebnisse öffnet sich nur dann, wenn mehr als ein Suchtreffer möglich ist. Der Debitor mit der größten Übereinstimmung wird als erstes Suchergebnis in der Auswahlliste angezeigt.
      • Ist lediglich eine Übereinstimmung möglich, wird das Suchergebnis direkt ins Feld Debitorenname übernommen.
      • Gibt es keinen Suchtreffer, erfolgt eine Fehlermeldung.

    Auf der Seite Debitorensuche Ergebnisse ist im Feld Herkunft ersichtlich, woher ein Suchergebnis stammt (zum Beispiel aus einer relativen Tabelle).
    In den Benachrichtigungen kann zudem die in der Volltextsuche Einrichtung definierte maximale Anzahl an Suchergebissen für den aktuellen Suchvorgang geändert werden. Nach einem Klick auf die Schaltfläche Erneut suchen ohne Limit werden Ihnen alle Suchergebnisse ohne Einschränkungen angezeigt.
