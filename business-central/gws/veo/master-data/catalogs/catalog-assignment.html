    Katalogzuordnung

    Mit der Katalogzuordnung können Sie Artikel bestimmten Katalogen (zum Beispiel Sommer, Winter, Ausverkauf und ähnliche) zuordnen. Dabei legen Sie fest, welche Artikel in welchen Katalog aufgenommen werden sollen. Diese Zuordnung kann außerdem als Grundlage für eine spätere Preisgestaltung verwendet werden.

    So legen Sie einen Katalog an

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kataloge ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Code eine Bezeichnung (zum Beispiel SOMMER) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Sommerbekleidung) ein.

    So ordnen Sie einen Artikel einem Katalog zu

    Ein Artikel kann mehreren Katalogen zugeordnet werden.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Artikel ein und klicken Sie auf das übereinstimmende Suchergebnis
    2. Öffnen Sie eine Artikelkarte. Klicken Sie alternativ im Menüband auf die Schaltfläche Neu, um einen neuen Artikel zu erstellen. Füllen Sie alle relevanten Belegkopffelder aus.
    3. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Verkauf > Kataloge. Wenn der Artikel schon zu einem Katalog zugeordnet ist, sehen Sie alle Kataloge, denen er zugeordnet ist.
    4. Klicken Sie im Menüband auf die Schaltfläche Neu.
    5. Klicken Sie in das Feld Katalogcode und wählen Sie einen Datensatz aus.
    6. Geben Sie im Feld Seite einen Wert ein, um festzulegen, auf welcher Seite der Artikel im Katalog zu finden ist.

    So legen Sie für einen Artikel einen Hauptkatalog fest

    Um die Preisgestaltung zu vereinfachen, können Sie jedem Artikel einen Hauptkatalog zuordnen.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Artikel ein und klicken Sie auf das übereinstimmende Suchergebnis
    2. Öffnen Sie eine Artikelkarte.
    3. Klicken Sie im Inforegister Lagerbestand in das Feld Hauptkatalog und wählen Sie einen Datensatz aus.
