    Erweiterte Artikelsuche

    Mit der Erweiterten Artikelsuche können Sie gezielt nach einem Artikel suchen, indem Sie mehrere Suchbegriffe verwenden. Sie können individuell einrichten, mit welchen Feldern auf der Artikelkarte die Suche abgeglichen werden soll.

    So richten Sie die Erweiterte Artikelsuche ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Volltextsuche Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Wählen Sie im Inforegister Suchoptionen Erweiterte Artikelsuche.
      Standardmäßig sind für die Erweiterte Artikelsuche die Suchfelder Nr., Beschreibung, Suchbegriff. Beschreibung 2 und Beschreibung 3 konfiguriert.
      So konfigurieren Sie bei Bedarf zusätzliche Suchfelder:
    3. Klicken Sie im Feld Suchfelder in einer neuen Tabellenzeile im Feld Feldnr. auf das Dreipunkt-Menü und wählen Sie ein Feld.
    4. Bestätigen Sie Ihre Auswahl mit OK.
    Tipp

    Im Feld Priorität können Sie jedem Suchfeld eine Priorität zuteilen und so die Darstellung der Suchergebnisse beeinflussen. Je höher die Priorität für ein Suchfeld ist, desto weiter oben in der Liste der Suchergebnisse wird ein Treffer einsortiert.

    So finden Sie einen Artikel mit der Erweiterten Artikelsuche auf der Seite Artikel

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Artikel ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Ressourcen > Volltextsuche.
    3. Wählen Sie im Inforegister Sucheinstellungen Erweiterte Artikelsuche.

    4. Geben Sie im Feld Suchbegriff mehrere Begriffe ein, die hilfreich sind, um einen bestimmten Artikel zu finden.

      Hinweis

      Geben Sie Suchbegriffe getrennt durch Leerzeichen ein. Sie können auch nach Teilbegriffen (Wortbeginn oder Wortanfang), nach Teilen der Artikelnummer sowie mit dem Platzhalter * suchen.

    So finden Sie einen Artikel mit der Erweiterten Artikelsuche in einem Beleg

    1. Öffnen Sie einen Beleg.
    2. Wählen Sie im Inforegister Zeilen eine Zeile.
    3. Geben Sie im Feld Volltextsuche einen Suchbegriff ein und bestätigen Sie Ihre Eingabe mit Enter.
    4. Wählen Sie im Inforegister Sucheinstellungen Erweiterte Artikelsuche.

    5. Ergänzen oder ändern Sie bei Bedarf Ihre Eingabe im Feld Suchbegriff.

      Hinweis

      Geben Sie Suchbegriffe getrennt durch Leerzeichen ein. Sie können auch nach Teilbegriffen (Wortbeginn oder Wortanfang), nach Teilen der Artikelnummer sowie mit dem Platzhalter * suchen.

    6. Wählen Sie im Inforegister Suchergebnisse einen Artikel und bestätigen Sie Ihre Wahl mit OK.
      Der gesuchte Artikel wird in die Belegzeile eingefügt.
