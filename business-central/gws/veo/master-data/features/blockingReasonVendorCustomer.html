    Sperrgründe für gesperrte Debitoren und Kreditoren einrichten

    Wird ein Debitor oder Kreditor gesperrt, können Sie den Grund in der Debitorenkarte oder Kreditorenkarte hinterlegen. Weitere Informationen finden Sie unter Debitoren sperren und Kreditoren sperren.

    So richten Sie einen Sperrgrund ein

    Hier am Beispiel eines Debitors:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Debitoren ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Debitorenkarte.
    3. Klicken Sie im Inforegister Adresse und Kontakt in das Feld Sperre und klicken Sie auf Neu, um einen neuen Datensatz anzulegen.
    4. Geben Sie im Feld Code eine Bezeichnung für einen Sperrgrund ein.
    5. Geben Sie im Feld Beschreibung eine Kurzbeschreibung ein.
    Hinweis

    In der Kreditorenkarte befindet sich das Feld Sperre im Inforegister Allgemein.
