    Auftragsstatus an Nordwest Webshop melden

    Bei der Bearbeitung von Verkaufsaufträgen können Sie den Status der aktuellen Auftragsbearbeitung an den Nordwest Webshop übermitteln.

    Hinweis

    Die Einrichtung des Webshops Nordwest erfolgt durch Ihren Berater.

    So richten Sie den Webshop Nordwest ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Webshops ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Code einen Code (zum Beispiel NORDWEST) für den Webshop Nordwest ein.
    4. Klicken Sie im Inforegister Allgemein in das Feld Webshop Anbieter und wählen Sie den Anbieter NW365.eShop aus.

    So richten Sie die Übermittlung des Auftragsstatus ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Webshops ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie für den Webshop Nordwest in das Feld Code (in unserem Beispiel NORDWEST), um die Seite Webshop zu öffnen.
    3. Geben Sie im Inforegister Nordwest die URL und den API Key ein.

    4. Aktivieren Sie in der Gruppe Runbook den Schalter für den Status, der übermittelt werden soll:

      Schalter aktiv Beschreibung
      Auftragsbestätigung bei Freigabe übermitteln Bei der Freigabe eines Auftrags wird ein Runbook Eintrag für die Auftragsbestätigung erstellt.
      Lieferavisierung bei Lieferung übermitteln Bei der Lieferung des Auftrags wird ein Runbook Eintrag für die Lieferavisierung erstellt.
      Zahlung bei vollständigem Zahlungseingang übermitteln Ist die aus dem Auftrag erstellte Rechnung vollständig bezahlt, wird ein Runbook Eintrag für die vollständige Zahlung erstellt.
    5. Wählen Sie anschließend in der Gruppe Datenaustauschdefinitionen für jeden Status eine Datenaustauschdefinition aus.
    Hinweis
    • Für den Status Freigegeben und für den Status Lieferung gibt es jeweils zwei mögliche Datenaustauschdefinitionen. Es können entweder nur Daten aus dem Verkaufskopf (zum Beispiel Bestellnummer oder externe Bestellnummer) oder zusätzlich Daten aus der Zeilenebene (zum Beispiel die bestätigte Artikelmenge) übermittelt werden.
    • Für den Status Lieferung wird auch die Trackingnummer für die Paketverfolgung aus dem Verkaufskopf übermittelt.

    So führen Sie Runbook Einträge aus

    Ist ein Verkaufsauftrag mit den importierten Daten aus dem Nordwest Webshop angelegt und freigegeben worden, wird nicht automatisch der Status übermittelt, sondern zunächst ein Runbook Eintrag erstellt, der noch auszuführen ist.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Webshop Runbook Einträge ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie einen Runbook Eintrag.
    3. Klicken Sie im Menüband auf die Schaltfläche Aktionen > Ausführen, um eine Statusmeldung an den Nordwest Webshop zu senden.
    Tipp

    Für jeden Runbook Eintrag ist im Feld Verknüpfter Datensatz ein Link zum zugehörigen Beleg hinterlegt, sodass Sie jederzeit einen Beleg prüfen können, bevor Sie einen Runbook Eintrag ausführen.

    Für jeden weiteren Schritt der Auftragsbearbeitung wie Lieferung, Teillieferung oder Fakturieren wird ein Runbook Eintrag erstellt, der einzeln auszuführen ist.

    Hinweis

    Klicken Sie im Menüband auf die Schaltfläche Aktionen > Verwerfen, wenn Sie einen Runbook Eintrag nicht senden wollen. Im Feld Verarbeitungsstatus wird der Wert dann auf Verworfen gesetzt. Der Runbook Eintrag ist aber nicht gelöscht. Sie können den Wert im Feld Verarbeitungsstatus wieder auf Ausstehend setzen, wenn Sie im Menüband auf die Schaltfläche Aktionen > Reaktivieren klicken.

    So richten Sie einen Aufgabenwarteschlangenposten für die Verarbeitung von Runbook Einträgen ein

    Um einen Aufgabenwarteschlangenposten für die Verarbeitung von Runbook Einträgen zu erstellen, gehen Sie vor wie in Aufgabenwarteschlangenposten beschrieben.

    Hinweis

    Verwenden Sie diese Werte für die Einrichtung:

    • Art des auszuführenden Objekts: Bericht
    • ID des auszuführenden Objekts: 70391250 (Execute Runbook)
