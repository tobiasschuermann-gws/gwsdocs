    Nordwest DataConnect

    Über die gevis ERP | VEO Schnittstelle zu Nordwest DataConnect finden Sie mithilfe der Funktion Volltextsuche alle Artikel, die von Nordwest angeboten werden. Zudem können Sie für Artikel, die in gevis ERP | VEO noch nicht angelegt sind, mit den übermittelten Informationen aus Nordwest DataConnect neue Artikelstammdaten erstellen.

    Hinweis

    Für die externe Bestandsabfrage ist es erforderlich, dass in der Handelsverband Einrichtung eine Kreditorennummer im Feld Handelsverband Kred.-Nr. eingetragen ist.

    Nordwest DataConnect kann in der Volltextsuche Einrichtung oder in Unterstütztes Setup eingerichtet werden.

    Hinweis

    Die Einrichtung von Nordwest DataConnect erfolgt durch Ihren Berater.

    So richten Sie Nordwest DataConnect in der Volltextsuche Einrichtung ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Volltextsuche Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie das Inforegister DataConnect.
    3. Geben Sie im Feld URL den Wert https://dataconnect.nordwest.com/webservice ein.
    4. Geben Sie im Feld API Key den API Key ein.
    5. Wählen Sie optional im Feld Artikelvorlage einen Vorlagencode aus.
    6. Geben Sie im Feld Artikel identifizieren anhand den Wert Artikelnr. 2 ein.
    Hinweis

    Ob die Verbindung zu DataConnect hergestellt ist, können Sie prüfen, indem Sie im Menüband auf die Schaltfläche DataConnect Verbindungsprüfung klicken.

    So richten Sie Nordwest DataConnect in Unterstütztes Setup ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Unterstütztes Setup ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Nordwest auf die Schaltfläche Nordwest DataConnect Einrichtung.
    3. Geben Sie im Feld URL den Wert https://dataconnect.nordwest.com/webservice ein.
    4. Geben Sie im Feld API Key den API Key ein.
    5. Wählen Sie optional im Feld Artikelvorlage einen Vorlagencode aus.
    6. Geben Sie im Feld Artikel identifizieren anhand den Wert Artikelnr. 2 ein.
    7. Bestätigen Sie die Angaben mit Weiter und schließen Sie die Einrichtung mit Ende ab.

    Die Einstellungen werden automatisch im Inforegister Nordwest DataConnect der Volltextsuche Einrichtung übernommen.

    So führen Sie die Volltextsuche in der Liste Artikel aus

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Artikel ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Ressourcen > Volltextsuche.
    3. Klicken Sie in das Feld Suchanbieter und wählen Sie die Option Nordwest DataConnect aus.
    4. Geben Sie im Feld Suchbegriff ein Suchwort ein.
    5. Klicken Sie im Menüband auf die Schaltfläche Starte Suche.

    Im Inforegister Suchergebnisse werden Ihnen jetzt alle Artikel angezeigt, auf die der Suchbegriff zutrifft. Ob ein Artikel bereits in gevis ERP | VEO angelegt ist, erkennen Sie daran, dass im Feld Nr. ein Wert eingetragen ist.

    So führen Sie die Volltextsuche in Einkaufs- und Verkaufsbelegen aus

    1. Öffnen Sie einen Einkaufsbeleg oder einen Verkaufsbeleg.
    2. Geben Sie im Inforegister Zeilen im Feld Volltextsuche einen Suchbegriff ein und bestätigen Sie Ihre Eingabe mit Enter.
    3. Klicken Sie in das Feld Suchanbieter und wählen Sie die Option Nordwest DataConnect aus.
    4. Markieren Sie im Inforegister Suchergebnisse einen Artikel.
    5. Geben Sie im Feld Menge eine Anzahl ein und bestätigen Sie Ihre Eingabe mit OK.

    Der Artikel wird zusammen mit der Mengenangabe direkt in die Belegzeile übertragen.

    Hinweis

    Wenn es für den Artikel, der in die Belegzeile übertragen wurde, noch keine Artikelstammdaten in gevis ERP | VEO gibt, werden diese automatisch erstellt.

    So können Sie Artikeldetails in den Suchergebnissen abfragen

    1. Führen Sie eine Volltextsuche für Nordwest DataConnect aus wie oben beschrieben im Abschnitt "So führen Sie die Volltextsuche in der Liste Artikel aus" oder in "So führen Sie die Volltextsuche in Einkaufs- und Verkaufsbelegen aus".
    2. Markieren Sie im Inforegister Suchergebnisse eine oder mehrere Zeilen, für die Artikeldetails angezeigt werden sollen.
    3. Klicken Sie im Menüband auf die Schaltfläche Artikeldetails abfragen.
      Für die zuvor ausgewählten Artikel finden Sie jetzt in den Feldern der Suchergebnisse Angaben wie Preiseinheiten (Einkauf und Verkauf), Einkaufs- und Verkaufspreise, die Kreditorenummer und die Kreditoren-Artikelnummer. Im Feld Bild ist zusätzlich ein Artikelbild hinterlegt.

    Weitere Artikeldetails finden Sie auf der Seite Nordwest Artikeldetails.
    Markieren Sie einen Artikel auf der Seite Volltextsuche und klicken Sie im Menüband auf die Schaltfläche Artikeldetails anzeigen.

    • Im Seitentitel der neuen Seite sehen Sie die Artikelnummer (Feld Nr. 2) und die Produktbeschreibung für den Artikel aus Nordwest DataConnect.
    • In der Infobox Verweise finden Sie zusätzliche Bemerkungen, Zusatzinformationen und Bilder.

    • Zudem sind alle Informationen in fünf Inforegistern übersichtlich dargestellt:

      Inforegister Beispiele für Artikeldetails
      Allgemein Produktbeschreibung, Beschreibung Lang, Hauptmerkmale, Beschaffungszeit, Herkunftsland, Zollposition
      Einkauf Kreditorennummer, Bezugsquelle, Kreditoren-Artikelnummer, Einkaufspreis
      Verkauf Verkaufspreis
      Hersteller Original Herstellerartikelnummer, Hersteller Original Beschreibung
      Einheiten Basiseinheitencode, Dimensionen (Höhe, Breite, Länge), Gewicht
      Hinweis
      • Gibt es für einen Artikel noch keine Artikelstammdaten, können Sie diese jetzt anlegen. Klicken Sie im Menüband der Seite Nordwest Artikeldetails auf die Schaltfläche Artikel anlegen. Im Inforegister Allgemein erfolgt im Feld Nr. automatisch der Eintrag der in gevis ERP | VEO generierten Artikelnummer und die Informationen aus Nordwest DataConnect werden für den Artikel angelegt - inklusive aller Preise und Rabatte für den Einkauf als auch für den Verkauf.
      • Ist ein Artikel in gevis ERP | VEO angelegt, können Sie die zugehörige Artikelkarte aufrufen, indem Sie im Menüband auf die Schaltfläche Zugehörig > Ansicht klicken.
