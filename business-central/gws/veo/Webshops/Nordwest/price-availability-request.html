    Nordwest Preis- und Verfügbarkeitsanfrage

    Mit der Nordwest Preis- und Verfügbarkeitsanfrage senden Sie aus den Verkaufs- und Einkaufsbelegen heraus eine Anfrage an Nordwest, wenn Sie aktuelle Informationen zur Verfügbarkeit oder zum Einkaufspreis eines Artikels brauchen. Die von Nordwest zurückgemeldeten Daten werden direkt in die Artikelzeilen übertragen.

    Hinweis

    Die Einrichtung der Nordwest Preis- und Verfügbarkeitsanfrage erfolgt durch Ihren Berater.

    So richten Sie die Nordwest Preis- und Verfügbarkeitsanfrage ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Nordwest Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Ergänzen Sie die erforderlichen Parameter:
    Feld Beschreibung
    Preis-/Verfügbarkeit Endpunkt API URL Gibt den Endpunkt des Nordwest Webservices an. Die URL wird durch Nordwest vergeben und mit der derzeit bekannten URL voreingestellt.
    Azure Function URL Gibt die URL der Azure Function zur Weiterleitung der Anfrage an den Nordwest-Webservice an.
    API Key Gibt den API Key für den Zugriff auf die Azure Function an. Ist kein Wert angegeben, wird ein Standard-API Key aus dem dazugehörigen Azure Schlüsseltresor ausgelesen und verwendet.

    So senden Sie eine Preis- und Verfügbarkeitsanfrage aus einem Verkaufsangebot oder Verkaufsauftrag

    Die Anfrage kann im Menüband des Belegs oder im Menüband des Inforegisters Zeilen ausgeführt werden.

    So senden Sie eine Anfrage für alle Artikelzeilen eines Belegs

    1. Klicken Sie im Menüband auf die Schaltfläche Start > Nordwest Preis und Verfügbarkeit.

    So senden Sie eine Anfrage für eine oder für mehrere Artikelzeilen

    1. Markieren Sie eine oder mehrere Artikelzeilen.
    2. Klicken Sie im Menüband des Inforegisters Zeilen auf die Schaltfläche Zeile > Funktionen > Nordwest Preis und Verfügbarkeit.

    Die zurückgemeldeten Daten zum Preis und zur Verfügbarkeit finden Sie in diesen Feldern:

    Feld Beschreibung
    NW Externer Bestand Verfügbar Enthält den zurückgemeldeten Bestand des Artikels
    NW Externer Einstandspreis (MW) Enthält den aktuellen Einkaufspreis für den Artikel
    NW Externer Bestand Abfragedatum Enthält das Datum der Abfrage

    Zudem erfolgen Eintragungen in diesen Feldern:

    Feld Beschreibung
    Einstandspreis (MW) Aktueller Einkaufspreis aus Anfrage
    Preisherkunft Einstandspreis Externe Abfrage
    Preisherkunft Externe Preisabfrage
    Aktionspreis Nein

    So senden Sie eine Preis- und Verfügbarkeitsanfrage aus einer Einkaufsanfrage oder Einkaufsbestellung

    Die Anfrage kann im Menüband des Belegs oder im Menüband des Inforegisters Zeilen ausgeführt werden.

    So senden Sie eine Anfrage für alle Artikelzeilen eines Belegs

    1. Klicken Sie im Menüband auf die Schaltfläche Start > Nordwest Preis und Verfügbarkeit.

    So senden Sie eine Anfrage für eine oder für mehrere Artikelzeilen

    1. Markieren Sie eine oder mehrere Artikelzeilen.
    2. Klicken Sie im Menüband des Inforegisters Zeilen auf die Schaltfläche Zeile > Funktionen > Nordwest Preis und Verfügbarkeit.

    Die zurückgemeldeten Daten zum Preis und zur Verfügbarkeit finden Sie in diesen Feldern:

    Feld Beschreibung
    NW Externer Bestand Verfügbar Enthält den zurückgemeldeten Bestand des Artikels
    NW Externer Einstandspreis (MW) Enthält den aktuellen Einkaufspreis für den Artikel
    NW Externer Bestand Abfragedatum Enthält das Datum der Abfrage

    Zudem erfolgen Eintragungen in diesen Feldern:

    Feld Beschreibung
    EK-Preis (MW)
    (Nur wenn zurückgemeldeter EK-Preis ungleich 0)    		 Aktueller Einkaufspreis aus Anfrage
    Preisherkunft Einstandspreis
    (Nur wenn zurückgemeldeter EK-Preis ungleich 0)    		 Externe Preisabfrage

    So finden Sie Detailinformationen für jede Preis- und Verfügbarkeitsanfrage

    Für jede Anfrage wird ein Protokoll erstellt. In allen Belegen, in denen eine Preis- und Verfügbarkeitsanfrage ausgelöst wurde, wird nach erfolgreicher Transaktion eine Benachrichtigung über dem Menüband angezeigt. Klicken Sie dort auf die Schaltfläche Protokolle öffnen, um eine Liste mit detaillierten Protokolleinträgen zu öffnen.
