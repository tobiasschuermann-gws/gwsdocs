    Preisfindung nach Lieferadresse

    Für jeden Debitor können Sie mehrere Lieferadressen anlegen und für diese Lieferadressen unterschiedliche Verkaufspreise hinterlegen.

    Hinweis

    Da diese Funktion nur mit der neuen Preisfindung möglich ist, sollte in der Funktionsverwaltung die Funktionsaktualisierung: Neue Verkaufspreiskalkualtion aktiviert sein.

    So richten Sie Codes für Lieferadressen in der Debitorenkarte ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Debitoren ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Debitorenkarte.
    3. Klicken Sie im Menüband auf die Schaltfläche Debitor > Lief. an Adressen.
    4. Klicken Sie im Menüband auf die Schaltfläche Neu.
    5. Geben Sie im Inforegister Allgemein im Feld Code einen Code ein und legen Sie eine Lieferadresse an.

    So richten Sie Codes für Lieferadressen im Debitorencockpit ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Debitorencockpit ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Geben Sie im Inforegister Debitor Information im Feld Nr. eine Debitorennummer ein.
    3. Klicken Sie im Menüband auf die Schaltfläche Debitor > Lief. an Adressen.
    4. Klicken Sie im Menüband auf die Schaltfläche Neu.
    5. Geben Sie im Inforegister Allgemein im Feld Code einen Code ein und legen Sie eine Lieferadresse an.

    So legen Sie Preise für Lieferadressen an

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Verkaufspreislisten ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Inforegister Allgemein im Feld Code einen Code ein.
    4. Geben Sie im Feld Beschreibung eine Kurzbeschreibung für die neue Verkaufspreisliste ein.
    5. Sie haben zwei Möglichkeiten Preise für Lieferadressen zu hinterlegen:
      • Klicken Sie in das Feld Zuweisen zu Typ und wählen Sie die Option Alle Debitoren aus, wenn Sie eine Verkaufspreisliste für mehrere Debitoren anlegen wollen. In diesem Fall ist im Inforegister Zeilen in jeder Zeile im Feld Zuweisen zu Nr. eine Debitorennummer einzugeben.
      • Klicken Sie in das Feld Zuweisen zu Typ, wählen Sie die Option Debitor aus und geben Sie im Feld Zuweisen zu Nr. eine Debitorennummer ein, wenn Sie nur für einen Debitor spezifische Preise anlegen wollen. Die Debitorennummer wird dann im Inforegister Zeilen automatisch in jede Zeile übertragen.
    6. Klicken Sie in der Gruppe Anzeigen in das Feld Spalten anzeigen für und wählen Sie eine Option aus:
      • Preis, wenn Sie unterschiedliche Preise anlegen wollen;
      • Preis und Rabatt, wenn Sie neben spezifischen Preisen auch Rabatte für die unterschiedlichen Lieferadressen anlegen wollen;
      • Rabatt, wenn Sie nur Rabatte für die angelegten Lieferadresse anlegen wollen.
    7. Aktivieren Sie in der Gruppe Zeilenstandardwerte den Schalter Akualisieren von Standardeinstellungen zulassen.

    8. Geben Sie im Inforegister Zeilen im Feld Lief. an Code den Code für eine Lieferadresse ein.

      Hinweis

      Geben Sie im Feld Lief. an Code nichts ein, wenn Sie einen Verkaufspreis als Standard hinterlegen wollen.

    9. Geben Sie im Feld Produktnr. die Artikelnummer ein.
    10. Geben Sie im Feld VK-Preis den Verkaufspreis ein, der für die Lieferadresse gelten soll.
    11. Geben Sie alternativ oder zusätzlich im Feld Zeilenrabatt % einen Wert ein und setzen Sie einen Haken im Feld Zeilenrabatt zulassen, wenn Sie Rabatte hinterlegen wollen.
    12. Setzen Sie im Feld Rech.-Rabatt zulassen einen Haken, wenn der Verkaufspreis für eine Lieferadresse auch für den Rechnungsrabatt zugelassen werden soll.

    So wählen Sie im Verkaufsauftrag einen mit der Lieferadresse verknüpften Verkaufspreis

    1. Öffnen oder erstellen Sie einen Verkaufsauftrag.
    2. Markieren Sie im Inforegister Zeilen einen Artikel, für den Sie in Abhängigkeit von der Lieferadresse unterschiedliche Verkaufspreise wie oben beschrieben hinterlegt haben.
    3. Klicken Sie im Inforegister Lieferung und Abrechnung in das Feld Lief. an und wählen Sie die Option Abweichende Lieferadresse aus.
    4. Wählen Sie eine Lieferadresse aus und bestätigen Sie Ihre Auswahl mit OK.
    5. Bestätigen Sie im Dialogfenster die geänderte Lieferadresse mit Ja.
