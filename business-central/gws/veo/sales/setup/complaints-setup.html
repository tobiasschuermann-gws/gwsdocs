    Reklamationsgründe einrichten

    Wenn Sie für die Bearbeitung von Verkaufsreklamationen Reklamationsgründe anlegen, können Sie diese später auswerten. Außerdem können Sie für jeden Reklamationsgrund mögliche Zusatzpositionen hinterlegen, sodass die Zusatzgebühren automatisch zur Verkaufsreklamation hinzugefügt werden.

    So richten Sie Reklamationsgründe ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Reklamationsgründe ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Code eine Beschreibung (zum Beispiel BESCHÄDIGT) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Transportschaden) ein.
    4. Klicken Sie optional ins Feld Standardlagercode und wählen Sie ein Lager aus, wenn die reklamierte Ware zurückgesendet wird und in einem bestimmten Lager gebucht werden soll.
    Hinweis

    Setzen Sie im Feld Ohne Lagerbewertung einen Haken, wenn bei einem bestimmten Reklamationsgrund der Artikel in einem Lager gebucht werden soll ohne den Lagerwert zu erhöhen, weil zum Beispiel der Artikel vernichtet werden soll oder nur zur Reparatur vor Ort ist.

    So hinterlegen Sie Zusatzpositionen für Reklamationsgründe

    Hinweis

    Aktivieren Sie auf der Seite Einrichtung Ein-/Ausschalter im Inforegister ERP, Gruppe Artikel den Schalter Artikelzusatzpositionen aktivieren.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Reklamationsgründe ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie einen Reklamationsgrund.
    3. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Zusatzpositionen.
    4. Klicken Sie in das Feld Nr. und wählen Sie einen Datensatz aus oder klicken Sie auf Neu, um einen neuen Datensatz anzulegen (zum Beispiel VK-WIEDEREINLAG).
    5. Setzen Sie optional im Feld Pauschal einen Haken.
    6. Geben Sie im Feld Preis die Höhe der Zusatzgebühr ein.
