    Tresenverkauf einrichten

    Der Tresenverkauf bietet die Möglichkeit eventuelle Lageraktivitäten wie beispielsweise die Nutzung eines Warenausgangs und einer Kommissionierung zu übersteuern. Dies ist hilfreich, wenn ein Kunde vor Ort beraten wird und den Artikel direkt mitnehmen möchte. Eine schnelle Ausgabe der Ware an den Kunden ist wichtig, wodurch logistische Aktivitäten übersprungen werden müssen.
    Für den Tresenverkauf benötigen Sie eine Belegvorlage, die im Verkaufsauftrag ausgewählt werden kann.

    So richten Sie den Tresenverkauf ein

    Hier am Beispiel für den Tresenverkauf mit Lieferschein:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Belegvorlagen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Inforegister Allgemein im Feld Code eine Bezeichnung (zum Beispiel TRESENVERK_LS) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Tresenverkauf mit Lieferschein) ein.
    4. Klicken Sie in das Feld Belegart und wählen Sie die Option Verkaufsauftrag aus.
    5. Legen Sie optional im Inforegister Felder vorbelegen Belegvorlagenzeilen an für Felder, die Sie im Beleg vorbefüllen möchten. Hierdurch wird die Erfassung des Belegs erleichtert, da (Pflicht-) Felder vorausgefüllt werden können. Für das Beispiel Tresenverkauf mit Lieferschein können Sie beispielsweise folgende Felder vorbelegen:
    Tabellen-ID Feld-ID Wert
    36 (Sales Header - Verkaufskopf) 27 (Shipment Method - Lieferbedingung) Abholung
    36 (Sales Header - Verkaufskopf) 105 (Shipping Agent Code - Zustellerart) Abholung
    36 (Sales Header - Verkaufskopf) 5750 (Shipping Advice - Lieferanweisung) Komplettlieferung
    36 (Sales Header - Verkaufskopf) 5374730 (FAV Document Status - Belegstatus) WA-Sperre
    1. Aktivieren Sie im Inforegister Funktionen die Schalter Lieferbare Auftragsmenge im Auftrag reservieren und Lieferfähige Menge als Liefermenge im Auftrag setzen.
    2. Klicken Sie in das Feld Drucken nach Buchung und wählen Sie die Option Lieferung aus.
    Tipp

    Über das Feld Drucken nach Buchung können Sie differenzieren, ob explizit nur ein Lieferschein, eine Rechnung oder beide Belege ausgedruckt werden sollen. Zur Abdeckung aller Eventualitäten können Sie mehrere Belegvorlagen mit unterschiedlichen Ausdruck-Optionen anlegen.
