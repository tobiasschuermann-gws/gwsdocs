    Automatisierte Auftragsabwicklung einrichten

    Mithilfe dieser Funktion wird bei Freigabe eines Verkaufsauftrags der Belegzeilenstatus einer Verkaufszeile automatisch freigegeben, sofern die Verkaufsauftragszeile als Direktlieferung oder Spezialauftrag gekennzeichnet ist. Wird der Status eines Verkaufsauftrags zurückgesetzt, wird auch der Belegzeilenstatus der zugehörigen Verkaufsauftragszeilen automatisch zurückgesetzt.

    Die Einrichtung der automatisierten Auftragsabwicklung für Direktlieferungen und Spezialaufträge erfolgt in mehreren Schritten:

    • Belegzeilenstatus aktivieren
    • Belegzeilenstatus einrichten
    • Workflows zum Freigeben und Zurücksetzen des Belegzeilenstatus einrichten
    • Buch.-Blattvorlagen anlegen
    • Aufgabenwarteschlangenposten einrichten

    So aktivieren Sie den Belegzeilenstatus

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Ein-/Ausschalter ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister ERP den Schalter Belegzeilenstatus aktivieren.

    So richten Sie den Belegzeilenstatus ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Belegzeilenstatus ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Geben Sie in der nächsten freien Zeile im Feld CODE eine Bezeichnung für eine freigegebene Belegzeile (zum Beispiel FREI) ein.
    3. Geben Sie in der gleichen Zeile im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Belegzeile freigegeben) ein.
    4. Geben Sie in der nächsten freien Zeile in das Feld CODE eine Bezeichnung für eine nicht freigegebene Belegzeile (zum Beispiel OFFEN) ein.
    5. Geben Sie in der gleichen Zeile im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Belegzeile offen) ein.

    So richten Sie die Workflows zum Freigeben und Zurücksetzen des Belegzeilenstatus ein

    Bevor Sie die Workflows einrichten können, müssen Sie die Workflow-Erweiterung aktivieren.

    So aktivieren Sie die Workflow-Erweiterung

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Ein-/Ausschalter ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister ERP den Schalter Workflow-Erweiterung aktivieren.

    So richten Sie den Workflow zum Freigeben des Belegzeilenstatus ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Workflows ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie im Menüband die Drop-Down-Liste der Schaltfläche Neuer Workflow aus Vorlage und klicken Sie auf Neu.
    3. Geben Sie im Feld CODE eine Bezeichnung (zum Beispiel FREIGABE) ein.
    4. Geben Sie im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Auftragsfreigabe) ein.
    5. Wählen Sie im Inforegister Workflowschritte im Feld Wenn Ereignis über das Dreipunkt-Menü die Option Ein Verkaufsbeleg wird freigegeben aus.
    6. Bestätigen Sie Ihre Auswahl mit OK.
    7. Klicken Sie in das Feld Bei Bedingung, um die Seite Ereignisbedingungen zu öffnen.
    8. Klicken Sie im Inforegister Filter: Verkaufskopf auf die Schaltfläche + Filter und wählen Sie über die Drop-Down-Liste Verfügbare Felder die Option Belegart aus.
    9. Öffnen Sie im neu hinzugefügten Feld Belegart die Drop-Down-Liste und setzen Sie einen Haken für die Option Auftrag.
    10. Bestätigen Sie Ihre Auswahl mit OK.
    11. Klicken Sie in das Feld Dann Antwort, um die Seite Workflowreaktionen zu öffnen.
    12. Wählen Sie im Feld Antwort auswählen über das Dreipunkt-Menü die Option Setzen Sie den Belegzeilenstatus auf %1 aus und bestätigen Sie Ihre Auswahl mit OK.
    13. Wählen Sie im Inforegister Optionen für die ausgewählte Reaktion im Feld Belegzeilenstatus über die Drop-Down-Liste den zuvor angelegten Status (in unserem Beispiel FREI) für eine freigegebene Belegzeile aus.
    14. Aktivieren Sie im Inforegister Optionen für die ausgewählte Reaktion den Schalter Bestellung freigeben.
    15. Bestätigen Sie Ihre Auswahl mit OK.
    16. Aktivieren Sie auf der Seite Workflow den Schalter Aktiviert.

    So richten Sie den Workflow zum Zurücksetzen des Belegzeilenstatus

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Workflows ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie im Menüband die Drop-Down-Liste der Schaltfläche Neuer Workflow aus Vorlage und klicken Sie auf Neu.
    3. Geben Sie im Feld CODE eine Bezeichnuung (zum Beispiel ZURÜCK) ein.
    4. Geben Sie im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Belegzeilenstatus zurücksetzen) ein.
    5. Wählen Sie im Inforegister Workflowschritte im Feld Wenn Ereignis über das Dreipunkt-Menü die Option Ein Verkaufsbeleg wird zurückgesetzt aus.
    6. Bestätigen Sie Ihre Auswahl mit OK.
    7. Klicken Sie in das Feld Bei Bedingung, um die Seite Ereignisbedingungen zu öffnen.
    8. Klicken Sie im Inforegister Filter: Verkaufskopf auf die Schaltfläche + Filter und wählen Sie über die Drop-Down-Liste Verfügbare Felder Belegart.
    9. Öffnen Sie im neu hinzugefügten Feld Belegart die Drop-Down-Liste und setzen Sie einen Haken für die Option Auftrag.
    10. Bestätigen Sie Ihre Auswahl mit OK.
    11. Klicken Sie in das Feld Dann Antwort, um die Seite Workflowreaktionen zu öffnen.
    12. Wählen Sie im Feld Antwort auswählen über das Dreipunkt-Menü die Option Setzen Sie den Belegzeilenstatus auf %1 aus und bestätigen Sie Ihre Auswahl mit OK.
    13. Wählen Sie im Inforegister Optionen für die ausgewählte Reaktion im Feld Belegzeilenstatus über die Drop-Down-Liste den zuvor angelegten Status für eine nicht freigegebene Belegzeile (zum Beispiel OFFEN).
    14. Bestätigen Sie Ihre Auswahl mit OK.
    15. Aktivieren Sie auf der Seite Workflow den Schalter Aktiviert.

    Weitere Informationen finden Sie unter Workflows in Dynamics 365 Business Central.

    So legen Sie die Buch.-Blattvorlagen an

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Debitoren und Verkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Wählen Sie im Inforegister Auftragsbezogene Bestellungen im Feld Artikelbeschreibung bei Direktlieferung über die Drop-Down-Liste die Option Verkaufsauftrag aus.
    3. Wählen Sie im Feld Artikelbeschreibung bei Spezialaufträgen über die Drop-Down-Liste die Option Verkaufsauftrag aus.
    4. Aktivieren Sie den Schalter im Feld Buchung der EK-Rechnung vor VK-Rechnung zulassen.
    5. Wählen Sie im Feld Vorschlagsvorlagenname Direktlieferung über die Drop-Down-Liste einen Eintrag aus.
    6. Wählen Sie im Feld Buch.-Blattname Direktlieferung über das Dreipunkt-Menü​ einen Eintrag aus.
    7. Wählen Sie im Feld Vorschlagsvorlagenname Spezialauftrag über die Drop-Down-Liste einen Eintrag aus.
    8. Wählen Sie im Feld Buch.-Blattname Spezialauftrag über das Dreipunkt-Menü​ einen Eintrag aus.
    9. Wählen Sie im Feld Belegzeilenstatus 'Freigeben' über die Drop-Down-Liste die Option Frei aus.

    Weitere Informationen finden Sie unter Buch.-Blätter in Microsoft Dynamics 365 Business Central verwenden.

    So richten Sie den Aufgabewarteschlangenposten ein

    Legen Sie für Direktlieferungen und Spezialaufträge je einen eigenen Aufgabenwarteschlangenposten an.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Aufgabenwarteschlangenposten ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Wählen Sie im Inforegister Allgemein im Feld Art des auszuführenden Objekts über die Drop-Down-Liste die Option CodeUnit aus.
    4. Geben Sie im Feld ID des auszuführenden Objekts die CodeUnit ID 5374391 ein oder klicken Sie auf das Dreipunkt-Menü und wählen Sie die CodeUnit ID 5374391 aus.
    5. Bestätigen Sie Ihre Auswahl mit OK.
    6. Geben Sie im Feld Parameterzeichenfolge
      • Direkt: Für den Aufgabenwarteschlangenposten für Direktlieferungen ein.
      • Spezial: Für den Aufgabenwarteschlangenposten für Spezialaufträge ein.
    7. Aktivieren Sie im Inforegister Wiederholungen die Schalter der Wochentage, an denen der Aufgabenwarteschlangenposten ausgeführt werden soll.
    8. Geben Sie im Feld Startzeit eine Uhrzeit (zum Beispiel 08:00:00) ein, zu der die Ausführung des Aufgabenwarteschlangenpostens starten soll.
    9. Geben Sie im Feld Endzeit eine Uhrzeit (zum Beispiel 17:30:00) ein, zu der die Ausführung des Aufgabenwarteschlangenpostens enden soll.

    10. Geben Sie im Feld Anzahl der Minuten zwischen Ausführungen in Ziffern ein, wie viele Minuten zwischen den Wiederholungen liegen sollen oder übernehmen Sie den voreingestellten Wert von 1440.

      Hinweis

      Es müssen mindestens fünf Minuten zwischen den Wiederholungen liegen.

    11. Klicken Sie im Menüband auf die Schaltfläche Start > Status auf 'Bereit' festlegen.

    Weitere Informationen finden Sie unter Verwenden von Aufgabenwarteschlangen für die Aufgabenplanung.​
