    Verkaufsangebote einrichten

    Über ein Verkaufsangebot kann einem Kontakt oder einem Debitor ein unverbindliches Angebot unterbreitet werden.

    So richten Sie Verkaufsangebote ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Debitoren und Verkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Wählen Sie im Inforegister Nummernserie im Feld Angebotsnummern eine Angebotsnummernserie aus. Weitere Informationen zum Erstellen einer Nummernserie finden Sie unter Erstellen von Nummernkreisen.

    So richten Sie die automatische Archivierung von Verkaufsangeboten ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Debitoren und Verkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Wählen Sie im Inforegister Archivierung im Feld Angebote archivieren die Option Immer oder Fragen aus um festzulegen, dass Angebote automatisch archiviert werden sollen, wenn sie gelöscht werden.
    3. Aktivieren Sie im Inforegister Archivierung - Erweitert den Schalter Angebot bei Freigabe archivieren um festzulegen, dass Angebote automatisch archiviert werden sollen, wenn sie freigegeben werden. Weitere Informationen finden Sie unter Angebot bei Freigabe archivieren.
    4. Wählen Sie im Inforegister Archivierung - Erweitert im Feld Verkauf - Archivieren beim Druck als PDF die Option Benutzer fragen aus um festzulegen, dass das Archivieren von Angeboten abgefragt werden soll, wenn sie als PDF Datei gedruckt werden.

    So richten Sie die Möglichkeit ein, für gesperrte Debitoren ein Angebot zu erstellen

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Ein-/Ausschalter ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister Verkauf & Marketing den Schalter Angebot gesperrte Kunden aktivieren.
