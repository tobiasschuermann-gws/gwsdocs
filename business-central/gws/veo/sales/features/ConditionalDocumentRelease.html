    Regelbasierte Belegfreigabe

    Mit der regelbasierten Belegfreigabe können Sie Bedingungen festlegen, die bei der Freigabe oder dem Zurücksetzen des Status angewendet werden. Abhängig davon, ob eine Bedingung erfüllt ist oder nicht, wird eine entsprechende Aktion ausgeführt. Jede Regel entspricht einer Zeile innerhalb der Belegfreigabegruppe.

    So aktivieren Sie die regelbasierte Belegfreigabe

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Ein-/Ausschalter ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister ERP den Schalter Regelbasierte Belegfreigabe aktivieren.

    So erstellen Sie eine regelbasierte Belegfreigabegruppe

    Hier am Beispiel eines Verkaufsauftrags, der freigegeben wird, wenn Artikel mit einem Deckungsbeitrag über 20% enthalten sind:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Regelbasierte Belegfreigabegruppen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie im Inforegister Allgemein in das Feld Art und wählen Sie die Option Freigeben aus.
    4. Geben Sie im Feld Code eine Beschreibung (zum Beispiel DB % > 20%) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Freigabe bei DB in % > 20%) ein.
    5. Klicken Sie in das Feld Bereich und wählen Sie die Option Verkauf aus.
    6. Klicken Sie in das Feld Belegart und wählen Sie die Option Auftrag/Bestellung aus.
    7. Klicken Sie in das Feld Aktion bei Erfüllung aller Regeln und wählen Sie die Option Keine aus.
    8. Klicken Sie in das Feld Aktion bei Widerspruch und wählen Sie die Option Fehler aus.
    9. Klicken Sie im Inforegister Regelbasierte Belegfreigabe in das Feld Belegbereich und wählen Sie die Option Belegzeilen aus.
    10. Klicken Sie in das Feld Zeilenart und wählen Sie die Option Artikel aus.
    11. Geben Sie in das Feld Feld Nr. den Wert 5374273 ein für das Feld DB %.
    12. Klicken Sie in das Feld Bedingung und wählen Sie die Option Größer als aus.
    13. Geben Sie in das Feld Wert als Text den Wert 20 ein.
    14. Klicken Sie in das Feld Status und wählen Sie die Option Zertifiziert aus. Die Belegfreigabegruppe ist nun aktiv.
    Hinweis
    • Wenn ein Verkaufsauftrag freigegeben wird, erfolgt die Überprüfung dieser Belegfreigabegruppe.
    • Wenn das Feld DB % einen Wert größer als 20 aufweist, bleibt der Auftrag unverändert und wird freigegeben.
    • Wenn das Feld DB % einen Wert kleiner oder gleich 20 hat, wird die im Feld Aktion bei Widerspruch ausgewählte Option Fehler ausgelöst. In diesem Fall wird eine Fehlermeldung angezeigt und der Verkaufsauftrag wird nicht freigegeben.

    So erweitern Sie die Belegfreigabegruppe mit einer weiteren Bedingung

    Unser Beispiel DB % > 20% erweitern wir mit einer weiteren Bedingung, dass der Verkaufsauftrag nur freigegeben wird, wenn Artikel mit einem Deckungsbeitrag über 30% enthalten sind. Zunächst ist es erforderlich, einen Mindestdeckungsbeitrag für die Artikel zu hinterlegen, um später die Belegfreigabegruppe erweitern zu können.

    So hinterlegen Sie einen Mindestdeckungsbeitrag bei einem Artikel

    Um einen Mindestdeckungsbeitrag zu hinterlegen, verfahren Sie wie in Deckungsbeitrag von Schritt 1 bis 3 beschrieben.

    Hinweis

    Verwenden Sie diesen Wert für die Einrichtung:

    • Mindest-DB %: 30

    So erweitern Sie die Belegfreigabegruppe mit einer weiteren Bedingung

    Hier am Beispiel unserer Belegfreigabegruppe DB % > 20%:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Regelbasierte Belegfreigabegruppen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie die Belegfreigabegruppe DB % > 20%.
    3. Klicken Sie in das Feld Status und wählen Sie die Option In Entwicklung aus.
    4. Klicken Sie im Menüband des Inforegisters Regelbasierte Belegfreigabe auf die Schaltfläche Neue Zeile.
    5. Klicken Sie im Inforegister Regelbasierte Belegfreigabe in das Feld Belegbereich und wählen Sie die Option Belegzeilen aus.
    6. Klicken Sie in das Feld Zeilenart und wählen Sie die Option Artikel aus.
    7. Geben Sie in das Feld Feld No. den Wert 5374273 ein für das Feld DB %.
    8. Klicken Sie in das Feld Bedingung und wählen Sie die Option Größer als oder Gleich aus.
    9. Klicken Sie in das Feld Tabellenname und wählen Sie die Option Artikel aus.
    10. Geben Sie in das Feld Tabellenfeld No. den Wert 5374273 ein für das Feld Mindest-DB %.
    11. Klicken Sie in das Feld Status und wählen Sie die Option Zertifiziert aus. Die Belegfreigabegruppe ist nun wieder aktiv.
    Hinweis
    • Wenn ein Verkaufsauftrag freigegeben wird, erfolgt die Überprüfung durch die Belegfreigabegruppe.
    • Wenn das Feld DB % einen Wert größer oder gleich als der Wert im Feld Min. DB% für den Artikel aufweist, bleibt der Verkaufsauftrag unverändert und wird freigegeben.
    • Wenn das Feld DB % einen Wert kleiner als der Wert im Feld Min. DB % für den Artikel aufweist, wird die im Feld Aktion bei Widerspruch ausgewählte Option Fehler ausgelöst. In diesem Fall wird eine Fehlermeldung angezeigt und der Verkaufsauftrag wird nicht freigegeben.
    • Wenn auf der Artikelkarte kein Mindestdeckungsbeitrag hinterlegt ist, greift ausschließlich die erste Regel mit einem Deckungsbeitrag über 20%.
