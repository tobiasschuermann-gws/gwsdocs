    Proformabelege

    Proformabelege können entweder manuell erstellt werden oder aus Verkaufsangeboten und Verkaufsaufträgen generiert werden. Diese Belege sind so strukturiert, dass sie vielseitig verwendet werden können (zum Beispiel als Proformalieferschein).

    So richten Sie Proformabelege ein

    So aktivieren Sie die Proformalieferung

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Ein-/Ausschalter ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister ERP, Gruppe Proformabelege den Schalter Proformalieferung aktivieren.

    So hinterlegen Sie Proformabelegnummern

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Debitoren und Verkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister Proformabelege den Schalter Verkaufslieferungsnummern für Proformabelege verwenden. Klicken Sie alternativ in das Feld Proformabelegnummern, wenn Sie selbst definierte Nummernserien nutzen möchten und wählen Sie einen Datensatz aus oder klicken Sie auf Neu, um einen neuen Datensatz anzulegen. Weitere Informationen finden Sie unter Nummernserien erstellen.
    3. Klicken Sie optional in das Feld Mengenänderung in Auftrag übernehmen und wählen Sie eine der Optionen aus:
      • Abfrage (empfohlen): Nachdem der Verkaufsauftrag gebucht ist, wird ein Dialogfenster angezeigt, um mögliche Mengenänderungen zu übernehmen.
      • Nie: Die Mengenänderungen werden nie übernommen.
      • Immer: Die Mengenänderungen werden immer übernommen.
    4. Aktivieren Sie optional den Schalter Einstandspreis als VK-Preis verwenden.

    So richten Sie den Bericht "Proformalieferung" in den erweiterten Berichtseinstellungen ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Erweiterte Berichtseinstellungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Berichts-ID den Wert 5490568 ein für den Bericht Proformalieferung.

    So richten Sie den Bericht "Proforma-Rechnung" in den erweiterten Berichtseinstellungen ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Erweiterte Berichtseinstellungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Berichts-ID den Wert 5374126 ein für den Bericht Proforma-Rechnung.

    So richten Sie den Bericht "Proformabeleg" in der Berichtsauswahl - Verkauf ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Berichtsauswahl - Verkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie in das Feld Verwendung und wählen Sie den Wert Proformabeleg aus.
    3. Geben Sie im Feld Berichts-ID den Wert 5490568 ein für den Bericht Proformalieferung.

    So richten Sie den Bericht "Proforma-Rechnung" in der Berichtsauswahl - Verkauf ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Berichtsauswahl - Verkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie in das Feld Verwendung und wählen Sie den Wert Proforma-Rechnung aus.
    3. Geben Sie im Feld Berichts-ID den Wert 5374126 ein für den Bericht Proforma-Rechnung.

    So erstellen Sie einen Proformalieferschein

    Hier am Beispiel eines Verkaufsauftrags:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Verkaufsaufträge ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie einen Verkaufsauftrag. Klicken Sie alternativ im Menüband auf die Schaltfläche Neu, um einen neuen Verkaufsauftrag zu erstellen. Füllen Sie alle relevanten Belegkopffelder aus und fügen Sie Belegzeilen hinzu.
    3. Klicken Sie im Menüband auf die Schaltfläche Start > Buchen....
    4. Wählen Sie die Option Proforma Lieferung aus und bestätigen Sie Ihre Auswahl mit OK.
    Hinweis
    • Im Proformalieferschein wird ausschließlich die Menge aus dem Feld Zu liefern übernommen.
    • Sobald ein Verkaufsauftrag gebucht ist, erfolgt die Verknüpfung mit der Infobox Belegzeilenverknüpfung.
    • Zudem wird im Verkaufsauftrag im Inforegister Zeilen das Feld Menge in Proformalieferschein gefüllt.

    So ändern Sie einen Proformalieferschein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Proformalieferscheine ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie einen Proformalieferschein.
    3. Geben Sie im Feld Menge die zutreffende Menge ein.

    4. Bestätigen Sie das Dialogfenster mit Ja.

      Hinweis

      Das Dialogfenster erscheint nur, wenn Sie, wie in So hinterlegen Sie Proformabelegnummern beschrieben, die Option Abfrage gewählt haben.

    5. Klicken Sie im Menüband auf die jeweils zutreffende Schaltfläche:
      • Buchen: Buchen Sie den Verkaufsauftrag als Lieferung zum aktuellen Proformabeleg.
      • Senden: Senden Sie den Proformabeleg an den Debitoren.
      • An E-Mail anhängen: Senden Sie den Proformabeleg mit allen verknüpften Dokumenten per E-Mail.
      • Proformalieferung drucken: Drucken Sie den Proformabeleg aus.

    So buchen Sie einen Proformalieferschein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Proformalieferscheine ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie einen oder mehrere Proformalieferscheine.
    3. Klicken Sie im Menüband auf das Symbol Drop-Down der Schaltfläche Ausgewählte buchen > [Ihre Schaltfläche].
    4. Bestätigen Sie das Dialogfenster mit Ja.

    So erstellen Sie eine Proformarechnung

    Hier am Beispiel eines Verkaufsauftrags, der im Anschluss ausgedruckt werden soll:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Verkaufsaufträge ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie einen Verkaufsauftrag. Klicken Sie alternativ im Menüband auf die Schaltfläche Neu, um einen neuen Verkaufsauftrag zu erstellen. Füllen Sie alle relevanten Belegkopffelder aus.
    3. Klicken Sie im Menüband auf die Schaltfläche Start > Proformarechnung senden....
    4. Klicken Sie in das Feld Drucker und wählen Sie eine Option (zum Beispiel Ja (Standardeinstellungen verwenden)) aus und bestätigen Sie Ihre Auswahl mit OK.
