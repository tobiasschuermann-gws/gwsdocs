    Bezahlung bei Lieferung

    Für Ware, die direkt bei der Auslieferung bezahlt werden soll, können Sie mit der Buchung der Lieferung auch die dazugehörige Rechnung buchen und beides gleich ausdrucken. Auf dem Lieferschein wird zudem der Vermerk "Bezahlung bei Lieferung" angedruckt, so dass sichergestellt ist, dass die Ware nur gegen Bezahlung ausgeliefert wird.

    So hinterlegen Sie in der Debitorenkarte den Zahlungsbedingungscode
    Bezahlung bei Lieferung

    1. Öffnen Sie eine Debitorenkarte.
    2. Klicken Sie im Inforegister Zahlungen in das Feld Zlg.-Bedingungscode und wählen Sie den Datensatz BBL aus.
      Der Code wird dann für diesen Debitor bei der Erstellung eines Verkaufsauftrags oder eines Warenausgangs automatisch eingefügt.
    Hinweis

    Setzen Sie auf der Seite Zahlungsbedingungen für den Code BBL im Feld Bezahlung bei Lieferung einen Haken. So wird bei der Buchung einer Lieferung automatisch auch eine Rechnung erstellt.

    So richten Sie für den Druck von Lieferschein und Rechnung eine Belegvorlage für den Verkaufsauftrag ein

    Im Verkaufsauftrag ist es nicht möglich, mit einem Klick auf eine Schaltfläche gleichzeitig einen Beleg zu buchen und drucken. Deshalb ist hier eine Belegvorlage einzurichten.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Belegvorlagen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Inforegister Allgemein im Feld Code einen Code ein (zum Beispiel DRUCKLIEF).
    4. Klicken Sie in das Feld Belegart und wählen Sie die Option Verkaufsauftrag aus.
    5. Klicken Sie dann im Inforegister Funktionen in das Feld Drucken nach Buchung und wählen Sie die Option Lieferung aus.

    So buchen und drucken Sie Lieferschein und Rechnung aus dem Verkaufsauftrag heraus

    1. Öffnen oder erstellen Sie einen Verkaufsauftrag.
    2. Klicken Sie im Inforegister Allgemein in das Feld Belegvorlage und wählen Sie den Code aus, den Sie für die Belegvorlage angelegt haben (in unserem Beispiel DRUCKLIEF).
    3. Klicken Sie im Menüband auf die Schaltfläche Start > Buchen....
    4. Bestätigen Sie im Dialogfenster die zusätzliche Buchung der Rechnung mit Ja.
    5. Klicken Sie im Dialogfenster auf Nein, wenn die gebuchte Rechnung nicht geöffnet werden soll.
    Hinweis

    Zudem können Sie auch hier die Lieferung und die dazugehörige Rechnung gleichzeitig buchen und ausdrucken, wenn der Zahlungsbedingungscode BBL hinterlegt ist:

    • Warenausgang: Klicken Sie im Menüband auf die Schaltfläche Start > Warenausgang buchen > Buchen und drucken.
    • Lagerkommissionierung: Klicken Sie im Menüband auf die Schaltfläche Buchen und drucken....
    • Logistikterminal: Ist im Feld Druckart in der Logistik Terminal Einrichtung der Druck des Lieferscheins eingerichtet, wird auch die Rechnung gedruckt. Der Befehl für das Buchen ist POSTWHSE.
