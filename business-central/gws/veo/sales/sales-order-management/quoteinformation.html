    Angebotsinformation im Verkaufsauftrag

    Sobald Sie im Verkaufsauftrag eine Artikel- oder Ressourcennummer eingeben, wird geprüft, ob es für einen Debitor offene Verkaufsangebote mit dem gleichen Artikel oder der gleichen Ressource gibt. Wenn es solche Verkaufsangebote gibt, erscheint im Verkaufsauftrag eine Angebotsinformation.

    So richten Sie die Angebotsinformation ein

    In der Einrichtung Debitoren und Verkauf legen Sie fest, ob die Angebotsinformation bei der Auftragserfassung zuerst als Benachrichtigung oder direkt als Angebotsliste angezeigt werden soll. Zudem können Sie Optionen auswählen, um die Liste der angezeigten Verkaufsangebote einzuschränken.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Debitoren und Verkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Verkaufsangebote in das Feld Angebotsinformation im Auftrag anzeigen und wählen Sie eine Option aus:
      • Benachrichtigung: Wenn es für einen Debitor passende Verkaufsangebote gibt, erscheint im Verkaufsauftrag eine Benachrichtigung.
      • Aus Angebotsliste auswählen: Wenn es für einen Debitor passende Verkaufsangebote gibt, öffnet sich direkt eine Angebotsliste.
    3. Klicken Sie optional in das Feld Angebotsinformation einschränken und wählen Sie eine Option aus: Lief. an Code (Lieferadresse), Verkaufskanal (zum Beispiel Strecke) oder Lief. an Code und Verkaufskanal. Eine Angebotszeile wird dann vorgeschlagen, wenn der Wert in der ausgewählten Option den Wert des Auftrags hat oder leer ist.
    Hinweis
    • In der Angebotsinformation werden nur Angebotszeilen von Verkaufsangeboten berücksichtigt, die freigegeben sind und bei denen das Datum im Feld Gültig bis-Datum für Angebot nicht abgelaufen ist.
    • Sie können ein Verkaufsangebot aus der Angebotsinformation ausschließen, indem Sie im Inforegister Allgemein den Schalter Einmaliges Angebot aktivieren. Dieses Verkaufsangebot wird dann in den Angebotsinformationen nie angezeigt.

    So prüfen und übernehmen Sie Angebotszeilen

    In der Angebotsinformation können Sie die gelisteten Verkaufsangebote öffnen und prüfen. Zudem können Sie aus den Angebotszeilen die Preise, Rabatte, Mengen und Einheiten in die Verkaufsauftragszeilen übernehmen.

    So öffnen Sie ein Verkaufsangebot

    • Wenn Sie auf der Seite Einrichtung Debitoren und Verkauf die Option Benachrichtigung als Angebotsinformation ausgewählt haben:
      Klicken Sie in der Benachrichtigung auf die Schaltfläche Angebotszeilen anzeigen, um die Seite Verkaufszeilen zu öffnen. Markieren Sie eine Zeile und klicken Sie im Menüband auf die Schaltfläche Beleg anzeigen.
    • Wenn Sie auf der Seite Einrichtung Debitoren und Verkauf die Option Aus Angebotsliste auswählen als Angebotsinformation ausgewählt haben:
      Sobald Sie einen Artikel in den Verkaufszeilen anlegen, öffnet sich die Seite Verkaufszeilen. Markieren Sie eine Zeile und klicken Sie im Menüband auf die Schaltfläche Beleg anzeigen.

    So übernehmen Sie Zeilen aus einem Verkaufsangebot

    • Markieren Sie auf der Seite Verkaufszeilen eine Zeile und bestätigen Sie Ihre Auswahl mit OK.
    • Wenn Sie auf der Seite Einrichtung Debitoren und Verkauf die Option Benachrichtigung als Angebotsinformation ausgewählt haben:
      Klicken Sie in der Benachrichtigung auf die Schaltfläche Vom zuletzt erstellten Angebot übernehmen, wenn Sie die Angebotszeile aus dem zuletzt erstellten Verkaufsangebot übernehmen wollen.
