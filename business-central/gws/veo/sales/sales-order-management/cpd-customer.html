    CpD Debitoren

    Wenn für Einmalkunden (Privatkunden oder Gesellen) Ware bestellt werden muss und eine Lieferadresse erforderlich ist, kann der Vorgang nicht über den Barverkauf abgewickelt werden. Um für diese Kunden eine Bestellung anlegen zu können, brauchen Sie ein Debitorenkonto.
    In gevis ERP | VEO können Sie im Schnellverfahren einen Debitor erstellen, ohne ein komplettes Debitorenkonto anzulegen. Zudem ist es möglich, CpD Debitoren wieder freizugeben: Die Debitorendaten werden gelöscht und die Debitorennummer kann neu vergeben werden. Auf diese Weise wird die Anzahl der CpD Debitoren limitiert.

    Um diese Funktion in den Verkaufsbelegen oder im Debitorencockpit nutzen zu können, richten Sie zunächst eine Konfigurationsvorlage für CpD Debitoren ein. Mit dieser Konfigurationsvorlage legen Sie fest, welche Felder für den neuen Debitor automatisch mit Werten befüllt werden. Definieren Sie dann Pflichtfelder, die bei der Anlage von CpD Debitoren von den Anwendern ausgefüllt werden müssen und ordnen Sie diese zum Schluss der neuen Konfigurationsvorlage zu.

    Hinweis

    Sollten Felder oder Schalter in Ihrer Ansicht fehlen, können Sie diese über Personalisieren einblenden. Weitere Informationen finden Sie unter Ihren Arbeitsbereich personalisieren.

    So richten Sie eine Konfigurationsvorlage für CpD Debitoren ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Konfigurationsvorlagen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Code einen Code (zum Beispiel CPD) und im Feld Beschreibung eine Kurzbeschreibung für diesen Code ein.
    4. Geben Sie im Feld Table ID den Wert 18 ein für die Tabelle Debitor.
    5. Klicken Sie im Inforegister Zeilen in das Feld Feldname und wählen Sie ein Feld aus, das für Einmalkunden vorbelegt sein soll (zum Beispiel Debitorenbuchungsgruppe). Bestätigen Sie Ihre Auswahl mit OK.
    6. Wählen Sie im Feld Standardwert über das Dreipunkt-Menü den zutreffenden Datensatz für das zuvor ausgewählte Feld aus.

    So richten Sie CpD Pflichtfeldergruppen ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld CpD Pflichtfeldergruppen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Code einen Code (zum Beispiel CPDPFLICHT) und im Feld Name eine Kurzbeschreibung für die Pflichtfeldergruppe ein.
    4. Klicken Sie im Menüband auf die Schaltfläche Felder, um die Seite CpD Pflichtfelder zu öffnen.
    5. Wählen Sie im Feld Feldnr. über das Dreipunkt-Menü ein Feld aus, das bei der Schnellanlage des CpD Debitors von Anwendern ausgefüllt werden muss (zum Beispiel Name oder Adresse) und bestätigen Sie Ihre Auswahl mit OK.
    6. Definieren Sie im Feld Sortierreihenfolge die Reihenfolge der Felder bei der Erfassung eines neuen Debitors.
    7. Setzen Sie im Feld Optional einen Haken, wenn ein Feld zwar angezeigt werden soll, die Eingabe in diesem Feld jedoch nicht verpflichtend ist.

    So richten Sie CpD Zuordnungen ein

    Hier am Bespiel unserer Konfigurationsvorlage CPD und der Pflichtfeldgruppe CPDPFLICHT:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld CpD Zuordnungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie in das Feld Konfigurationsvorlagencode und wählen Sie den Datensatz CPD aus.
    3. Klicken Sie in das Feld Pflichtfeldergruppe Code und wählen Sie den Datensatz CPDPFLICHT aus.

    So legen Sie CpD Debitoren in den Verkaufsbelegen an

    Einen CpD Debitor können Sie im Verkaufsauftrag und Verkaufsangebot sowie in der Verkaufsrechnung, Verkaufsreklamation und Verkaufsgutschrift anlegen.

    Hier am Beispiel eines Verkaufsauftrags:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Verkaufsaufträge ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.

    3. Klicken Sie im Menüband auf die Schaltfläche Aktionen > Funktion > CpD Debitor.

      Hinweis

      Wenn Sie mehrere Konfigurationsvorlagen für Einmalkunden (zum Beispiel CPD Inland und CPD Ausland) angelegt haben, öffnet sich zunächst die Seite CpD Zuordnungen. Markieren Sie die Zeile für den zutreffenden Konfigurationsvorlagencode und bestätigen Sie Ihre Auswahl mit OK.

    4. Geben Sie auf der Seite Dialog Werte für die zuvor definierten Pflichtfelder ein und bestätigen Sie Ihre Eingabe mit OK.
      Im Verkaufsauftrag sind jetzt im Inforegister Allgemein alle Pflichtfelder für den neuen Debitor gefüllt und im Feld Debitorennr. wurde eine Debitorennummer zugewiesen.

      Hinweis

      In der Debitorenkarte des neu angelegten Debitors ist zusätzlich im Inforegister Allgemein im Feld CpD Kunden der Wert In Verwendung gesetzt.

    So legen Sie CpD Debitoren im Debitorencockpit an

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Debitorencockpit ein und klicken Sie auf das übereinstimmende Suchergebnis.

    2. Klicken Sie im Menüband auf die Schaltfläche Aktionen > Funktionen > CpD Debitor anlegen.

      Hinweis

      Wenn Sie mehrere Konfigurationsvorlagen für Einmalkunden (zum Beispiel CPD Inland und CPD Ausland) angelegt haben, öffnet sich zunächst die Seite CpD Zuordnungen. Markieren Sie die Zeile für den zutreffenden Konfigurationsvorlagencode und bestätigen Sie Ihre Auswahl mit OK.

    3. Geben Sie auf der Seite Dialog Werte für die zuvor definierten Pflichtfelder ein und bestätigen Sie Ihre Eingabe mit OK.
      Im Debitorencockpit ist jetzt im Inforegister Debitor Informationen im Feld Nr. die Debitorennummer für den neu angelegten Debitor eingetragen.

      Hinweis

      In der Debitorenkarte des neu angelegten Debitors ist zusätzlich im Inforegister Allgemein im Feld CpD Kunden der Wert In Verwendung gesetzt.

    4. Klicken Sie im Menüband des Debitorencockpits auf die Schaltfläche Start > [Ihre Schaltfläche] (zum Beispiel Verkaufsangebot,Verkaufsrechnung,Verkaufsauftrag,Verkaufsgutschrift oder Verkaufsreklamation), um einen Beleg zu erstellen.

    So geben Sie CpD Debitoren wieder frei

    Es werden nur CpD Debitoren freigegeben, für die keine Aufträge oder offene Debitorenposten existieren. Durch die Freigabe eines CpD Debitors wird in der Debitorenkarte im Feld CpD Kunde der Wert Freigegeben gesetzt. Die Debitorennummer kann dann für einen neuen CpD Debitor verwendet werden. Voraussetzung ist, dass der neue CpD Debitor mit derselben Konfigurationsvorlage erstellt wird.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld CpD Debitoren freigeben ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie optional im Inforegister Filter: Debitor in das Feld Nr. und wählen Sie einen Datensatz aus, um einen einzelnen CpD Debitor freizugeben.
    3. Bestätigen Sie die Freigabe mit OK.

    Um einen Aufgabenwarteschlangenposten für die Freigabe von CpD Debitoren zu erstellen, gehen Sie vor wie in Aufgabenwarteschlangenposten beschrieben.
