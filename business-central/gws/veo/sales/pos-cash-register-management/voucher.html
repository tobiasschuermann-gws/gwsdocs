    Gutscheine

    Im Kassenverkauf können Sie Gutscheine verkaufen, die Bezahlung per Gutschein ausführen oder den Restwert eines Gutscheins auszahlen.

    So richten Sie den Verkauf von Gutscheinen ein

    Um den Verkauf von Gutscheinen einzurichten, gehen Sie vor wie in So richten Sie den Verkauf von Gutscheinen ein beschrieben.

    So verkaufen Sie einen Gutschein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kassenverkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie in der Infobox Zahlung, Gruppe Stammdaten auf die Kachel Alle Artikel.
    3. Markieren Sie die Zeile mit dem Gutscheinartikel und bestätigen Sie Ihre Auswahl mit OK.

    4. Geben Sie im Feld Gutscheinwert einen Betrag ein und bestätigen Sie Ihre Eingabe mit OK.

      Hinweis

      In Inforegister Artikel wird eine Verkaufszeile für den erstellten Gutschein hinzugefügt. Im Feld Gutscheinnummer finden Sie die automatisch erstellte Gutscheinnummer, die auf dem Gutschein angedruckt wird.

    5. Klicken Sie in der Infobox Zahlung, Gruppe Aktionen auf eine Kachel, um den Verkauf abzuschließen. Weitere Informationen zum Kassiervorgang finden Sie unter [Kassenvorgänge].

    So führen Sie die Bezahlung per Gutschein aus

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kassenverkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Scannen Sie einen Artikel.
    3. Klicken Sie in der Infobox Zahlung, Gruppe Aktionen auf die Kachel Mehrzweck Gutschein.
    4. Scannen Sie den Gutschein.

    5. Bestätigen Sie die Gutscheinzahlung mit OK.

      Hinweis

      Nach jeder Bezahlung mit einem Gutschein wird auf der Seite Gutscheinposten eine neue Zeile erstellt. Im Feld Postenart ist der Wert Eingelöst gesetzt.

    So zahlen Sie den Restbetrag eines Gutscheins aus

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kassenverkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Kassenarbeitsplatz > Gutscheine.
    3. Markieren Sie einen Gutschein.
    4. Klicken Sie im Menüband auf die Schaltfläche Start > Rückabwickeln.
    5. Bestätigen Sie in der Zahlungskarte den Zahlbetrag mit OK.
    6. Zahlen Sie den Restbetrag aus.

    So sehen Sie alle Informationen zum Gutschein ein

    Auf der Seite Gutscheine finden Sie eine Liste aller bisher erstellten Gutscheine und die dazugehörigen Posten.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kassenverkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Kassenarbeitsplatz > Gutscheine.
    3. Klicken Sie im Menüband auf die passende Schaltfläche:
      • Verwalten > Ansicht: Sie können allgemeine Informationen zu einem Gutschein einsehen.
      • Posten > Gutscheinposten: Sie können einzelne Gutscheinposten einsehen.
      • Berichte > Gutscheine drucken: Sie können einen Gutschein manuell drucken.
