    Fiskaltrust einrichten

    Mithilfe der Schnittstelle von gevis ERP | VEO können Sie Fiskaltrust für Ihr Kassensystem nutzen. Sie können alle Transaktionen lückenlos erfassen und die Abrechnung der Mehrwertsteuer vollständig dokumentieren.

    So hinterlegen Sie eine E-Mail-Adresse für Fehlermeldungen

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Fiskaltrust Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Geben Sie im Feld E-Mail Fiskaltrust Admin eine E-Mail-Adresse ein, um Nachrichten an einen Administrator senden zu können.
    3. Aktivieren Sie optional den Schalter Benutzermeldungen überspringen, wenn Nachrichten nicht an den Benutzer, sondern direkt an den Administratoren gesendet werden sollen.
    4. Aktivieren Sie optional im Inforegister Abschluss den Schalter TAR-Export umgehen, wenn der Download einer TAR-Datei bei täglichen, monatlichen und jährlichen Abschlüssen umgangen werden soll.

    So prüfen Sie das Fiskaltrust Mehrwertsteuer-ID Mapping

    Von Fiskaltrust werden verschiedene Mehrwertsteuer-IDs vorgegeben und den Buchungsgruppen in gevis ERP | VEO zugeordnet (zum Beispiel die Fiskaltrust Mehrwertsteuer-ID Regelsteuersatz (19.00%) der Mehrwertsteuer-Geschäftsbuchungsgruppe INLAND und der Mehrwertsteuer-Produktbuchungsgruppe MWST.19). Diese Datensätze werden bei der Installation erstellt.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Fiskaltrust MwSt.-ID Mapping ein und klicken Sie auf das übereinstimmende Suchergebnis.

    2. Kontrollieren und korrigieren Sie bei Bedarf die Zuordnungen.

      Hinweis

      Insbesondere in Zeilen, in denen im Feld MwSt.% die Option 0,00 zugeordnet wurde, ist zu prüfen, ob im Feld Fiskaltrust MwSt-ID der richtige Wert angezeigt wird: Nicht Steuerbar (0,00%) oder Umsatzsteuerfrei (0,00%).

    So richten Sie eine Cashbox ein

    Eine Cashbox ist die Schnittstelle zwischen gevis ERP | VEO und Fiskaltrust. Richten Sie pro Filiale eine Cashbox ein.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Fiskaltrust Cashboxen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Geben Sie im Feld Cashbox ID den von Fiskaltrust zur Verfügung gestellten Wert ein.
    3. Geben Sie im Feld Name den Namen der Cashbox ein.
    4. Geben Sie im Feld Zugriffstoken den von Fiskaltrust zur Verfügung gestellten Zugriffstoken ein.

    So richten Sie ein Terminal ein

    Richten Sie pro Kassenschublade ein Terminal ein.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Fiskaltrust Terminals ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie in das Feld Kassenschublade und wählen Sie einen Datensatz aus.
    3. Geben Sie im Feld Kassenseriennummer einen Wert ein.
    4. Klicken Sie in das Feld Cashbox ID und wählen Sie einen Datensatz aus.
    5. Geben Sie im Feld URL die von Fiskaltrust zur Verfügung gestellte URL ein.
    6. Klicken Sie optional im Menüband auf die Schaltfläche Verbindungstest, um zu prüfen, ob die Verbindung erfolgreich hergestellt werden kann.
    7. Klicken Sie im Menüband auf die Schaltfläche Registrieren, um den Kassenarbeitsplatz zu registrieren.
    Hinweis
    • Klicken Sie im Menüband auf die Schaltfläche Deregistrieren, wenn Sie einen Kassenarbeitsplatz unwiderruflich abmelden wollen.
    • Klicken Sie im Menüband auf die Schaltfläche Journal exportieren, wenn Sie Journal-IDs exportieren wollen.

    So richten Sie Timeouts ein

    Die Zeiten, nach denen die Anfragen an Fiskaltrust abgebrochen werden, sind standardmäßig vorbelegt. Abweichende Timeouts können allgemein oder für einzelne Terminals eingerichtet werden.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Fiskaltrust Timeouts ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie in das Feld Terminal ID und wählen Sie einen Datensatz aus, um ein Timeout für ein einzelnes Terminal einzurichten. Lassen Sie alternativ das Feld leer, wenn Sie ein Timeout für alle Terminals einrichten wollen.
    3. Klicken Sie in das Feld Anwendungsfall und geben Sie den von Fiskaltrust vorgegebenen Wert ein.
    4. Klicken Sie in das Feld Beschreibung und geben Sie eine Kurzbeschreibung für den Anwendungsfall ein.
    5. Klicken Sie in das Feld Timeout und geben Sie die Zeit ein, nach der eine Anfrage abgebrochen werden soll.

    So richten Sie Journal Arten ein

    Journal Arten sind standardmäßig vorbelegt. Sie können für den Export des Journals weitere von Fiskaltrust bereitgestellte Journal Arten hinzufügen.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Fiskaltrust Journal Arten ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Geben Sie im Feld Journal ID das Mehrwertsteuer-Kennzeichen ein, das zu den Gebührenpositionen in Fiskaltrust hinzugefügt werden soll.
    3. Geben Sie im Feld Beschreibung eine Kurzbeschreibung des Mehrwertsteuer-Kennzeichens ein.
    4. Geben Sie im Feld Version die Versionsnummer ein, mit der die Exportanfrage an die API von Fiskaltrust gesendet wird.
