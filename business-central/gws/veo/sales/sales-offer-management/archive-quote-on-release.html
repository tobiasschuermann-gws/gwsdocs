    Archivierung von Verkaufsangeboten bei Freigabe

    Wenn Sie für einen Kunden mehrere Verkaufsangebote erstellt haben, wird bei jeder Freigabe des Verkaufsangebots eine archivierte Versionerstellt. Entscheidet sich der Kunde letztendlich für eine vorherige Version, können Sie den archivierten Beleg jederzeit wiederherstellen.

    So richten Sie die Archivierung von Verkaufsangeboten bei Freigabe ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einrichtung Debitoren und Verkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Aktivieren Sie im Inforegister Archivierung den Schalter Angebot bei Freigabe archivieren.

    Weitere Informationen finden Sie unter Verkaufsangebote einrichten.

    So archivieren Sie eine Version des Verkaufsangebots

    1. Öffnen oder erstellen Sie ein Verkaufsangebot.
    2. Klicken Sie im Menüband auf die Schaltfläche Freigeben.
      Das Verkaufsangebot wird automatisch im Archiv gespeichert.
    3. Klicken Sie im Menüband auf das Symbol Drop-Down der Schaltfläche Freigeben und klicken Sie dann auf Status zurücksetzen, wenn Sie eine neue Version des Verkaufsangebots erstellen wollen.
      Sie können das Verkaufsangebot jetzt ändern.

    4. Klicken Sie erneut im Menüband auf die Schaltfläche Freigeben, um das geänderte Verkaufsangebot als neue Version zu archivieren.

      Hinweis

      Sie können auf diese Weise beliebig viele Versionen eines Verkaufsangebots erstellen.

    So stellen Sie archivierte Versionen eines Verkaufsangebots wieder her

    1. Öffnen Sie ein Verkaufsangebot.
      Im Inforegister Allgemein im Feld Anzahl archivierter Versionen sehen Sie, wie viele Versionen für ein Verkaufsangebots bereits archiviert wurden.
    2. Klicken Sie in das Feld Anz. archivierter Versionen, um die Seite Verkaufsübersichtsarchiv zu öffnen.
      Hier finden Sie alle archivierten Versionen des aktuell geöffneten Verkaufsangebots.
    3. Wählen Sie eine Versionsnummer aus und klicken Sie in das Feld Nr., um die Seite Verkaufsangebotsarchiv zu öffnen.

    4. Klicken Sie im Menüband auf die Schaltfläche Wiederherstellen..., um die archivierte Version in das Verkaufsangebot zu übertragen.

      Hinweis

      Sie können eine archivierte Version nur öffnen, wenn der Status des Verkaufsauftrags vorher auf Offen zurückgesetzt wurde.

    5. Bestätigen Sie im Dialogfenster die Wiederherstellung der gewählten Version mit Ja.
    6. Bestätigen Sie mit OK, wenn eine weitere Version archiviert werden soll.
    7. Bestätigen Sie die Archivierung mit OK.
    8. Schließen Sie die Seite Verkaufsangebotssarchiv über die Schaltfläche Zurück.
    9. Schließen Sie die Seite Verkaufsübersichtsarchiv mit einem Klick auf die Schaltfläche Schließen.
    10. Klicken Sie auf der Seite Verkaufsangebote in das Feld Nr., um die wiederhergestellte Version zu öffnen.

    So finden Sie alle archivierten Versionen eines Verkaufsangebots im Verkaufsangebotsarchiv

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Verkaufsangebotsarchive ein und klicken Sie auf das übereinstimmende Suchergebnis.

    2. Klicken Sie in das Feld Nr., um ein archiviertes Verkaufsangebot zu öffnen.

      Hinweis

      Wenn Ihnen die Nummer des gesuchten Verkaufsangebots nicht bekannt ist, klicken Sie im Menüband auf die Schaltfläche Suchen und geben Sie einen Suchbegriff ein, um zum Beispiel nach dem Debitorennamen, der Debitorennummer, der Verkaufsangebotsnummer oder dem Kontakt zu suchen.
