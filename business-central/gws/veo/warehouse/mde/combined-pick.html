    Sammelkommissionierung

    Ist ein Artikel in mehreren Verkaufsaufträgen angelegt, kann die Arbeit im Lager mithilfe der Sammelkommissionierung effektiver gestaltet werden, da der Lagermitarbeiter nicht mehrmals denselben Lagerplatz aufsuchen muss, um einen Artikel zu kommissionieren. In der Sammelkommissionierung werden Artikel und Varianten mit dem gleichen Einheitencode und Lagerplatz zusammengefasst und gesammelt mit dem MDE-Gerät erfasst. Die Aufteilung findet automatisch bei der Registrierung statt. Bevor Sie eine Sammelkommissionierung mit Ihrem MDE-Gerät durchführen können, muss zunächst die MDE-Einrichtung abgeschlossen sein.

    So richten Sie die mobile Menüoption "Sammelkommissionierung" ein

    Damit Sie die Sammelkommissionierung nutzen können, muss zunächst eine Zuordnung zu einer mobilen Menüoption erfolgen. Verfahren Sie wie in MDE einrichten beschrieben.

    Hinweis

    Verwenden Sie folgenden Wert für die Einrichtung:

    • Mobile Menüoptionen: CombinedPick (Sammelkommissionierung)

    So richten Sie die Sammelkommissionierung in der Lagerortkarte ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Lagerorte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Lagerortkarte.
    3. Klicken Sie im Inforegister Allgemein in das Feld Kumulierte Lageraktivitätszeilen und wählen Sie eine Option aus:
      • (leer): Es findet keine Zusammenfassung der Kommissionierzeilen statt. (Standard)
      • Wenn kummulierbar: Eine Zusammenfassung von Kommissionierzeilen findet nur statt, wenn mindestens ein Artikel der Kommissionierung zusammengefasst werden kann.
      • Immer: Eine Zusammenfassung von Kommissionierzeilen findet immer statt.
    Hinweis
    • Eine Zusammenfassung von Kommissionierzeilen erfolgt nur dann, wenn Lagerort, Lagerplatz, Artikelnummer, Variantencode und Einheitencode identisch sind.
    • Auf der Seite Kumulierte Lageraktivitätszeilen finden Sie eine Übersicht der zusammengefassten Kommissionierzeilen.

    So aktivieren Sie die Behälterkommissionierung

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kommissionierungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Kommissionierung.

    3. Klicken Sie ins Feld Behälterkommissionierung Aktiviert und wählen Sie die Option Ja für Behälterkommissionierung aus.

      Hinweis

      Wenn der Wert aus der Mobile WMS Einrichtung gezogen werden soll, wählen Sie hier Standard (Ja) oder Standard (Nein) aus - jeweils in Abhängigkeit davon, was in der Mobile WMS Einrichtung aktiviert ist.

    Führen Sie anschließend mit dem MDE-Gerät eine Sammelkommissionierung durch.

    Hinweis

    Auf der Seite Warenausgang wird die erfasste Menge der Sammelkommissionierung automatisch auf die einzelnen Kommissionierzeilen im Inforegister Zeilen im Feld Menge kommissioniert verteilt. Die erfassten Behälternummern einer Behälterkommissionierung finden Sie im Feld Sammelbehälter-ID. Werden mehrere Behälternummern zu einer Warenausgangszeile erfasst, so werden diese durch ein Kommazeichen getrennt.
