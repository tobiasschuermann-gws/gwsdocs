    Lagerorte

    So richten Sie einen Lagerort ein

    Um einen Lagerort zu erstellen, verfahren Sie wie in Lagerorte einrichten beschrieben.

    So richten Sie den Standardlagerort für einen Artikel ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Artikel ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Artikelkarte.
    3. Klicken Sie im Inforegister Lagerbestand in das Feld Standardlagerortcode und wählen Sie einen Lagerort aus, den Sie als Standard definieren möchten.
    Hinweis
    • Sofern ein vorab festgelegter Lagerort in der Debitorenkarte hinterlegt ist, wird automatisch eine Hintergrundberechnung des Lagerbestands durchgeführt. Dieser Lagerbestand wird im Inforegister Lagerbestand im Feld Lagerbestand Primärlagerort in der Artikelkarte angezeigt.
    • Sobald ein Artikel in einem Verkaufsauftrag selektiert wird, überträgt sich der festgelegte Standardlagerort automatisch in die Artikelzeile.
    • Bei Auswahl eines Artikels in einem Fertigungsauftrag wird der vorab bestimmte Standardlagerort automatisch übernommen und als Lagerortcode in das Inforegister Buchung eingefügt. Es ist wichtig, dass das Feld Herkunftsnr. im Inforegister Allgemein mit einem Datensatz befüllt ist.

    So richten Sie einen benutzerbezogenen Lagerort ein

    Wenn ein Mitarbeiter gezielt auf einen bestimmten Lagerort zugreifen soll, können Sie einen benutzerbezogenen Lagerort einrichten. Bei der Erstellung eines Verkaufsbelegs wird der zugewiesene Lagerort automatisch übernommen. Dies geschieht unabhängig von dem Lagerort, der in der Debitorenkarte hinterlegt ist. Es besteht weiterhin die Möglichkeit, den Lagerort im Verkaufsbeleg manuell anzupassen.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Benutzer Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie in das Feld Benutzer-ID und wählen Sie einen Benutzer aus.
    4. Klicken Sie in das Feld Lagerortcode und wählen Sie einen Lagerort aus, der diesem Benutzer zugeordnet werden soll.

    So sehen Sie den Lagerbestand des Primärlagerorts für einen Artikel ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Artikel ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Artikelkarte.
    3. Klicken Sie im Inforegister Lagerbestand in das Feld Lagerbestand Primärlagerort, um alle Artikelposten an diesem Primärlagerort einzusehen.
