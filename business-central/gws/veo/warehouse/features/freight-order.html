    Frachtaufträge

    Frachtaufträge sind abrechenbare Lieferungen, die entweder mengenbasiert oder pauschal abgerechnet werden können. Sie können als eingehende Rechnungen sowohl für Einkaufs- als auch für Verkaufslieferungen fakturiert werden.

    So richten Sie den Frachtauftrag ein

    So hinterlegen Sie Frachtkosten

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Kreditoren & Einkauf Einr. ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Allgemein in das Feld Zu-/Abschlag Frachtauftrag Verkauf und Zu-/Abschlag Frachtauftrag Einkauf und wählen Sie einen Datensatz aus oder klicken Sie auf Neu, um einen neuen Datensatz anzulegen. Weitere Informationen finden Sie unter Artikelzuschläge für zusätzliche Handelskosten verwenden.

    So hinterlegen Sie Zusteller

    Zusteller, die für den Transport von Fracht eingesetzt werden sollen, müssen einem Kreditoren zugeordnet sein.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Zusteller ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie einen Zusteller. Klicken Sie alternativ im Menüband auf die Schaltfläche Neu, um einen neuen Zusteller anzulegen.
    3. Klicken Sie in das Feld Kreditorennr. und wählen Sie einen Kreditor aus.

    So richten Sie die Berichte in der Berichtsauswahl - Einkauf und in den erweiterten Berichtseinstellungen ein

    Weitere Informationen finden Sie unter Berichtsauswahl für Belege.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Berichtsauswahl - Einkauf ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie in das Feld Verwendung und wählen Sie die Option Frachtauftrag aus.
    3. Geben Sie im Feld Berichts-ID den Wert 5374141 ein für den Bericht Frachtauftrag.
    4. Klicken Sie im Menüband auf die Schaltfläche Erweiterte Berichtseinstellungen, um die Seite Berichtseinstellung für den Bericht Frachtauftrag zu öffnen.
    5. Gehen Sie vor wie in Erweiterte Berichtseinstellungen einrichten beschrieben.

    So erstellen und drucken Sie einen Frachtauftrag

    Es ist nur möglich, je Kreditor oder Debitor einen oder mehrere Belege auszuwählen und einen Frachtauftrag zu erstellen.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Frachtauftrag erstellen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie in das Feld Zusteller und wählen Sie einen Zusteller aus.
    3. Geben Sie im Feld Frachtkosten pauschal oder Frachtkosten pro Tonne einen Wert ein.
    4. Klicken Sie in das Feld Zu-/Abschlagverteilungsmethode und wählen Sie eine Option aus, um festzulegen, wie die Frachtkosten auf die Positionen verteilt werden sollen.
    5. Setzen Sie im Feld Zeile ausgewählt in jeder Zeile einen Haken, für die ein Frachtauftrag erstellt werden soll.
    6. Klicken Sie im Menüband auf die Schaltfläche Frachtauftrag erstellen.
    7. Klicken Sie optional im Menüband auf die Schaltfläche Zugehörig > Vor-/Nachtexte > [Ihre Schaltfläche].
    8. Gehen Sie vor wie in Vor- und Nachtexte für Belege beschrieben.
    9. Schließen Sie die Seite Textzuordnung über die Schaltfläche Zurück.
    10. Klicken Sie im Menüband auf die Schaltfläche Berichte > Frachtauftrag... .
    11. Geben Sie den Frachtauftrag über Senden an... oder Drucken aus.
    Hinweis
    • Die im Frachtauftrag enthaltenen Artikel der Einkaufsbestellung oder dem Verkaufsauftrag können Sie im Frachtauftrag im Inforegister Einkaufszeilen für Frachtauftrag oder Verkaufszeilen für Frachtauftrag einsehen.
    • Die verknüpften Belege können in der Einkaufsbestellung oder im Verkaufsauftrag über die Infobox Belegzeilenverknüpfung aufgerufen werden.
    • Bei Lieferung der Einkaufsbestellung erfolgt die Verteilung der Frachtkosten automatisch.

    So erstellen Sie einen Frachtauftrag direkt aus einem Beleg

    Ein Frachtauftrag kann direkt aus einer Einkaufsbestellung oder einem Verkaufsauftrag erstellt werden.
    Hier am Beispiel einer Einkaufsbestellung:

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einkaufsbestellungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Einkaufsbestellung.
    3. Klicken Sie im Menüband auf die Schaltfläche Aktionen > Funktion > Frachtauftrag erstellen.

    So erstellen Sie eine Einkaufsrechnung für den Frachtauftrag

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Einkaufsrechnungen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.

    3. Wählen Sie im Inforegister Allgemein im Feld Kreditorenname über das Dreipunkt-Menü einen Kreditor aus und bestätigen Sie Ihre Auswahl mit OK.

      Hinweis

      Der Kreditor, den Sie hier auswählen, muss identisch sein mit dem Kreditor, den Sie auf der Seite Zusteller hinterlegt haben.

    4. Geben Sie im Feld Kred.-Rechnungsnr. die Rechnungsnummer ein, die auf Ihrer Eingangsrechnung vermerkt ist.
    5. Klicken Sie im Menüband des Inforegisters Zeilen auf die Schaltfläche Zeile > Funktionen > Wareneingangszeilen abrufen... .
    6. Markieren Sie eine oder mehrere Wareneingangszeilen und bestätigen Sie Ihre Auswahl mit OK.
    7. Markieren Sie die Zeilen, bei denen im Feld Art der Wert Zu-/Abschlag (Artikel) zugeordnet ist.
    8. Klicken Sie im Menüband des Inforegisters Zeilen auf die Schaltfläche Zeile > Verknüpfte Informationen > Artikel Zu-/Abschlagszuweisung.
    9. Klicken Sie im Menüband auf die Schaltfläche Einkaufslieferzeilen holen.
    10. Markieren Sie eine oder mehrere Einkaufslieferzeilen und bestätigen Sie Ihre Auswahl mit OK.
    11. Geben Sie im Feld Menge für Zuweisung einen Wert ein.
    12. Schließen Sie die Seite Artikel Zu-/Abschl.-Zuw. (EK) - FRACHT Frachtzuschlag mit Schließen.
