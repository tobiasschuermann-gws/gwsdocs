    Grüner Punkt einrichten und verwalten

    Der Grüne Punkt ist ein international anerkanntes Zeichen, das auf Produkte und Verpackungen angebracht wird, die zur Wiederverwertung bestimmt sind. Diese Funktion ermöglicht Ihnen eine umfassende Auswertung des versendeten Verpackungsmaterials. Dafür werden Informationen über die verwendeten Verpackungen und die entsprechenden Mengen ausgegeben.

    So richten Sie Verpackungsmaterialien ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Verpackungsmaterialien ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Code eine Bezeichnung (zum Beispiel KARTON) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Kartongröße M) ein.

    So richten Sie Verpackungsgruppen ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Verpackungsgruppen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Code eine Bezeichnung (zum Beispiel KARTONVERPACKUNG) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Kartongröße M braun) ein.

    So richten Sie Verpackungsmengen ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Verpackungsmengen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie in das Feld Verpackungsgruppencode und wählen Sie einen Verpackungsgruppencode (zum Beispiel KARTONVERPACKUNG) aus.
    4. Klicken Sie in das Feld Verpackungsmaterial und wählen Sie ein Verpackungsmaterial (zum Beispiel KARTON) aus.
    5. Geben Sie im Feld Menge in Basiseinheit einen Wert (zum Beispiel 2,00) ein.

    So hinterlegen Sie einen Debitor als Selbstlizenzierer

    Mit der Lizenzierung kann der Debitor das Symbol des Grünen Punkts auf seinen Produkten verwenden. Mit dieser Einrichtung legen Sie fest, dass dieser die Verantwortung übernimmt, seine Verpackungen eigenständig beim Grünen Punkt anzumelden. Debitoren mit einer Lizenzierung werden bei der Auswertung der Verpackungsgruppen nicht berücksichtigt.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Debitoren ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Debitorenkarte.
    3. Aktivieren Sie im Inforegister Statistik den Schalter Selbstlizenzierer.

    So geben Sie eine Auswertung für Verpackungsgruppen aus

    Mithilfe der Auswertung werden die verwendeten Verpackungsmaterialien ausgegeben. Dabei können Sie die Debitoren beispielsweise nach ihrer Buchungsgruppe oder nach dem Buchungsdatum filtern.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Verpackungsgruppe Auswertung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie optional in das Feld Debitoren Buchungsgruppen Filter und wählen Sie einen Datensatz aus.
    3. Richten Sie optional im Inforegister Filter: Artikelposten Ihre Filter ein.
    4. Geben Sie die Auswertung über Senden an... oder Drucken aus.
