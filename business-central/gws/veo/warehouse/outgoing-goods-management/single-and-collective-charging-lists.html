    Einzelladelisten und Sammelladelisten

    Je nach Bestellmenge und Bestellverhalten lassen sich Einzelladelisten oder Sammelladelisten erstellen. Für Kunden, die große Mengen einer Artikelgruppe bestellen, kann es hilfreich sein, Einzelladelisten zu den bestellten Artikeln zu erstellen. Dagegen ist bei Kleinbestellungen eine Sammelladeliste ausreichend.

    So richten Sie die Funktion Nachdrucken ein

    Für das Nachdrucken von Ladelisten müssen verschiedene Einstellungen vorgenommen werden:

    • Standardmäßig ist es nicht möglich, Ladelisten mehrfach auszugeben. Erteilen Sie einzelnen Benutzern die Berechtigung zum Nachdrucken der Ladelisten.
    • Aktivieren Sie bei den Lagerorten, für die ein Nachdrucken möglich sein soll, die Funktion Warenausgang.
    • Ordnen Sie einem Artikel, der auf einer Einzelladeliste oder Sammelladeliste ausgegeben werden soll, eine Lagergruppe zu.
    • Ordnen Sie einem Debitor, für den Einzelladelisten ausgegeben werden sollen, die entsprechenden Lagergruppen zu.

    So erteilen Sie die Berechtigung zum Nachdrucken der Ladelisten

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Benutzer Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie einen Benutzer.
    3. Setzen Sie im Feld Berechtigung Ladeliste nachdrucken einen Haken.

    So aktivieren Sie die Funktion Warenausgang für einen Lagerort

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Lagerorte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Lagerortkarte.
    3. Aktivieren Sie im Inforegister Logistik den Schalter Warenausgang erforderlich.

    So ordnen Sie einem Artikel eine Lagergruppe zu

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Artikel ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Artikelkarte.
    3. Klicken Sie im Inforegister Lager in das Feld Lagergruppe und wählen Sie einen Datensatz aus oder klicken Sie auf Neu, um einen neuen Datensatz anzulegen.

    So ordnen Sie einem Debitor eine Lagergruppe zu

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Debitoren ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Debitorenkarte.
    3. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Debitor > Einzelladelistengruppen, um die Seite Einzelladelistengruppen zu öffnen.
    4. Klicken Sie in das Feld Lagergruppe und wählen Sie einen Datensatz aus oder klicken Sie auf Neu, um einen neuen Datensatz zu anzulegen.

    So drucken Sie Einzelladelisten und Sammelladelisten

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Warenausgänge ein und klicken Sie auf das übereinstimmende Suchergebnis.

      Hinweis

      Der Warenausgang kann nur von Lagermitarbeitern bearbeitet werden. Weitere Informationen finden Sie unter Lagermitarbeiter einrichten.

    2. Öffnen Sie einen Warenausgang.
    3. Klicken Sie im Menüband auf die Schaltfläche Aktionen > Ladeliste > [Ihre Schaltfläche]:
      • Drucken Einzelladeliste: Die Einzelladeliste wird zum ersten Mal gedruckt.
      • Drucken Sammelladeliste: Die Sammelladeliste wird zum ersten Mal gedruckt.
      • Nachdrucken Einzelladeliste: Die Einzelladeliste wird ein weiteres Mal gedruckt.
      • Nachdrucken Sammelladeliste: Die Sammelladeliste wird ein weiteres Mal gedruckt.
