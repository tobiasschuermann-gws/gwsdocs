    Werkzeuge einrichten

    Mithilfe der Werkzeugverwaltung haben Sie die Möglichkeit, Werkzeuge anzulegen, zu bearbeiten und zu verwalten, um diese optimal in Ihre Produktionsprozesse einzubinden und deren Einsatz zu optimieren. Das von Ihnen angelegte Werkzeug kann auf der Artikelkarte und in den Artikelvarianten ausgewählt werden.

    So richten Sie die Werkzeugverwaltung ein

    Weitere Informationen finden Sie unter Nummernserie.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Werkzeuge Einrichtung ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Inforegister Werkzeugverwaltung in das Feld Werkzeugnummern und wählen Sie einen Datensatz aus oder klicken Sie auf Neu, um einen neuen Datensatz anzulegen.

    So richten Sie eine Werkzeugart ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Werkzeugart ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Geben Sie im Feld Code eine Bezeichnung (zum Beispiel G HAMMER) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel Gummihammer) ein.

    So richten Sie die Werkzeugkosten ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Werkzeugkosten ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.
    3. Klicken Sie in das Feld Kreditorennr. und wählen Sie einen Kreditor aus oder klicken Sie auf Neu, um einen neuen Kreditor anzulegen.
    4. Klicken Sie in das Feld Werkzeugart und wählen Sie eine Werkzeugart aus.
    5. Geben Sie im Feld Kosten einen Betrag ein.

    So richten Sie ein Werkzeug ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Werkzeuge ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.

    3. Richten Sie im Inforegister Allgemein Ihr Werkzeug über diese Felder ein:

      Feld Beschreibung
      Kreditorennr. Gibt die Kreditorennummer an. Klicken Sie in das Feld Kreditorennr. und wählen Sie den Kreditor aus, den Sie auf der Seite Werkzeugkosten für dieses Werkzeug hinterlegt haben.
      Nr. Gibt die Nummernserie des ausgewählten Werkzeugs an. Wählen Sie über das Dreipunkt-Menü die Nummernserie aus, die Sie auf der Seite Werkzeuge Einrichtung hinterlegt haben.
      Beschreibung Gibt eine Beschreibung des ausgewählten Werkzeugs an. Geben Sie eine Kurzbeschreibung (zum Beispiel Gummihammer 50mm) ein.
      Status Gibt an, in welchem Status sich das Werkzeug befindet. Es gibt die Status Neu, Gelagert und In Benutzung.
      Werkzeugart Gibt die Art des Werkzeugs an. Klicken Sie in das Feld und wählen Sie eine Werkzeugart aus.
      Breite Gibt die Breite des Werkzeugs an. Geben Sie die Breite des Werkzeugs ein.
      Höhe Gibt die Höhe des Werkzeugs an. Geben Sie die Höhe des Werkzeugs ein und verwenden Sie dieselbe Maßeinheit wie im Feld Breite.
      Größe Gibt die Größe des Werkzeugs an. Die Größe des Werkzeugs wird aus den Feldern Breite und Höhe automatisch berechnet.
      Beschaffungswert Gibt den Beschaffungswert des Werkzeugs an. Der Beschaffungswert wird automatisch aus dem Feld Größe und den zum Werkzeug und Kreditor gehörenden Kosten, die auf der Seite Werkzeugkosten im Feld Kosten zu finden sind, berechnet.

    So erstellen und aktualisieren Sie einen freigegebenen Fertigungsauftrag

    So richten Sie das Werkzeug und die Beschaffungsmethode am Artikel ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Artikel ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Artikelkarte. Klicken Sie alternativ im Menüband auf die Schaltfläche Neu, um einen neuen Artikel zu erstellen. Weitere Informationen finden Sie unter Neue Artikel registrieren.
    3. Klicken Sie im Inforegister Beschaffung, Gruppe Fertigung in das Feld Werkzeug und wählen Sie das von Ihnen erstellte Werkzeug (in unserem Beispiel Gummihammer 50mm) aus.
    4. Klicken Sie im Inforegister Beschaffung in das Feld Beschaffungsmethode und wählen Sie die Option Fertigungsauftrag aus, damit Sie im Nachgang den Fertigungsauftrag aus dem Verkaufsauftrag erstellen können.
    Hinweis

    Alternativ können Sie Ihr Werkzeug über die Artikelvariante einrichten:

    • Klicken Sie in der Artikelkarte im Menüband auf die Schaltfläche Zugehörig > Artikel > Varianten. Klicken Sie dann auf der Seite Artikelvarianten in das Feld Werkezeug und wählen Sie das von Ihnen erstellte Werkzeug (in unserem Beispiel Gummihammer 50mm) aus.
    • Weitere Informationen finden Sie unter Produktvarianten verwalten.

    So erstellen Sie einen freigegebenen Fertigungsauftrag aus einem Verkaufsauftrag

    Um freigegebene Fertigungsaufträge im Inforegister Zeilen der Werkzeugkarte anzuzeigen, können Sie diese über den Verkaufsauftrag neu erstellen.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Verkaufsaufträge ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu, um einen neuen Verkaufsauftrag zu erstellen. Füllen Sie alle relevanten Belegkopffelder aus und fügen Sie Belegzeilen hinzu.
    3. Klicken Sie im Inforegister Zeilen in das Feld Nr. und wählen Sie den Artikel aus, bei dem Sie Ihr Werkzeug (in unserem Beispiel Gummihammer 50mm) hinterlegt haben.
    4. Klicken Sie im Menüband auf die Schaltfläche Aktionen > Planen > Planung.
    5. Klicken Sie im Menüband auf die Schaltfläche FA erstellen.
    6. Wählen Sie auf der Berichtsanforderungsseite Möchten Sie einen Fertigungsauftrag für diesen Verkaufsauftrag erstellen? im Feld FA-Status die Option Freigegeben aus und bestätigen Sie Ihre Auswahl mit Ja.
    7. Bestätigen Sie das Dialogfenster mit OK.
