    Lagerklassen für eine Lagerzone oder einen Lagerplatz einrichten

    Lagerklassen erleichtern die Zuweisung von Artikeln (zum Beispiel für Tiefkühlware oder Gefahrgut) zu bestimmten Lagerplätzen. Ein Lagerplatz oder eine Lagerzone kann mehreren Lagerklassen zugeordnet werden.

    So richten Sie die Logistik am Lagerort ein

    Weitere Informationen finden Sie unter Lagerorte einrichten.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Lagerorte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Lagerortkarte.
    3. Aktivieren Sie im Inforegister Logistik, Gruppe Einkauf, Verkauf, Service und Umlagerung den Schalter Wareneingang erforderlich.
    4. Aktivieren Sie im Inforegister Logistik, Gruppe Projekt den Schalter Lagerplatz notwendig.

    5. Aktivieren Sie optional den Schalter Gesteuerte Einlag. u. Kommiss., um erweiterte Logistikfunktionen am Lagerort zu nutzen.

      Hinweis
      • Nach der Aktivierung des Schalters Gesteuerte Einlag. u. Kommiss. werden die Schalter der Gruppe Einkauf, Verkauf, Service und Umlagerung automatisch gesperrt und ausgegraut. Zudem wird in der Gruppe Projekt der Schalter Zuordnung verwenden automatisch aktiviert.
      • Um weitere Logistikfunktionen am Lagerort nutzen zu können, gehen Sie vor wie in Eingehende und ausgehende Abläufe in erweiterter Lagerkonfiguration mit gesteuerter Einlagerung und Kommissionierung und Artikel und Lagerorte für die gesteuerte Einlagerung und Kommissionierung einrichten beschrieben.
      • Wenn Sie den Schalter Gesteuerte Einlag. u. Kommiss. aktivieren, müssen Sie eine Lagerzone und eine Lagerplatzart hinterlegen. Um einen Zonencode für die Lagerzone und einen Lagerplatzartcode für die Lagerplatzart einzurichten, gehen Sie vor wie in So richten Sie einen Zonencode ein und So richten Sie einen Lagerplatzartcode ein beschrieben.

    So richten Sie Lagermitarbeiter ein

    Damit Sie Lagerklassen zuordnen können, sollte mindestens einen Benutzer als Lagermitarbeiter hinterlegt sein. Um einen Lagermitarbeiter zu hinterlegen, gehen Sie vor wie in Lagermitarbeiter einrichten beschrieben.

    So richten Sie Lagerklassen ein

    Lagerklassen ermöglichen die Aufteilung des Lagers in Abschnitte, um bestimmte Anforderungen der Lagerung erfüllen zu können.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Lagerklassen ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Geben Sie im Feld Code eine Bezeichnung (zum Beispiel GEKÜHLT) und im Feld Beschreibung eine Kurzbeschreibung (zum Beispiel 2 Grad Celsius) ein.

    So richten Sie eine Lagerzone am Lagerort ein

    Eine Lagerzone ermöglicht die Einhaltung von spezifischen Anforderungen (zum Beispiel eine bestimmte Kühltemperatur). Zusätzlich können am Lagerort weitere Lagerzonen eingerichtet werden.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Lagerorte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie einen Lagerort.
    3. Klicken Sie im Menüband auf die Schaltfläche Lagerort > Zonen.

    4. Richten Sie die Lagerzone über diese Felder ein:

      Feld Beschreibung
      Code Gibt den Code für die Lagerzone an.
      Beschreibung Gibt eine Beschreibung der Lagerzone an.
      Lagerplatzartencode Gibt den Lagerplatzartencode für die Lagerzone an. Die Lagerplatzart bestimmt den eingehenden und ausgehenden Lagerdurchlaufprozess.
      Lagerklassencode Gibt den Lagerklassencode der Lagerzone an. Artikel mit gleichem Lagerklassencode können in der Lagerzone gelagert werden.
      Lagerhilfsmittelcode Gibt den Code des Lagerhilfsmittels an, der in dieser Lagerzone genutzt werden soll.
      Zonenpriorität Gibt die Priorität für eine Lagerzone an, die allen Lagerplätzen innerhalb dieser Zone zugeordnet wird.
      Zuordnungslagerplatzzone Gibt an, ob es sich um eine Zuordnungszone des Lagerplatzes handelt.

    So ordnen Sie eine Lagerklasse einer Lagerzone zu

    Ein Lagerklasse ermöglicht die Lagerung von ähnlichen Waren innerhalb einer Lagerzone.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Lagerorte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie einen Lagerort.
    3. Klicken Sie im Menüband auf die Schaltfläche Lagerort > Zonen.
    4. Markieren Sie eine Lagerzone, der eine Lagerklasse zugeordnet wird.
    5. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Zone > Lagerklassen.
    6. Klicken Sie im Menüband auf die Schaltfläche Neu oder klicken Sie alternativ in die nächste freie Zeile, um eine Lagerklasse für diese Lagerzone zuzuordnen. Die Werte in den Feldern Lagerortcode und Zonencode werden automatisch hinterlegt.
    7. Klicken Sie in das Feld Lagerklassencode und wählen Sie einen Datensatz aus.

    So richten Sie einen Lagerplatz am Lagerort ein

    Ein Lagerplatz ermöglicht die effiziente Lagerung von Waren. Zusätzlich können am Lagerort weitere Lagerplätze eingerichtet werden.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Lagerorte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie einen Lagerort.
    3. Klicken Sie im Menüband auf die Schaltfläche Lagerort > Lagerplätze.

    4. Richten Sie den Lagerplatz über diese Felder ein:

      Feld Beschreibung
      Code Gibt den Code für den Lagerplatz an.
      Beschreibung Gibt eine Beschreibung des Lagerplatzes an.
      Zonencode Gibt den Code der Zone an, in der sich der Lagerplatz befindet. Um den Zonencode einzurichten, gehen Sie vor wie in So richten Sie einen Zonencode ein beschrieben.
      Lagerplatzartencode Gibt den Code der Lagerplatzart an, die für den Lagerplatz gilt. Um den Lagerplatzartencode einzurichten, gehen Sie vor wie in So richten Sie einen Lagerplatzartcode ein beschrieben.
      Lagerklassencode Gibt den Code der Lagerklasse an, die sich auf den Lagerplatz bezieht.
      Lagerplatzpriorität Gibt die Priorität eines Lagerplatzes an. Artikel mit einer hohen Priorität werden bevorzugt bearbeitet.
      Leer Gibt an, dass der Lagerplatz keine Artikel enthält. Der Haken wird automatisch gesetzt.
      Dediziert Gibt an, dass die Bestände im Lagerplatz vor der Kommissionierung für andere Bedarfsanforderungen reserviert sind.
      Letztes Lagerplatzinventurdatum Gibt das Datum und die Uhrzeit der letzten Registrierung einer anstehenden Lagerplatzinventur an, sobald es eine registrierte Lagerplatzinventur für diesen Lagerplatz gibt.
      Wegeoptimierungs Priorität Gibt die Priorität für die Lagerplatz-Wegeoptimierung für Kommissionierungen an.
      Lagerplatzpriorität Code Gibt den Code für die Lagerplatzpriorität eines Lagerorts an.
      Priorität Gibt die Priorität für die Einlagerung an.
      Hinweis
      • Sollten die Felder Zonencode, Lagerplatzartcode und Lagerklassencode in Ihrer Ansicht fehlen, können Sie diese über Personalisieren einblenden. Weitere Informationen finden Sie unter Ihren Arbeitsbereich personalisieren.
      • Wird in dieser Lagerzone ein neuer Lagerplatz erstellt, übernimmt dieser automatisch die Lagerklassen der Lagerzone.
      • Durch die Einrichtung des Lagerorts werden die Lagerplätze im Wareneingang, im Warenausgang und bei der Kommissionierung automatisch hinterlegt.

    So richten Sie einen Zonencode ein

    Der Zonencode ermöglicht die genaue Bestimmung des Lagerplatzes innerhalb des Lagers.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Lagerorte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie einen Lagerort.
    3. Klicken Sie im Menüband auf die Schaltfläche Lagerort > Lagerplätze.
    4. Klicken Sie in das Feld Zonencode und wählen Sie einen Datensatz aus.

    So richten Sie einen Lagerplatzartcode ein

    Der Lagerplatzartcode ermöglicht die Kategorisierung des Lagerplatzes nach bestimmten Kriterien (zum Beispiel Temperaturanforderungen).

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Lagerorte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie einen Lagerort.
    3. Klicken Sie im Menüband auf die Schaltfläche Lagerort > Lagerplätze.
    4. Klicken Sie in das Feld Lagerplatzartcode und wählen Sie einen Datensatz aus.

    So ordnen Sie eine Lagerklasse einem Lagerplatz zu

    Die Zuordnung einer Lagerklasse zu einem Lagerplatz ermöglicht die Kategorisierung von Waren nach bestimmten Kriterien (zum Beispiel Gewicht) zur Optimierung von Lagerraum, Sicherheit und Effizienz der Lagerverwaltung.

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Lagerorte ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Markieren Sie einen Lagerort.
    3. Klicken Sie im Menüband auf die Schaltfläche Lagerort > Lagerplätze.
    4. Markieren Sie einen Lagerplatz.
    5. Klicken Sie im Menüband auf die Schaltfläche Zugehörig > Lagerplatz > Lagerklassen.
    6. Klicken Sie im Menüband auf die Schaltfläche Neu, um die Lagerklasse für diesen Lagerplatz zuzuordnen oder klicken Sie alternativ in die nächste freie Zeile. Die Werte in den Feldern Lagerortcode, Zonencode und Lagerplatzcode werden automatisch gesetzt.
    7. Klicken Sie in das Feld Lagerklassencode und wählen Sie einen Datensatz aus.
