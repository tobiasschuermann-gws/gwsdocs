    Gefahrgüter einrichten

    Die Artikelstammdaten können um Gefahrgutinformationen ergänzt werden. Bei Lieferungen von Gefahrgutartikeln werden entsprechende Gefahrgutpunkte berechnet und Gefahrgutposten erstellt. Diese Informationen sind in tabellarischer Form auf den Belegen Verkaufsrechnung, Lieferschein und Proformalieferschein enthalten. Des Weiteren enthält die Artikelbeschreibung in den Belegen sämtliche Gefahrgutinformationen, die hinterlegt sind.

    So richten Sie das Gefahrgut ein

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Gefahrgüter ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Klicken Sie im Menüband auf die Schaltfläche Neu.

    3. Richten Sie das Gefahrgut über diese Felder ein:

      Feld Option Beschreibung Beispiel
      Gefahrgutcode Geben Sie einen eindeutigen Gefahrgutcode ein. GFNL01
      UN Nummer Geben Sie die UN Nummer des Gefahrguts ein. UN1824
      Stoffbezeichnung Geben Sie die Stoffbezeichnung des Gefahrguts ein. Natronlauge
      Bezeichnung Geben Sie optional eine weitere Beschreibung ein Ätzend, bei Kontakt schwere Verbrennungen
      Verpackungsart Wählen Sie optional eine Verpackungsart aus. PALETTE
      Beförderungskategorie 0, 1, 2, 3, 4 Wählen Sie die Beförderungskategorie des Gefahrguts aus. 2 (es gelten besondere Vorschriften)
      Faktor Wird automatisch berechnet und hinterlegt Gibt den Faktor des Gefahrguts an, der für die Berechnung der Gefahrgutpunkte verwendet wird. 3
      Gefahrgutklasse Geben Sie die Gefahrgutklasse ein. Klasse 8 (Ätzende Stoffe)
      Tunnelbeschränkungscode B, B1000C, B/D, C, C5000D, C/D, C/E, D, D/E, E Wählen Sie einen Tunnelbeschränkungscode aus. E (Keine Beschränkung für den Transport durch Tunnel)
      Gefahrzettelmuster Geben Sie optional das Gefahrzettelmuster des Gefahrguts ein. 1.2+6.3
      Verpackungsgruppe I, II, III Wählen Sie eine Verpackungsgruppe aus. II (mittelgefährlich)

    So kennzeichnen Sie einen Artikel als Gefahrgut

    1. Klicken Sie im Menüband Dynamics 365 Business Central auf das Symbol Lupe , geben Sie im Suchfeld Artikel ein und klicken Sie auf das übereinstimmende Suchergebnis.
    2. Öffnen Sie eine Artikelkarte.
    3. Klicken Sie im Inforegister Gefahrgut in das Feld Gefahrgutcode und wählen Sie einen Code aus.
    4. Geben Sie im Feld Gefahrgutmenge einen Wert ein.
    Hinweis
    • Die Gefahrgutmenge wird anhand dieser Formel berechnet:
      Menge geliefert * Gefahrgutmenge (Artikelstammdaten)
    • Auf Basis dieser errechneten Gefahrgutmenge werden die Gefahrgutpunkte berechnet:
      Gefahrgutmenge * Faktor (Gefahrgutstammdaten)
