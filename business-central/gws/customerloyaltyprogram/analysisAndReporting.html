    Analyse und Auswertungen mit Kundenkarten

    Verkaufsanalyseansichten für Kundenkarten

    Sowohl an Kundenkarten als auch an Kundenkartenarten können Standard Dimensionen hinterlegt werden. Diese können herangezogen werden, um Verkaufsanalyseansichten aufzubauen. Für mehr Informationen finden Sie hier die Microsoft Hilfe: Analysieren von Daten nach Dimensionen.

    Beispiel um Daten für Verkaufsanalyseansichten zu erzeugen

    1. Stellen Sie sicher, dass eine Kundenkarte mit Debitor mit einer Kundenkartenart im System angelegt ist.
    2. Legen Sie einen neuen Verkaufsauftrag an.
    3. Identifizieren Sie einen Debitoren mithilfe der Kundenkarte. Zu diesem Zeitpunkt sind bereits die Standard Dimensionen der Kundenkartenart und der Kundenkarte selbst am Auftrag hinterlegt.
    4. Buchen Sie den Auftrag zur Rechnung durch.
    5. Folgen Sie nun der Microsoft Hilfe Analysieren von Daten nach Dimensionen.

    Debitorenpostenauswertung in Excel

    1. Öffnen Sie die Seite Debitorenposten.
    2. Suchen Sie auf der Seite Debitorenposten nach der Aktion Dimensionsfilter festlegen.
    3. Geben Sie die gewünschten Dimensionsfilter ein und schließen Sie die Seite.
    4. Wählen Sie die Aktion In Excel öffnen aus.
