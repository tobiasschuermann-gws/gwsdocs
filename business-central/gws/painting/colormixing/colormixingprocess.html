    Farbmischprozess

    Wenn die Einrichtungen ausgeführt wurden, kann der Basisartikel im Verkaufsbeleg ausgewählt werden. Die Seite Variante auswählen wird automatisch geöffnet. Alternativ können Sie die Seite Variante auswählen über die Funktion Variante erstellen in den Zeilenfunktionen öffnen.

    Auf dieser Seite können Sie eine vorhandene Variante auswählen oder eine neue Variante erstellen. Die Beschreibung der Variante wird in die Verkaufszeile übernommen. Zusätzlich können Sie hier aus den eingerichteten Preiszuschlägen einen Datensatz auswählen. Der Zuschlagspreis wird aus dem am Artikel eingerichteten Preiszuschlag übernommen und kann hier bei Bedarf geändert werden. Die Kombination aus Variante und Preiszuschlag wird gespeichert und kann erneut genutzt werden.

    Wird die Seite Variante auswählen mit OK geschlossen, wird die Verkaufszeile mit dem Variantencode und den Informationen aus dem Preiszuschlag gefüllt. Der Verkaufspreis wird um den gewählten Preiszuschlag erhöht.

    Das Mischen der Basisfarbe mit dem gewünschten Farbcode erfolgt über die Montageaufträge. Wird in der Verkaufszeile das Feld Menge für Auftragsmontage gefüllt, wird im Hintergrund ein Montageauftrag zur Erzeugung der Farbe erstellt. Dabei wird der Basisartikel verbraucht und die gewünschte Farbe als Variante zugebucht, um die Bestandsveränderungen im System nachvollziehbar darzustellen.
